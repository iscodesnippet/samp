家律師事務所也對其發起集體訴訟，控告瑞幸咖啡作出虛假誤導違反美國證券法。

瑞幸咖啡收到那斯達克交易所的下市通知時，於19日對外聲明要求舉行聽證會，希望在聽
證會結果發布前，繼續在納斯達克上市。董事長陸正耀表示，納斯達克不等最終調查結果
就要求公司退市，出乎意料，對此個人深感失望和遺憾。

間週三（20日）上午7點復牌。

: 2.原文內容：
: 美股昨天在尾盤出現下殺走勢，主要因新聞網站STAT News的報導質疑Moderna周一宣布新
: 冠病毒疫苗試驗結果的有效性。相關消息拖累標普500指數由紅翻黑，道瓊指數終場跌逾
: 390點。以下整理上述報導中，對這支疫苗的初步試驗結果的質疑之處。（延伸閱讀：疫
: 苗試驗受疑拖累尾盤下殺 美股道瓊挫跌390點）
: ●為何大多是文字敘述，而非數據
: 當Moderna的新冠病毒疫苗初步試驗結果在周一轟動媒體時，公布的多數內容是文字，而
: 非數據。但對科學家而言，數據勝過一切。
: ●美國國家過敏與傳染病研究所（NIAID）維持沉默
: 美國國家過敏和傳染病研究所（NIAID）和Moderna合作開發這支疫苗，由NIAID的科學家
: 打造疫苗架構或原型，而該機構正在進行第一階段測試。本周稍早Moderna公布的結果，
: 是來自NIAID率領的第一階段測試數據。
: 在以往，美國國家過敏和傳染病研究所通常會大肆宣傳自己的發現，但這一回卻保持沉默
: ，本周一並未發布任何消息，也不願對Moderna的聲明提出回應。
: ●只揭露8名受試者的抗體結果
: Moderna聲明中指出，所有45名受試對象體內都產生結合抗體（binding antibodies），
: 但僅8名受試者產生中和抗體（Neutralizing antibody），也就是能對抗新冠病毒的抗體
: 。
: 外界無法得知其他37名受試者的結果，這不代表他們沒產生中和抗體，原因是要測試出中
: 和抗體更為耗時，而且必須在生物安全第三級實驗室執行。Moderna公布8名受試者的結果
: ，是因為目前進度只到如此。
: 此外，第一階段的受試者年齡介於18歲至55歲，但Moderna未公布上述產生中和抗體的8名
: 受試者年齡。這代表如果產生中和抗體者集中在年輕族群，那麼較年長者對疫苗的反應尚
: 不得而知。
: ●疫苗持久性？
: 這些產生中和疫苗受試者，是在第二次接種疫苗後的兩周後進行抗體測試。
: 約翰霍普金斯大學疫苗研究人員Anna Durbin表示：「這還非常早期。我們不知道這些抗
: 體是否持久。」
: Moderna的股價繼18日大漲20%至80美元後，19日挫跌10.41%至每股71.67美元。
