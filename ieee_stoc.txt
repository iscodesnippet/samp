: 配息也有人說是公司拿你的錢 左手買進的100發到1元到你的右手
: 但是要建立在公司"有賺錢"的情況
: 賺一元 配一元 簡單來說就是會有個時間差
: 假設配1元 隔年公司又賺了1元
: 那公司的淨值又回歸到100元 這時理論上股價就會回到100
: : 然後我看很多文章說還有個填權息 就是如果這支股票看好 之後就會漲回100 甚至更高

: 這邊你的理解只能算對一半
: 股息是公司發的 但股價是投資人炒得
: 公司確實把獲利分給了股東 但如你所說股價是後面的人買上來的這點沒錯!
: 但是你忽略了 為什麼股價會被買上來
: 影響股價的因素很多:公司獲利能力優秀、消息面、前景看好的集體共識
: 當公司淨值上來 且持續獲利 而股價卻不變的情況下
: 會不會有一批人 也想進場分這杯羹
: 如果股價沒有填息 但公司持續賺錢，股價低於淨值，就屬於公司被低估的情況
: 那股票漲上來的機會就很大
: 假如 大家發現這間公司很賺錢 本益比低 獲利好
: 也想進來分一杯羹
: 努力買到股價接近淨值 甚至超過淨值
: 因為有些人認為雖然高於淨值 但是"我少賺一點沒關係 至少我穩定賺錢啊"
: 所以還是會高於淨值買進
: 這道理就像特別股 高於收購價 但還是有人會買
: 總而言之 你把配息當作是投資人買進幫你填息，你才能獲利
: 但是這個觀念有點導因為果 通常投資人發現一個好的標的 就會買入搭上順風車
: 只要公司前景 獲利不變 且持續獲利 但股價被低估(低本益比、低淨值比...等)
: 包括公司派、法人等等 都不會錯過著個賺錢的機會
: 雖然不見得會會飆漲 但是至少會朝他應有的價值邁進
: 進而填息 這樣你清楚了嗎
