新人進當沖界一開始最喜歡沖10元 100元左右的標的
因為跳檔級距的關係，這種價位一檔的金額比較大
以為跳一檔扣掉手續費證交稅就可以賺很多

但是新人往往忘記了，你輸一檔輸更多(加上手續費和證交稅)

什麼價格的標的比較好? 其實跳檔越細的越好

新手眼裡只有目標利潤可能賺多少
老手在意的是進場風險萬一賠多少

什麼時候可以在台股玩百元tick流? 等證交稅非常低的時候

※ 引述《alan80620 (安)》之銘言：
: 很多人都在詢問手續費怎麼算
: 這邊就在此統一回覆說明
: 買=手續費 0.1425%
: 賣=手續費 0.1425%+證交稅0.3%
: (當沖證交稅減半為0.15%)
: 以100元股票為例
: 今日買100元股票 明日賣100元股票
: 手續費未打折:142.5+142.5+300=585元
: 手續費打三折:42.75+42.75+300=385.5元
: 所以一百元股票(買+賣)就差了近200元的手續費
: 假設當沖 當日買賣同一檔100元股票
: 手續費未打折:142.5+142.5+150=435元
: 手續費打三折:42.75+42.75+150=235.5元
: 所以有些當沖者在說他是tick流
: 假設當沖 買進100元股 賣出100.5元
: 手續費假設三折 手續費+證交稅會是:
: 43+43+150=236
: 獲利500 - 手續+證交費236 = 264
: 也就是跳一檔升0.5元隨即賣出
: 扣掉手續費即可獲利264元
