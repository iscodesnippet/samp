大大你好 拜讀了你許多文章，真是覺得大大的操作上以及想法上真的是藝術層次

尤其有個想法真的是認同，那就是操作的時候自己的執念是最不重要，甚至是阻礙的
因為怎樣執著在上面，都跟現實面的價格一點關聯都沒有，也無法影響

小弟我的操作模式是波段，標的是黃金石油外匯，之前的績效還可以，但因為最近
的標的全都是走趨勢型，導致我的波段失利，又加上止損線我拉很長，所以損失慘重
有點沒自信

在這邊算是跟大大分享，加上可能自己投資方法有盲點，所以想看看大大對我的方法
有沒有錯誤的地方：
小弟玩波段，不過波段雖然核心概念是找出波峰波股，但找的方式有各種方法。
小弟找的方式是兩個核心概念：
1. 只要出現明顯大k棒（各標的大小不一），代表有>53%機率會回撤，是進入點
2. 運用大的停損線以及小的停利線（大約是5:1~10:1）去製造高勝率（90%）

實際上測試概念可行的方式是 首先用程式去做隨機買賣，單純運用停損停利去找出
90%以上勝率（時間線大約是1分鐘線～15分鐘線為一個k棒），期望值為0的點，
然後再導入明顯大k棒來當作買賣的判斷依據去取代隨機買賣，如果原本的90%勝率，期望
值為0的隨機買賣，因為導入之後而變成勝率增加（但停損停利不變），期望值自然就會
變成是正的，那麼這個就可以導入做買賣，
其實說穿了就是投機的概念，賭說我這樣設定停損停利，外匯都會自然地回撤，證據就是
隨機買賣勝率90%，核心就是運用大的止損範圍讓價格隨著時間以及外匯的回撤習性，回到
原本的價格，甚至回來之後還頂到我的止盈，讓隨機買賣有高勝率。再加上賭說大k棒會
略微的增加回撤的機率，因此原本期望值為零，就因為這略微增加的機率，變成正期望值

然後實際操作就是程式交易來當作基準，實際上做手動交易：只要趨勢目前跟程式交易是
反向的，我就手動交易下去。止損停利點跟原本是一樣的，這樣一來我可以擴大我的獲利
空間，縮小我的止損

因此只要不是趨勢型的走向，這個方法大致很好，但最大問題點就在於如果
走向是趨勢型（也就是勝率90%以外的那個10%發生了），我手動加碼的量一大，即使
止損範圍不大，但仍然損失慘重。

聽起來方法不太像波段對吧？但其實真正做波段是在我手動加碼的時候（而不是程式）
因為停損線是固定的，加上確認了勝率是90％，這意味著當股票跑到損失的那邊（還沒到
停損現之前），有90%會回撤。於是手動加碼就是意味著在這90%回撤的狀況去找轉折點，
也就是波鋒波股。 找的方式通常都是看近期的支撐線在哪，就在那邊下

我思考過後，其實我自己看到的缺點是
1. 當價格是跑到損失那一方時，我通常都會手癢手動加碼（沒有按照紀律在支撐線的點
去做手動下單，而是感覺k棒越來越長時就開始一直加碼），並且調高止損，導致只要
真正趨勢出現時，我就會損失很重
2. 沒有良好的資金管理方式，只覺得機會一來就下大注，加上我玩的是保證金模式，所以
風險就會更加大了。我自己下注小的時候是資金10% 大的時候甚至80~90%都有，導致常常
在爆倉邊緣
3. 盲目的加碼，絲毫沒有考慮實際上的趨勢在哪

我自己的分享大概是這樣，一方面想看看大大指點小弟我有沒有盲點沒看到的

二方面是有點想偷師大大的技術，最主要的是想補足我第三個盲點，也就是如何去找出
趨勢所在。這一塊是我完全沒有涉獵過的地方。
我是學技術指標，加上數學背景，所以任何技術指標怎麼算的都沒問題，但沒有任何
圖形學背景。但看大大好像在除了籌碼，價量以外的技術面，是以圖形學的型態當作依據
雖然我不熟，但不論如何我不想放過任何學習的機會

因此想了解一下，大大是怎麼去看趨勢的
比如圖形學的型態（假設大大技術面是只看型態的話），是大家所看的頭線頸線，
波浪理論這些，是大大的依據嗎？以及價量關係有沒有一些基本的想法呢？

再來是大大的止損是怎麼設定的？這也是我很頭痛的一點。因為止損空間只要一大，那麼
意味著把自己的資金全然交給價格的回撤性。因此只要趨勢一出現，就一去不回頭了
而且止損空間大，也同時代表著有大量的時間，價格都在損失的區間繞來繞去，萬一
這期間有消息不利，就會碰觸到我的止損線。

因此想知道“止損拉高來提高勝率”，這個想法是正確的嗎？
是不是可以請大大分享您的止損設定呢？

偷師技術雖然很厚臉皮，但是因為小弟我的標的應該跟大大毫無重疊，所以應該不至於
影響大大，先謝謝大大啦 哈ＸＤＤＤ

: 更多的部份有興趣的人可自行去網路上搜尋過去文章以免佔太多版面
: 個人操作的商品包括台股現貨、個股期貨、權證、台指期、台指選擇權、CB/CBAS
: 交易以主觀交易為主, 週期多數偏短, 包含當沖 (現貨、個股期)
: 隔日沖至數日的短波段 (現貨、權證), 較長週期的部份多數是選擇用 CB 標的來持有

: 2018 年預備的操作資金約為 3000 多萬左右, 但其實很少有機會用滿 (平均約不到 20

: 操作主要依據即時盤面的訊息搭配籌碼面、技術面、和一些消息面, 基本面則非常少用到
: (頂多每月月初的營收 MoM、YoY 會特別注意一下, 但從不做公司未來發展之類的預測)
: 操作的核心則如同 ID 所示, trade like shadow
: 哪裡有趨勢或獲利機會就如同影子一樣隨著勢做交易, 沒有偏見、沒有執念、
: 沒有一定的方向, 保持彈性不要有太多個人定見
: 畢竟來市場的目的是賺錢而不是證明你的看法很準、你的理論很行 (志在開課的人例外)
: 因為你的看法其實對股價未來的走勢一點影響力也沒有...
: 標作的標的一部份是前一天自選, 一部份是當天即時遴選
: 自選股即依據前面所講的籌碼面、技術面、量大價幅大之類的價量條件來篩選
: 或是因發佈特定消息而能預期隔天會有大波動的股票 (如財報公佈、收購、產業變動等)
: 盤中即時遴選股則會掃瞄個股是否有出現量能變化、成交速度加快、五檔明細異動、
: 股價波動變大之類的特徵 (相較於該股前一段時間的表現) 來回過濾篩選
: 我個人也很看重「相對強弱」的比較, 例如亞股跟美股相比、台股指數跟亞股指數相比、
: 族群跟台股指數相比、個股強弱跟族群相比、個股今日表現跟前幾日走勢相比..等等
: 然後做出今日應偏多或偏空思考、下單資金部位的大小、避險或鎖單比例等決策
: 記得以前有說過盤勢就像一齣連續劇, 有它進行的起承轉合和時空背景
: 股價的變化其實是果, 股票的「成交」才是因, 有時多多思考究竟哪些東西會影響成交
: 比起整天在尋找神奇指標會有更意想不到的幫助
: 而對於籌碼面的研究, 我知道很多人每天收盤就是等著看「自己持股」的法人買賣超、
: 資券餘額等數據, 然後即使五項數據都對持股不利只有一項有利, 還是會自我催眠這項
: 有利因素將會戰勝其它五項不利因素而使得自己解套, 這樣的研究方向根本對操作無益
: 我自己過去是花不少的時間在研究個股的贏家分點或法人在該股上的買點、買法、掛法

這種利用制度瑕疵短線操作的方法容易受到法規政策變化影響

二、
近幾年開始加入中長期波段操作
試圖去尋找一些標錯價的股票
在板上發表過幾檔標的文(華泰、高鐵、中砂)
由特定事件出發
對於該檔標的細部拆解
認為有機會後擬定操作策略進場
這樣的操作與原本自己的短線操作完全不同
但實際上兩邊都接觸之後
卻又能激盪出不同的思考火花

三、
儘管由學生時期至今10年的投資經驗
大概於股市獲利近500萬
但對於這些收入感到不確定性
持續懷疑自己只是一個長期幸運的人
因此過去習慣每年都將獲利放很大一部分回到較保守的資產中(例如:CASH)
導致這幾年複利累積的效果不好
直到近三年進入職場，有了不錯的薪資收入之後
才有比較顯著的複利效果，直到今年投資收益絕對金額才破7位數
穩定的薪資收入可作為投資風險的後盾
而早期累積的投資經驗則可作為創造可能性的突破口

四、總結
自己以一個初出社會的小資族理財來講
儘管板上各種大神的神蹟對帳單讓人充滿嚮往
畢竟我們每個人都對於財富自由充滿渴望
但築夢要踏實，先把腳步踩穩才能一步一步往目標前進
進入股市第一課就要知道風險跟報酬是一體兩面
對於一個正常一般普通家庭的小孩而言
初期賺錢的效率薪資收入絕對高於您在股市殺進殺出
但投資的收益則是需要時間的累積才能展現成效

看完這些對帳單，正確的心態應該是：

1.尋找其中可吸收為己用的部分
2.視為一個追尋的動力即可

盪是主基調，個股兩極分化將持續強化，巨額解禁將進一步加深大盤強勢震盪力度，但結
構性行情機會仍不可忽視。
