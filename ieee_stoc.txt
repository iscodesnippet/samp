在轉機，並預測在雲計算需求的推動下，2020年的收入和利潤將超過華爾街的預期。

該公司在其備受關注的數據中心業務中的銷售額增長了19％，幫助其超過了第四季度的利
潤和營收預期，並使其股票在延長交易中上漲了7％。

首席財務官喬治戴維斯（George Davis）在接受採訪時說，第四季度對雲計算提供商的
銷售額同比增長48％，預計這一趨勢將在今年持續。

根據Refinitiv的IBES數據，這家總部位於加利福尼亞州聖克拉拉的芯片製造商預計，
2020財年收入約為735億美元，比華爾街共識高出10億美元。

戴維斯沒有指定具體的客戶，而是所謂的“超大規模”雲提供商，例如美國的Amazon
Web Services（O：AMZN）和Microsoft Corp（O：MSFT）以及阿里巴巴集團控股（N：
BABA）和百度（O：BIDU）都為數據中心購買了英特爾晶片，這些晶片的容量出租給了大
型企業。


戴維斯對路透社表示：
"我們看到的是雲廠商的強勁需求"
"我不得不讚揚本季度的超大規模產品"

英特爾表示，其向新一代晶片製造技術的過渡進展比預期的要好，並且將提高其生產個人
計算機晶片的能力，這表明過去一年困擾晶片製造商的製造困境已開始緩解。

“我們認為這將幫助我們彌合我們一直在遇到的客戶需求方面的差距。我們認為所有這些
都將在2020年得到解決，並且實際上我們將能夠在年底開始建立庫存一年來，這已經兩年
了。”戴維斯告訴路透社。

英特爾的樂觀前景是在周三晶片製造商德州儀器（O：TXN）樂觀的預測以及上週台灣半導

英特爾已將其核心市場（如個人計算機和數據中心）擴大了一倍，兩者均超出了分析師對
第四季度的預期。

英特爾客戶計算業務的收入滿足了個人電腦製造商的需求，仍然是銷售額的最大貢獻，第
四季度增長了2％，達到100億美元，超過了FactSet估計的97.4億美元。

英特爾在處理器市場上的最大競爭對手Advanced Micro Devices Inc（O：AMD）的股票上

Cascend證券分析師埃里克羅斯（Eric Ross）對路透社表示： AMD（股票）上漲是因

我覺得就算是第3種人也要有一定的知識與勇氣才敢下單常抱賺錢


我最早知道股票的事情是在我很小很小的時候  因為親戚很熱衷
細節都已經模糊了  我只記得小猶太姊姊很正我超喜歡小猶太姊姊
我長大以後取的老婆要長的像小猶太姊姊    當時對股票完全無感
如果不知道小猶太姊姊是誰  其實也不重要  這跟股票沒啥關係


多年以後我親戚大量買進當時不到20元(親戚說18以下)的玉晶光
那時候我只知道買對股票可以發大財  對於買錯股票還是沒感覺
這時期親戚大力勸說我老爸(散戶代表)跟進
我老爸買了幾張出在不到60  其他親戚也出在不到80
那一波只有那個親戚自己抱到300以上  確定不到400
親戚後來跟我說那時候他已經說到只差沒有跪下求我爸繼續買


又過了好幾年我也進入股市(技術分析)
那時候有個零沒那麼多的男人喊買AMD  這段歷史大家都清楚我就不細說了
後來AMD先漲一波到15盤整還有回跌破10

最初看到辣個男人喊AMD的時候  我也沒放在心上
那個時候產業研究對我來說是旁門左道
後來AMD漲破10元之後我被嚇到深切反省  並想起自己也是電資背景
然後好好的了解半導體產業與半導體相關個股之間的關聯性
終於我了解了甚麼是產業研究

然後我跟我親戚(這時期零也是很多)說AMD的時候  我親戚完全不放在心上
後來我繼續勸說台積電很好應該買進  我親戚還是覺得代工沒有前途
2019年初我跟我親戚聯發科可以買    我親戚依然覺得IC設計不行

2020年我親戚跟我說  台積電不行  政治風險很高
因為不去美國設廠美國價值不夠  所以美國會扶植英特爾  台積電GG


結論:1.交易股票是自己的事情  真的要自己做功課認真學習才能真正了解市場
2.就算聽到高手說的話    如果自己不瞭解股市大多是聽不下去
3.就算自己是高手如果不跟隨時代進步也很難持續發大財

: 先說內線交易。上市公司若須處理土地或資產、增減資本、配發股息或股票等，將影響股
: 價漲跌至鉅。上述的種種行為均須召開股東大會，並經董事會依法定程序同意，再經會計
: 師查核簽證後，才能公開。
: 在消息未公開之前，事先獲悉內線消息的董監事、大股東與主辦人員，先行買進或放空，
: 再利用投資大眾普遍熱中追逐短線價差的心理，在市場資訊不足的情況下，故意發布利多
: 或利空消息，引誘投資大眾上鉤，並從中謀取不當利益，此一舉動即稱之為「內線交易」
: 。
: 另外，公司接獲或流失大訂單、新產品開發成功或失敗、營運出現重大轉機或虧損等，也
: 都是董監事、大股東以及經理人等最先知道，所以常聽到有人嘲諷說：「千線萬線（指技
: 術分析常用的移動平均線與趨勢線等）不如一條內線。」
: 內線消息當然管用，內線交易必然會賺錢，但一般投資人根本無處獲知內線消息，而且從
: 事內線交易乃違法行為，遭逮獲者依法除必須歸還價差利益之外，還會被課以刑責，最高
: 可處2年以下有期徒刑。
: 再來談談股市高手。我把股市裡面真正的高手區分為下列3個等級：
: 一、初段高手
: 此一等級的高手深諳道氏股價理論與艾略特波浪理論，熟知班傑明．葛拉漢（Benjamin G
: raham）的價值理論，精通羅伯．愛德華（Robert D. Edwards）與約翰．麥基（JohnMage
: e）的種種技術分析，熟讀過《股票作手回憶錄》（Reminiscences of a Stock Operator
: ）與《傑西．李佛摩股市操盤術》（How to Trade in Stocks），他們有能力在每次的股
: 價循環中逮到底部，大膽買進，而且抱牢持股賣在頭部。
: 二、中段高手
: 此等級的高手比初段高手還要高一級，他們非但能逮到底部大膽買進，抱牢持股賣在頭部
: ，而且有能力在空頭行情中抓到B波的高點狠狠放空，多空兩頭都賺。
: 三、高段高手
: 此等級的高手比中段高手更高一級，乃是高手中的高手，他們有能力研判出每一個波段（
: 指波浪理論8個波段的每一個波段而言）的高低點，第1波做多，第2波放空，第3波做多，
