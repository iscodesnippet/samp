武漢肺炎美國淪陷 CDC證實華盛頓州出現首例

2020-01-22 04:01聯合報 華盛頓記者張加／即時報導

美國聯邦疾病防治中心（CDC）21日宣布，美國華盛頓州出現武漢肺炎首例，但目前對美
國公眾的風險仍低。

CDC於美東時間21日下午舉行電話記者會，由呼吸道疾病暨免疫中心主任梅森尼爾（
Nancy Messonnier）與華盛頓州的代表回答問題；梅森尼爾指出，現已確認武漢肺炎首例
，也預期在美國和全球會出現更多病例。

梅森尼爾表示，基於過去的經驗，CDC知道做好準備至關重要，此事肯定引起關切，但CDC
仍認為，武漢肺炎目前對美國公眾的風險仍低。

根據CDC記者會提供的資訊，一名30多歲男子於1月15日進入西雅圖，16日自行向醫生通報
，相關機構隨即聯繫CDC，男子目前狀況良好，也願與相關機構合作；這名男子近期到武
漢旅遊，並未到武漢郊區的海鮮市場，據他所說，並不認識感染武漢肺炎的人士。

CDC於17日啟動在紐約甘迺迪機場、洛杉磯國際機場和舊金山國際機場，篩檢來自中國大
陸武漢的旅客；梅森尼爾21日表示，本周將在亞特蘭大國際機場與芝加哥歐海爾機場也啟
動篩檢機制，同時將旅行健康提醒，從第一級升至第二級。

醫生去年12月發現武漢郊區一個海鮮市場的工作人員和顧客陸續出現由病毒感染的新型肺
炎，引起發燒、咳嗽和呼吸困難等症狀；武漢的肺炎疫情病原體初步判定為新型冠狀病毒
，但該病毒不同於以往發現的人類冠狀病毒，2020年1月12日世界衛生組織（WHO）將「武
漢肺炎」正式命名為「2019新型冠狀病毒」（2019-nCoV）。

武漢肺炎不斷擴散且有人類死亡案例，目前中國大陸已確診共317例，死亡6例；周邊國家
泰國確診2例、台灣、日本、南韓各確診一例。

18:18 2020/01/21 中時電子報 盧伯華

(22日00:10更新）

大陸武漢爆發新型冠狀病毒疫情，連日來大量出現新增確診與疑
似病例，讓人目不暇給。由於不斷有新的確診與疑似病例出現，

: 甚至有可能取代軍事，而成為政治施壓的主要工具。
: 因此，日韓貿易戰對台灣是個警訊；獲得M1A2T戰車或F-16V戰機的安全感可能只是假象，
: 從經貿安全角度，台灣其實非常脆弱。
: 當前新型態的經貿戰爭主要是針對生產與銷售兩端，或者是切斷高科技產品的供應鏈讓生
: 產難以持續，亦或是徵收高關稅以打擊對手銷售。前者需要具有壟斷地位的原材料優勢，
: 例如大陸的稀土、美國的晶片或日本的精密化工原料，後者則需擁有廣大市場。台灣雖然
: 在全球供應鏈上地位舉足輕重，但這兩項優勢都沒有。
: 台灣的優勢是製造，透過進口加工或代工出口提高附加價值以創造財富。這使台灣成為典
: 型的貿易國家，2018年出口在GDP占比高達66.75%，進口56.31%。這種淺碟經濟對新型態
: 經貿戰爭幾乎完全沒有抵抗力，一旦貿易受限，台灣經濟就將陷入立即衰退的危機。
: 雖然在貿易戰威脅下，大陸電子組裝產能外移，台灣零組件出口也隨之轉移，今年1至5月
: 台灣對大陸（含香港）出口因而下降9.9%；但大陸還是台灣最大的貿易夥伴，出口占比仍
: 高達38.6%。這表示大陸如果對台灣提高關稅，將有超過1/3的出口廠商須考慮外移，否則
: 喪失大陸市場的競爭力。這在大陸維持惠台政策時自然不會，但台灣若在中美戰略競爭中
: 選邊美國，則即便大陸不報復，也無需繼續給台灣貿易優惠。
: 台灣經貿受衝擊
: 這指的是ECFA早收清單的免稅。當年反對ECFA的質疑，如大陸廉價農漁產品將充斥本地，
: 以及台商赴大陸設廠掏空台灣等，都並未出現；反而是台灣高價農漁產品得以免稅銷往大
: 陸，並以大量出超受惠。2018年台灣對大陸出超金額高達831.5億美金，而國民總生產也
: 不過6014億美金。
: 依據WTO規定，ECFA早收轉為FTA有10年大限，將於1年多後到期，而台灣對《兩岸貨貿協
: 議》與《兩岸服貿協議》卻遲遲不推，大陸有正當理由終止ECFA。若此，包括農、漁產品
: 在內的539項商品免稅優惠就將取消，衝擊台灣經貿的嚴重性可想而知。
: 台灣是貿易孤兒，僅與巴拿馬、瓜地馬拉等6個貿易小國簽FTA或經濟合作協議，今年1到5
: 月對新南向18國也僅出超61.7億美金，貿易利益主要仍來自大陸。當年高調反對ECFA，支
: 持太陽花反服貿運動的蔡政府，要致力維持ECFA早收確實有些尷尬。然而，台灣經貿安全
: 如此脆弱，政府卻一再以「撿到槍」概念操作「親美抗中」，令人憂心是否過頭。從日韓
: 貿易戰的啟示，屆時國人要擔心將不是軍事武力，而是貿易制裁。
