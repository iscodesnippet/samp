《生醫股》瞄準細胞治療商機，高端大型細胞製備中心下季啟用

15:232019/12/11 時報資訊 郭鴻慧

高端疫苗(6547)大型細胞製備中心(CPC, Cell Processing Center)預計將會在2020年Q1
正式啟用，配合細胞治療聯盟夥伴提出特管辦法送件，進攻細胞治療市場。


高端疫苗指出，CPC廠規格除了符合現今台灣法規要求的GTP，更以最高規格的PIC/S GMP
作為設計基礎，目前共配置12間細胞製備場所（CPU, Cell Processing Unit），其中9間
將供應自體細胞的生產，3間則作為異體細胞的量產製造。該廠具備完整的製造、QC、QA
能力，主要以CDMO委託開發製造服務為主，依客戶需求提供生產、製程開發與優化的服務
。

細胞治療已成為國際醫療趨勢，台灣在今年特管辦法開放之後，許多廠商與醫院積極合作
，向衛福部提出細胞治療技術申請，截至11/15止，衛福部醫事司有超過100件申請案在審
核中。


過去國內細胞治療產業僅有部分大型醫院與生技公司擁有小規模GTP實驗室，細胞治療產
能相當有限。高端疫苗說，在新法施行後、法規趨嚴的產業轉型時，對高規格的生產場地
需求大增，公司以專精的細胞培養技術、無菌製備操作及GMP相關經驗，直接以最高等級
的細胞製備中心取得產業先機，創造更多的市場合作機會。

高端疫苗的CPC廠位於新竹高鐵站附近，具絕佳地理位置，未來配合廠商與北、中、南部
的醫院進行合作，皆可在最短的時間內提供病患最高品質的細胞製備服務，有望吸引更多
廠商與醫院的合作意願。

(時報資訊)

2019 年「日本化」(Japanification) 現象已成主流，全球達到創紀錄的負殖利率債券規
模，市場經濟學家分析，2020 年，如此情況更將廣泛蔓延全球，屆時美國、英國與澳洲
等國家皆免不了進入負殖利率。

https://i.imgur.com/1PShCJK.jpg

美國
摩根大通長期投資策略高級顧問 Jan Loeys 認為，像這樣的經濟慘劇可能進一步蔓延，
甚至擴張到美國，屆時美國經濟可能只需再陷入一次「普通衰退」，即會屈服於零殖利率
。

上週，美國發布強勁的非農就業數據，暫時緩解人們對美國經濟衰退的擔憂，但該國的殖
利率曲線已顯示衰退情緒正在醞釀當中。

Loeys 表示，如果經濟衰退不幸成真，到時候聯準會可能效仿日本、歐元區央行，將利率
降至零再重新啟動量化寬鬆政策，而由於美國政治僵局和全球儲蓄過多的風險，多年來，
美國持續將債券殖利率保持在零左右。

為避免情況發生時無所防備，Fed 官員此前也提出情境模擬，聯準會理事 Lael Brainard
在上月表示若發生經濟衰退，導致必須將基準利率降至零的情況發生，聯準會將使用利
率設定上限 (Interest Rate Cap) 政策，即透過資產購買方式，將短、中短期債券殖利
率維持在一定水平。

野村資產管理的 Richard Hodges 則表示，Fed 可能別無選擇，因在財政政策出爐之前，

利率。

澳洲央行上週將基準利率維持在 0.75% 的歷史低位，市場認為，該國可能跟隨歐洲和日

: 分析師表示，這不應該責怪小米，而是華為禁售的效應。諮詢機構 Counterpoint 駐港分
: 析師 Flora Tang 指出，由於無法使用 Google 服務，華為必須將銷售重點轉回國內，且
: 由於民族主義高漲，中國民眾大買華為手機，不僅是國外產品滯銷連國內其他品牌也受影
: 響，尤其是小米。
: 原本就相當競爭的手機市場，華為反而成為其他中國品牌的巨大壓力，據數據顯示，在今
: 年第三季，小米僅銷售近 860 萬部手機，比去年同期降低了 38%。當然如 Vivo 和
: Oppo 也受到了打擊。也有其他報告也呈現同樣看法，據分析，Vivo 去年市占有近 22.6%
: ，而今年第三季僅有 17.9%，而 Oppo 的市占率也從 21.1% 下降至 17.4%，小米的市占
: 率從 13.1% 下降到 9%。
: 相較之下，華為的市占率從 24.9% 激增至 42.4%，基本上就是鯨吞了 Vivo、Oppo 及小
: 米的市占，中國市場逐漸成為一場零和賽局。其他如蘋果當然也有所影響，但原本在中國
: 就僅有 7% 的市占，如今衰退至約 5.2%。整體來講，中國手機廠商受傷甚重，華為出現
: 一家獨大的架勢。且中國已開始 5G 商轉，目前華為是唯一能自製 5G 晶片的中國手機供
: 應商，未來在國內優勢將會更加明顯。
: 其他品牌則計劃積極進攻國外市場，如小米已宣布將提前進軍日本。在歐洲和東南亞也是
: 如此，中國廠商逐漸強化布局，打算拿下華為失去的國外市場，如 Oppo 最近在曼谷開設
: 了第一家旗艦店，更計劃在馬來西亞、印尼、越南和菲律賓等大肆展店。當然搶得就是時
: 機，若順利的話，應還是可能彌補國內的損失。不過目前來看，營運仍在低谷。
: 3.心得/評論：
: ※必需填寫滿20字
: 出乎我意料之外 我以為華為會在國內跟中國製手機廝殺 結果沒想到在愛國心的支持下
: 華為一統中國市場 打得小米OPPO滿地找牙 不過中國市場就是這麼大 還可能萎縮的情
: 況下 其他中國製手機也只能出海外搶奪華為失去的市場 看起來小米股票似乎一路向下!
