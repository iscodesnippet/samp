新冠救命藥「奎寧+日舒」5天痊癒？　醫：數據好到驚人

TVBS王翊綺
發布時間 2020年3月22日10:16
更新時間 2020年3月22日10:16

新冠肺炎(COVID-19)疫情肆虐全球，目前全球逾萬人死亡，專家日夜研究救命解藥。日前
由美國藥廠研發的「瑞德西韋(Remdesivir)」被視為準新冠救命藥，目前台灣已開始進行
臨床實驗，不過法國媒體近日也曝光研究，奎寧(Chloroquine)加上日舒(Azithromycin)
，號稱5天可清除100%病毒！醫師透露，檢視了感染研究中心的臨床報告後，認為人類將
吹起反攻病毒的號角，直呼「數據好到不可思議！」

蕭捷健醫師於臉書上發文表示，這幾週以來，歐美國家疫情十分嚴峻，曾被寄予厚望的「
瑞德西韋」，最近發表的初期結論只展現50%左右的治癒率，跟標準治療相差無幾。至於
「奎寧」及「日舒」這兩類老藥則在新冠病毒中嶄露頭角。他指出，近日在法國的小型研
究中，包含沒有症狀者6人、有上呼吸道症狀者22人、有肺炎症狀者8人，「單獨使用奎寧
的病人，在第5天可使病毒量降到一半；使用奎寧和日舒的人，在第5天100%清除病毒。」
直呼如果大規模研究證實兩種老藥都有效果，「那真是踏破鐵鞋無覓處，救命藥就在倉庫
裡。」

吳偉愷醫師也發文說道，法國的小型研究臨床資料，雖然樣本數不高，但是結果卻令人鼓
舞。「這個結果對熟悉臨床研究變異個體性的人來說，是好到不可思議的數據，因此只要
數據足以採信，進行較大規模的臨床實驗成功率是相當高的。」

他並進一步說明，這次在法國馬賽大學感染研究中心發表的報告，作者是著名生物學家拉
烏特(Didier Raoult)，同時也是相當知名的感染醫學泰斗，其提供的研究數據通常都具
有指標性，因此法國政府立刻啟動30萬人規模的臨床實驗，美國FDA也同步跟進全面展開
研究。

多解釋只是口水之爭，

台灣人真的運氣很好，
2007/2008, 2020兩次風暴都能夠幸運度過受到很小傷害，
也跟這兩次的風暴剛好都遇到能力很強的政府有關係，
所以不知道這兩次風暴的嚴重性，
以這次為例，中國美國還有很多國家都已經封城，
台灣還可以很自由地在台灣國內到處走動遊玩吃東西，

也就是因為這樣，
很多台灣人都不知道這兩次風暴的真正嚴重性，
剛好我兩次風暴都有親身遇到，
非常了解嚴重性，
金融風暴當年，所有投資銀行瞬間全倒,
像是雷曼兄弟，Bear Stearns，
WaMu, GM也倒，一堆大銀行大公司通通都倒，

這次你到現在有聽到哪間大公司或是大銀行倒的？
疫情就算繼續一年，也不會有什麼大公司或大銀行倒閉，
只要大到不能倒的公司沒事，
那經濟體質就不會有太大問題

金融風暴當年畢業找工作，
即使史丹佛或是長春藤名校畢業，
也幾乎都找不到工作，
當年整整一兩年就業市場幾乎沒有職缺，
失業率是20%+,
一堆人自殺或是拿槍去公司掃射，
華爾街天天有人跳樓，

現在失業率還是非常低的3%-5%,
大部分的人還是在家工作，
一些被迫待在家裡的人，
也是一堆工作等著解封讓他們去做，
或是慣老闆強迫他們必須去公司工作，
所以失業情況真的沒有那麼嚴重

除了中國那種血腥手段的封城例外，
美國的封城跟台灣的防疫，
還是可以出門採買日常用品，
甚至因為預期心態而導致貨品被買光光，
零售業(尤其costco)跟網購的業績一定是比之前成長好幾倍，
另外現在不能出門吃飯變成吳伯毅的形式，
簡而言之，消費力大致上並沒有受到影響，
有些領域的消費力反而大增，
這次受傷最慘的就是娛樂業:NBA, MLB, 電影，夜店
大致上，民間還是有錢有消費力的

反觀金融風暴當年，
是瞬間全部的人都沒錢也都失業，
到處都空蕩蕩的，比現在還空，
餐廳沒人，電影院沒人，NBA沒錢買票，
一堆法拍屋但是沒人買，你現在有看到法拍屋嗎？
銀行不願意借錢出去,

除非彗星撞地球或是第三次世界大戰，
不然不可能有比金融風暴更慘的情況,
那是系統性全世界的人都在犯蠢好幾年導致的結果


※ 引述《skyprayer (SkyBotany)》之銘言：
: 第一場風暴是疫情風暴
: 這次疫情擴散 原本影響範圍僅在東亞
: 但短短一個月 世界主要經濟體都被捲入其中
: 疫情帶來人們生活和生命品質的減損
: 也對各國公衛和醫療體系產生巨大衝擊
: 伴隨而來的是緊張 恐慌 甚至是互相猜忌
: 第二場緊接而來的是經濟風暴
: 封鎖國境造成國際商旅直接重創
: 除了旅遊外 許多重要的跨國商業活動也都停擺
: 國際大型城市的封鎖 禁止集會活動 店面場所封閉 在家上班等措施
: 造成大量服務業停止服務  除了日常用品 防疫用品和儲備食品外
: 多數消費性商品的需求量急凍
: 許多對未來佈局的預期消費或資產佈局更是大幅取消或延期
: (小至婚宴  大至商業區的興辦計畫)
: 在家工作對已有此工作模式的產業和公司 衝擊相對小
: 但多數國家和產業 在家工作並非已建立的模式
: 因此產值和效率恐怕也會在初期大受影響 甚至有些公司直接選擇裁員
: 當工作受影響或威脅時  消費也會縮減 形成惡性循環
: 更慘的是 疫情不減  就算貨幣寬鬆  利息降低  幫助也有限
: 大量的貨幣流通 反而會造成市場更大動盪 甚至某些區域禁入惡性通膨
: (可能會看到惡性通膨 惡性通縮同時發生的奇觀)
: 金融海嘯和歐債危機很大一部分是靠量化寬鬆處理的
: 本質上的缺損和缺陷實際上並未完全修正和修補
: 到處流竄的熱錢和金融創新的發展
: 不知不覺又創造許多看起來不錯 實際上風險難以評估的金融商品和投資模式
: 舉例來說在台灣  這幾年大量增長的投資型保單
: 投資型保單之所以能在一定日期後給予報酬 仰賴保險公司的投資能力
: 在這種極端的情況下  多數資產投資瞬間轉成負報酬
: 這樣的風險損失  雖然可能還是由保戶承擔
: 但對此商品的不信任和怨恨將會大量增加
: 保險公司可能在短期內遇到大量解約和客訴
: 更慘的是  很多保險公司業務和主管 自己也抱了滿手......
: 除此外 新興ETF 網路跟單 智慧交易這些東西
: 都可能在當初設計時 未把如此規模的慘況列入考量
: 或是同時把某些判斷條件交由電腦決定 造成短時間大量同向自動交易
: 產生更難預料的動盪  (這或許可以解釋為何會如此高頻率的熔斷發生)
: 當動盪越來越大 或是突然平復  可能造成某些商品違約和崩潰
: 金融市場更詭異的骨牌秀發生可能性大增
: 第三場伴隨而來的是政治風暴
: 民主國家的政權除了忙於防疫和挽救經濟的同時
: 也忙著挽救自己和所屬政黨的聲望
: 有時反而讓防疫和挽救經濟脫離專業判斷
: 民眾的意見在這種需要特殊專業判斷的非常時期
: 有可能扮演錯誤方向的推力  造成民主政府選擇附和民意而非專業
: 造成更大的錯誤決定
: 非民主國家在這種時候則會採用更強硬的措施
: 鞏固自己在動亂時期的政權
: 另外也急需把眼前的混亂推給其他政治團體
: 這樣的舉動可能會造成國際間的衝突和緊張上升
: 也有一些國家計畫趁這一波國際混亂的狀況中
: 打擊敵對勢力或趁亂獲取利益
: 這三個風暴如今捲成一團 互相強化 讓局勢更加混亂
: 已經超過一般人能做判斷和思考的能力
: 所以面對看不懂的局勢和盤面
: 只好學著竹林七賢模式  具在公園裡 飲酒高歌 寫詩做詞 躲避亂世
: --------
: 這次的風暴  影響會遠遠超過金融海嘯
: 甚至勝過經濟大蕭條
: 在金融海嘯發生的那時
: 很多日常商業活動還是照正常進行
: 受影響的是大量參與衍生金融商品交易的公司和投資者
: 還有數個大型集團的股東和員工
: 對於沒有投資這些商品的人  你去問他金融海嘯那年有沒有甚麼特別感覺
: 大概會有不少人還真的沒有印象
: 所以在聯合救市 救公司後  能在短短時間內脫離風暴造成的損害
: 但這次的風暴  無論是否有資本投資的民眾  生活上都有明顯改變和影響
: 市場的供給和需求明顯出現巨大變化  只要認真去觀察 這不難察覺
: 難的是去承認後面難以置信的指標和數據 真的是事實的一部分
