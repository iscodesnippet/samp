不要亂算好嗎..............

如果不喜歡券商除息扣成本 你可以找不會這樣扣的券商

我剛好就有用到一間 台新證券就是除息不扣成本的券商

而且其實就算除息會扣成本 你也可以自己調整改回

一切問題都可以輕鬆解決

※ 引述《tensionship (nothing)》之銘言：
: 正常來說，報酬率算法為=報酬/支出成本
: 不過我自己算的跟券商軟體顯示的不同
: 想請問正確的算法應該怎麼算
: 一、成本=股價*股數，除權息要不要從股價去除？
: 假設說我買一張股價100元的股票，在還沒除權息之前我的成本是100*1=100
: 今年公司很大方股息配50元，除息之後，過一陣子就填權息回100元
: 如果照我自己的算法是
: 資產100*1+50(配息)=150，
: 報酬=資產-成本，150-100=50
: 報酬率=報酬/成本，50/100=50%
: 可是按照券商算法，他會在除權息時，就把股價扣除，成本變成(100-50)*1=50
: 資產100*1=100，(因為除息時已從股價扣除，所以配息不算資產)
: 報酬=資產-成本，100-50=50
: 報酬率=報酬/成本，50/50=100%
: 明明報酬一樣，可是報酬率卻差很多

到這邊都還算合理 唯一不合理的地方是不會有公司配息50%還能填回去

所以你用這種例子去算感覺差很多 照一般配息率3~5%會填息的公司來說

根本沒那麼大的差距 算起來就是 3/100 報酬率3%

或者是 3/97  報酬率3.09%

只是計算方式微妙不同 實際上並沒有太大差異


: 二、拿到股利再投入，算不算成本？
: 還是拿上面那個做舉例
: 假設說我買一張股價100元的股票，在還沒除權息之前我的成本是100*1=100
: 今年公司股息配50元，除息之後，我拿50元的配息再買一隻，過一陣子就填權息回100元
: 我的算法
: 如果股利再投入，算成本的算法是
: 資產100*2=200
: 報酬=資產-成本，200-(100+50)=50
: 報酬率=報酬/成本，50/(100+50)=33%
: 如果股利再投入，不算成本的算法是
: 資產100*2=200
: 報酬=資產-成本，200-100=100
: 報酬率=報酬/成本，100/100=100%

不要亂算 33%是什麼鬼 不過你想怎麼算不關我的事

畢竟每個人的邏輯能力不一嘛

: 券商算法
: 如果股利再投入，算成本的算法是
: 資產100*2=200
: 報酬=資產-成本，200-(50+50)=100
: 報酬率=報酬/成本，100/(50+50)=100%
: 如果股利再投入，不算成本的算法是
: 資產100*2=200
: 報酬=資產-成本，200-50=150
: 報酬率=報酬/成本，150/50=300%

...........你自己要怎樣亂算就算了 券商的系統才不會這樣亂算

不要開玩笑好嗎? 股利再投入 如果這動作是你自己操作的話

當初券商如果配息有減成本 你另外再買入當然也會加上成本

成本怎麼算都是100 怎麼可能會有系統算成300% 一定是100%

股利再投入 如果是券商自動操作 你的成本自然不會有任何變化

就是當初原始成本的100 那也不用說 報酬一定也是100%

: 如果用我的算法，除權息不扣股價，股利再投入不算成本是正確的
: 如果用券商算法，除權息要扣股價，股利再投入要算成本是正確的
: 那平常記帳到底要怎麼記阿？


重點是 不管怎麼算 在合理配息%數的情況下 根本不會有太大差異

其實可以不用去在意這點0.0X%的數字

如果真的很在意的話 可以自行調整成本 讓他算的跟你想的一樣

也可以從各家券商內挑出一間你最喜歡的算法

Brand, distribution, production, balance sheet and library.
Brand- 毫無疑問, Disney 全球品牌前五十, 廣受年輕人喜愛, 擁有很多熱門IP. VIAC有

比率是最高的, 都接近30%。從表面上看，這可能是一個風險因素，因為該人群中的電視
流失率最快。但是兩家公司都建立了具有大眾市場吸引力的流媒體服務，
例如Disney +和CBS All Access，可以進行導流.

Distribution- 影響力, 其他管道的分銷協議, 電視/流媒體生態系統之外的戰略合作夥
伴關係.

Production- 是否能產出有經濟效應和吸引力的內容, 在這個內容為王的時候. 根據統計
, original TV programs是一個吸引用戶的的重大因素; 如同持續不斷推出的Netflix
Originals所帶來的效果, 超過一半的用戶在Netflix上看的是Original TV program &
movie, Netflix的排行榜上有6成是Netflix Originals. 而DIS和VIAC也有足夠的硬體設
施來進行原創內容影集的拍攝與製作, 這種具有DTC服務, 內容生產能力的公司會有更多
能力來保護內容並從中獲利(EX: content licensing)

Library: 擁有電影和電視工作室的公司擁有最豐富的內容庫, Disney就不用說, CBS也有
超過15000小時的電視和電影內容, 未來還會持續投入同等時長15000小時的製作. 很多用
戶都會基於以前的熱門電影和影集而去訂閱該串流平台的會員服務. 但同時, 獲取這些熱
門影集的授權也越來越貴, 例如Netflix 2019 續約Friends花了$80mn-$100mn支付給
WarnerMedia, 2015才$30mn; 同樣的, Hulu為了South Park要支付給ViacomCBS的也越來
越高, 2019 HBO Max支付給ViacomCBS的價格為$105mn.

最後, balance sheet: 製作內容節目是非常燒錢的, 所以對於資產負債表和現金流有較

 標題  [心得] 技術分析沒有「盲點」                                            
 時間  Sun Aug  2 09:03:13 2020                                               
───────────────────────────────────────

這篇文章的標題，是有一點點反話。

先說說，筆者為何要寫文章好了。其目的只有一個：「反思」。為了反思自己的心得、技
術與能力，寫在網路上供人評斷、參考與批評，對我來說，或許會有一點點好處。就我的
觀念裡，交易是一件很孤獨的事情：不能跟人分享自己的成績，因為這毫無益處；不須跟
人分享自己的看法，因為這沒辦法給人幫助；不須寫文章給人參考，因為這是冷暖自知的
覺悟。寫出文章，只是對自己的反思，看看邏輯通不通，字句順不順，至於別人能得到什
麼「助益」？那就是上帝的事情了。

技術分析，這邊單指「形態學」：是有盲點的。所謂的「盲點」，就是會不知道現在市場
到底是要上？還是要下？「股價走勢」上下飄忽不定，看不懂！

會這樣！非常正常。

「可能」、「或許」、「應該」、「這樣機率大」，是在看線型圖時，一定會產生的「心
理狀態」。這邊就提出一個原則：「看不懂，就不要操作。」

我非常反對短線交易。尤其是：「持股時間不超過三個月」。在我的觀念裏，只要不是專
業的投機客，基本上沒有任何理由短線交易。一個趨勢的發展，需要醞釀，在準備的過程
中。內線人士會先收集籌碼，然後甩轎，當籌碼夠多了，發動攻勢，市場上不明白的人被
騙進去，股價炒高後，「利多消息發布」，更多人進場搶「便宜」，接著內線人士把貨全
部倒給散戶，大賺一筆後，回家數錢，買豪宅。再找下一個目標來炒股票賺錢。這樣的一
套劇本流程，幾乎都要「三、四個月左右」。一直不斷的發生。

這時候「形態學」的幫助，其實不是很大。真的要靠「直覺」跟「內線交易」，不然就是
被人坑殺。（當然，還是可以試試看。看看手腳快不快。）

不管是技術分析或基本分析，還是「籌碼分析」，或占星術，都沒有辦法「準確地買在最
低點，賣在最高點」，都只能夠抓「相對高點，相對低點」，靠著經驗出場。

像是：「矩形」（這種排列的特徵：是價格在線型圖上，來回擺盪，上下波動在兩個價位
間。）假設在60跟40元間，價格到了60就上不去，會回落；當掉到40時，就有許多買盤，
把價格往上推。這種圖形蠻常見的。代表有兩股勢力，一邊覺得60塊差不多了，覺得夠了
；另一邊覺得40塊，非常有價值、便宜，有多少收多少。在還沒有「突破」（超過60）或
「跌破」（掉到40以下），沒有公司內部消息的散戶，是不可能知道詳細情況的。忽然某
一天，一根長紅K線，突破60塊，來到65元，「接著劇本上演」在新聞上看到「什麼大單
、什麼技術突破...」，到那個時候，已經慢了好幾步了。

歷史一直都是這樣演的！從來沒有改變過。只是名字不同、價位不同罷了。

金融市場，是一個「自由競技的世界」。發生任何事情，都一定有他的理由。每個人都只
能夠在自己的「人脈、認知、性格...」整合這些「資訊」後，「行動」（不管是精明、
或愚蠢）然後產生「結果」。

「看不懂，就不要操作。」線圖是非常公開、透明的，每個一參與市場的人，都可以看到
，起跑點是一樣的。

如果價格來到自己判斷的「壓力」點位，獲利了結跟「放空」，就是自己的應該要做得事
情。當然，這需要勇氣跟「明智」。

感謝閱讀！今天這篇，就簡單寫這一些。希望未來能有更多反思。
