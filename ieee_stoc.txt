 標題  [新聞] 美國權威機構《消費者報告》初評 Model Y                          
 時間  Tue Jul 21 12:42:39 2020                                               
───────────────────────────────────────

1.原文連結：https://bit.ly/2CUL5Ek

2.原文內容：

美國權威機構《消費者報告》初評 Model Y：一部空間更大的 Model 3 進化版，但仍有
一些缺點未改善

不被廠商廣告左右的美國消費品評鑑機構《消費者報告》(CR：Consumer Reports) 最近
開始測試 Model Y，完整報告還沒出爐，但上週 CR 率先發布了第一眼印象報導，由團隊
成員先用輕鬆的對談方式，互相抒發對於 Model Y (LR AWD) 初步體驗後的心得。

鑒於在座之中許多人未來都可能成為 Model Y 車主，我們就先摘錄幾個重點，一起聽聽
CR 團隊怎麼說？

有什麼優點？

Model Y 保有了過去的優點，速度很快、充滿樂趣、搭載很先進的行車科技，駕馭起來也
就是部性能卓越的運動車型。
Model Y 就像一部改成五門掀背版的 Model 3，頭部空間更大、後座更好坐，後車廂也更
大更好用，可以說是在 Model 3 優異的基礎上再改進一些缺點，就是一部進化版的
Model 3。
儘管 Model Y 配備著較粗的 A 柱，但 CR 團隊認為駕駛獲得的行車視野相當清楚，這片
大範圍的前擋玻璃有很好的表現。
團隊成員對稱呼 Model Y 是 SUV、五門掀背或是跨界車都不是很在意，因為本質上它就
是一部車，大家只要知道它節能、空間更大即可，如何認定 Model Y 是某一種車型都無
所謂。


缺點是哪些？

Model Y 改進了一些 Model 3 的缺點，但沒有全部改進，例如中控螢幕，CR 團隊認為特
斯拉把所有功能都簡化到中控螢幕的作法是太過頭了。
承上，所有功能都必須去螢幕選單裡面找是很可笑的作法，連設定後視鏡、冷氣風量或吹
風方向都必須從螢幕點選，不算是人性化的設計。
卡片鑰匙開鎖 OK，但是啟動車輛要擺放的位置太奇怪，很多車主都弄丟過這卡片鑰匙，
或是不小心讓卡片滑落到座椅間隙下面，而 Model Y 絲毫沒有改進這一點。
作為一部應該很安靜的電動車，Model Y 卻不是一台很安靜的車，因為例如胎噪和風切聲
都很容易在車室內聽到，這個困擾是 Model 3 所擁有的，但還是傳承到了 Model Y 上。
特斯拉將行車安全相關的警示提醒都設計在中控螢幕是違反駕駛需求的。舉例來說，當準
備左轉，駕駛目光會看向左側、左前與左後方，但特斯拉的所有警示訊息都是顯示在反方
向的螢幕內，這樣會讓駕駛感到很困惑，到底是該看左邊的路面或是右邊的螢幕？駕駛注
意力反而因此被分散了。
頻繁的 OTA 不斷推出新功能，但每次的推送又改變了駕駛操作的方式。例如最近在美國
推送的「紅綠燈識別控制」，第一階段是強制在路口停止 (前行需要下指令)，現在又讓
車輛可以在綠燈時跟車前進 (前行不用下指令)，車主可能還在學著搞懂第一階段操作方
式，但特斯拉又在下一次的軟體更新改了規則。
特斯拉 Autopilot 系統在現階段仍不夠安全，尤其當中又有很多 BETA 測試版功能。特
斯拉的行銷方式容易讓車主誤以為 AP 和其他輔助功能很值得信賴，但事實上這些功能都
無法完全保證；即便發生事故的機率只有少少的幾趴，但若真的遇到，就很可能是一場嚴
重的慘劇。
鑒於 Model Y 還是很新的產品，市場上累積的數據還不夠多，此刻無法對車輛可靠性下
定論。



從這不到 30 分鐘的對談之下，我們可以很深刻地感受到 CR 團隊對特斯拉是不吝褒貶，
大家或許無法認同他們所有的觀點，但我認為至少在就事論事上面，CR 團隊是沒有問題
的。

如果對全部的內容有興趣，可以觀賞 CR 發布的完整影片，大概只有 20 多分鐘而已：

3.心得/評論：

簡單來說，Model 3 有的缺點 Model Y 一樣沒少

優點則是更大更省電(續航里程更高)

整體價格比 Model 3 高一些


上周新聞主要是 Model Y 的訂單有掉
