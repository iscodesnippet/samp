我盡量用我自己理解的來跟大家說

首先，要知道選擇權的結帳方式

選擇權有分週選和月選

週選是每個禮拜三結帳

月選是每個月的第三個禮拜三

再來，選擇權有分日盤跟夜盤

日盤開盤時間：08：45-13：45
夜盤開盤時間：15：00-05：00

像下圖就是選擇權選擇週選和月選&日夜盤的選項
https://i.imgur.com/2ORkaI9.png

202008W1代表8月第一週選擇權

每個禮拜三過了會出現下一週

202008月代表8月的選擇權（結帳時間是第三個禮拜三，所以第三週的週選也是這個月的月
選）

202008月全，這個全代表日夜盤都有，要玩夜盤就要買這個，晚上也會動

https://i.imgur.com/qL5zy8U.png

上圖就是選擇權的頁面

左邊call是看漲，右邊put是看跌

中間的12xxx是履約價

橘色的12700叫做價平（意思是跟現在台指大盤點數最接近）

其他的分價內跟價外

假設以履約價來看，call方12500代表已經漲過了，所以12500是價內

put方法12500還沒跌到，所以是價外

價格要看成交價那一格，需要保證金買方最簡單計算就是*50

我們看12700 call方成交價是190，代表你的期貨帳戶裡面要有190*50=9500元才能夠買一口

價內價格較貴，不過比較保險一點，因為如果結帳時間到了，你買在價外，通常下場就是價
格變成0.1，就是一口5元

雖然價平、價內跟價外是按照台指大盤的點數來決定，但是選擇權的成交價格取決於台指期
貨點數（夜盤只能看這個決定漲跌）如下圖

https://i.imgur.com/rUmGola.png

獲利方式：買賣call或買賣put

我建議新手都玩買方就好，買方單純就是買call和買put，買call只要漲了，你就會賺錢，
買put只要跌了你就會賺錢，相反就會賠錢，最多就是本金歸零

: : 【選擇權定義】
: : 一種可以交易的權利，買方跟賣方簽好契約，內容包含履約時間、標的物、履約價、數
: 量
: : https://i.imgur.com/kByndLF.jpg
: : 舉例一
: : 【買買權(Buy Call)與賣買權(Sell Call)】
: : 現在豬肉一斤 100 元，小明與小華約定好 8/1 小明要用一斤100元的價格跟小華買豬
: 肉
: : ，可以買10斤。
: : 履約時間： 2020/8/1 、  標的物：豬肉、 履約價：1斤100元、 數量：10斤
: : https://i.imgur.com/f9zl7UE.jpg
: : 雙方同意後，小明就給小華一筆保證金契約就成立了。
: : 這筆交易背後的概念是：
: : 小明視角：
: : 小明認為未來豬肉價格會長，所以現在先跟小華買了未來可以用100元買豬肉的權利
: : 這就是選擇權的「買買權」
: : 假設 8/1 豬肉上漲到200元，小明就能根據合約，用100元跟小華買到一斤豬肉，賺
: : 到價差100元x10斤=1000元
: : 假設 8/1 豬肉下跌到50元，小明不需要履約，就賠的是給小華的保證金。
: : https://i.imgur.com/8Ejtffr.jpg
: : 小華視角：
: : 小華認為未來豬肉價格會跌，所以現在先賣小明買進的權利
: : 這就是選擇權的「賣買權」，什麼都不用做，賺取保證金
: : 假設 8/1 豬肉上漲到200元，小明就能根據合約，用100元跟小華買到一斤豬肉，等
: : 於小華要賠給小明1000元
: : 假設 8/1 豬肉下跌到50元，小明不會履約，小華的就賺到了保證金。
: : https://i.imgur.com/iGgE2Oe.jpg
: : 舉例二
: : 【買賣權(Buy Put)與賣賣權(Sell Put)】
: : 現在同樣的豬肉一斤100元，小明與小華約定好 8/1 小明要用一斤100元的價格賣給小
: 華
: : 豬肉，可以賣10斤。
: : https://i.imgur.com/1wzCkk5.jpg

: : 小明認為未來豬肉價格會跌，所以現在先跟小華買了未來可以用100元賣豬肉的權利
: : 這就是選擇權的「買賣權」
: : 假設 8/1 豬肉上漲到200元，比他預期的高，他就不需要履約，就賠的是給小華的保
: : 證金。
: : 假設 8/1 豬肉下跌到50元，小明就能履約，用 100元 的價格賣豬人給小華，獲利就
: : 是(100-50)x10=500
: : https://i.imgur.com/ZtdTmIa.jpg
: : 小華視角：
: : 小華認為未來豬肉會長，所以同意未來用小明一斤100元的價格賣豬肉給小華
: : 這就是選擇權的「賣賣權」
: : 假設 8/1 豬肉上漲到200元，小明就不履約，小華就賺到保證金。
: : 假設 8/1 豬肉下跌到50元，小明就要履約，用 100元 的價格賣豬人給小華，賠的就

: : https://i.imgur.com/Mn3bqsM.jpg

通站在同一陣線，一起說服美國政府同意賣高通晶片給華為，以突破禁令關卡。

2020年7月29日，美國晶片大廠高通(Qualcomm)公布2020第三季會計年度財報，宣布已經
與華為(Huawei)簽訂新的長期專利授權協議，受到此消息激勵高通股價飆升12%。至於協
議細節未揭露。

到底這一筆新的長期專利授權金是多少？

根據高通財報資料，今年截至目前9月底，若不包括某些項目的收入約在55億至63億美元
，分析師平均估計為57.7億美元；若包括來自華為的未付款在內，則營收將在73億美元至
81億美元之間。因此，估算這筆華為未付款，可能就是新的長期專利授權金，應該在18億
至23億之間。

高通預估公司在第三財季，由於新冠疫情大流行，重創手機的銷售，高通第三季的營收較
去年同期幾乎大減50％，達到48.9億美元，凈利也同時下降61%，降至8.45億美元。高通
代表美國電信業景氣指標，它若不好，美國經濟會回溫嗎？

同時，高通也透露期待下一代蘋果iPhone之5G旗艦手機，可能會在今年稍晚時將首次亮相
。因而，帶動高通的5G晶片成長。之前，蘋果支付高通專利和解金45億美元以支付兩年專
利費，有此可見每年約20億至22億美元之間。這與華為支付的新授權協議很接近。

高通反壟斷案與專利授權金計算

由於，高通公司在通信晶片行業的優勢地位，既可從專利授權而獲得大筆授權金，這些專
利涵蓋了許多通信系統的基本專利技術，使得眾多手機製造商必須按每部手機售價支付一
定比率的費用給高通當做是過路費。同時，高通也賣自己設計的行動通信晶片給業者放入
產品中。高通表示，已與所有主要手機製造商簽署了專利授權協議。

高通與與華為曾於2019年6月先簽署短期授權協議，華為須向支付高通每季1.5億美元(每
年約5~6億美元)專利授權使用費，藉此達成暫時和解協議。

當然，企業營運若過於利用行業的優勢地位，很容易在不知不覺中落入反壟斷困境。不例
外，高通的知識產權的覆蓋範圍及其所產生的利潤已經在全球招來了法律和法規方面的挑
戰。

高通在美國反壟斷案，2018年法官已裁定高通必須授權通訊晶片給競爭對手，包括：英特
爾、三星等。也就是說，高通必須遵守FRAND承諾，以公平合理且無歧視的條款提供授權
條件給競爭對手。至於，專利授權金部分，法官說5％授權使用費不符合FRAND原則的。因
為，每家業者狀況皆不相同，如何每家就以5%來定價。所以，法院對高通的專利授權金判
定仍未決。

結語

高通與華為達成的長期授權及和解協議，主要針對高通無線技術專利授權供華為使用。至
於，高通能否出售驍龍晶片給華為，目前仍然不清楚美國政府的意向。如今高通已與華為
達成和解，並簽下一項長期專利協議，預期下一季將可進帳近 18 億美元的和解款項。這
對華為而言，或許是先放下身段示好，同時換得高通為取得這筆高額授權金，應該會協助
華為突破美國政府關卡而努力。

近來某一個小農媒體，以大量空拍照試圖營造太陽光電滅農的形象。搭配近來農委會有意

注目，許多人煩惱，如果連綠能都不能發展，人類究竟該用哪一種方式發電？

其中最困擾的莫過於蘋果供應鏈的科技業，蘋果手機的供應商必須使用一定比例的綠電，
但偏偏台灣的綠電才剛起步。以台灣設置最多太陽光電的台南為例，兩度引起Google關注
，不僅買綠電，還買地設資料庫。

都會在南科園區。此外，不僅跟政府買電，台積電也計畫投入600億元在屏東，興建太陽
能電廠。這些科技大廠對綠能的渴求，不只來自於企業社會責任(CSR)的要求，也是確保
注綠能，如今已是各種政府綠電說明會上的常客。回到綠電是否與農業衝突的問題，當國
外流行實施各種離農政策補助，台灣的老農卻有種電的選項，用最不破壞農地的方式獲得
固定回饋，徹底避免農地蓋水泥豪宅這種不可逆的破壞。

的案例。就連農委會都公布實驗數據，太陽光電不僅對文蛤養殖影響有限，還可避免冬季
寒害。

？太陽光電仍然是這個星球上，最普遍、最不影響地形地貌、最不影響生態的發電方式。

效減少碳排，幫助彼此渡過地球暖化危機。
