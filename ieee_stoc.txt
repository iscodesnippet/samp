5個月只賣43輛 納智捷品牌傳退出大陸市場

大陸車市低迷，台資裕隆集團傳出縮編、退市消息。每經網8日報導，裕隆集團計畫新一
輪縮減開支和轉型，並已經與東風汽車集團有限公司（簡稱東風汽車）就納智捷（Luxgen
）品牌退出大陸市場達成基本共識。
東風集團消息傳出，股東雙方正在探討東風裕隆的發展，目前沒有解決方案。至於新版發
展規畫何時發布，目前尚不清楚。繼東風雷諾之後，東風系又一合資公司也恐將黯然離場
。

東風裕隆成立於2010年12月，由陸資東風汽車公司與台資裕隆集團合資，分別持股50%。
其生產基地位於杭州，主要生產銷售納智捷品牌產品。產品年銷量在2015年達到巔峰6萬
餘台。2018年納智捷銷售跌破萬輛紅線，全年只賣出7,056輛，2019年銷量進一步下滑至
1,947輛，今年前五個月累計銷量僅43輛。

另，東風裕隆日前突然換帥，其總經理吳新發因個人原因申請退休，由現任裕隆集團大陸
總部總經理陳金泉接任。吳新發是東風裕隆的創建者之一，此時離開也引發諸多聯想。

現年61歲的吳新發已在裕隆集團工作37年，先後出任裕隆汽車製造公司業務部經理、裕隆
日產汽車公司總經理等職務。

由於涉及客戶服務和終止共同項目的法律程序，東風裕隆股東雙方仍在就納智捷退出大陸
市場商談細節。

納智捷官網顯示，旗下品牌有URX、U5 EV、U5 SUV等共八款車型，但僅有U5 SUV、優6
SUV和URX仍在銷售，其餘車型均停售。而在售車型中，僅有「國五」版本。大陸市場都在
向「國六」排放標準切換的狀態下，納智捷卻缺席。

: 基本上糖尿病足部潰瘍（以下簡稱DM foot）之所以麻煩，是因為傷口不容易好，
: 而為什麼不容易好呢？
: 因為糖尿病患者本身的血管狀況大多不好、神經狀況也不好，
: 而其中「血管狀況」才是治療成敗關鍵，傷口本身的照護反而是次要因素，
: 血管狀況很差的，你每小時專人換藥也沒用
: 然後我們看一下ON101的研究設計
: https://reurl.cc/D9VyGO
: 它的研究設計已排除了血管狀況不佳、傷口很爛的、本身營養不良的
: 這其實是一般研究設計的常態，會設計成「相對理想」的情況，
: 但拿到現實情況就是未知，
: 所以通常上市後還會有更多的實驗、尤其是非藥廠贊助的實驗
: 簡單來說，ON101在血管狀態不錯的DM foot有不錯的成績，
: 但大部分DM foot血管其實都是很爛的
: 然後這實驗中拿來對打的aquacel 本身也不是什麼糖尿病足部潰瘍的專用敷料，
: 它只是臨床上滿常用的一般傷口照護敷料，
: 就是吸水後會變成凝膠狀，維持傷口濕潤，讓組織較容易生長，

警察要求網絡信息平台移除涉嫌「危害國家安全」的信息的權力。

7月7日，多家科技公司宣布，將暫停處理港府要求分享用戶信息與數據的請求，其中包括
臉書（Facebook）、谷歌（Google）、推特（Twitter）、微軟（Microsoft）旗下的領英
（LinkedIn）、網絡會議技術公司Zoom和即時通訊軟件Telegram的母公司。

蘋果公司發言人表示，該公司正在對新法細則進行評估。

英國《金融時報》報導說，已經有至少一家西方大型科技公司正在考慮包括撤離在內的所
有選項。報導沒有指出具體是哪家公司。

數據隱私專家、加拿大全球隱私與安全設計中心（Global Privacy & Security by Desi-
gn Centre）執行總監安卡沃基安（Ann Cavoukian）對美國之音說：「我的理解是，臉
書和谷歌現在都在考慮離開香港……因為它們知道，它們掌握的所有用戶信息都可以被中
國當局獲取。沒有人希望這樣，臉書的用戶當然不希望這樣。」


不與港府合作只是暫時做法

中國政府則試圖通過「釋法」來打消公眾對於「港版國安法」箝制言論自由的擔憂。

有中國中央政府官方背景的「香港邁向明天關注組」6月末發表《香港國家安全立法釋疑
（20問），其中一項「釋疑」條文說，「港版國安法」出台後，「香港市民可以如常使用
Facebook、Instagram、Whatsapp等社交媒體和通訊工具，通信自由、言論自由等權利完
全可以得到保障」 。

「釋疑」加了一句：「國安立法只針對極少數人和少數幾個嚴重危害國家安全的行為、活
動，對大多數公眾沒有影響。」

言論自由倡導者擔心，北京對香港的控制愈發嚴厲，「港版國安法」的「國安」定義涵蓋
範圍之廣，可能讓任何敢言民眾都成為監管者眼中的「少數人」。

國際組織人權觀察中國部資深研究員王松蓮對美國之音說，「港版國安法」規定了一套「
在以往的體系中不存在的政治罪名」，並「鼓勵警方動用寬泛的權力來調查這些罪名」，
從根本上顛覆了香港的法治。

分析認為，西方網絡科技公司拒絕與港府合作只是權宜之計。

王松蓮美國之音說：「可以想像，一旦當局要求他們提供信息，他們這種（拒不合作的）
立場會使他們與當局發生衝突。」

她繼續說：「他們的法定代表人和僱員可能會因為不遵守法律而受到懲罰。」


警方可繞開法庭直接搜查網絡資料

臉書公司公佈的統計數字顯示，該公司在2019年下半年共收到香港政府的241個分享數據
請求，臉書只配合提供了其中46%的請求。

臉書公司網站發表的官方聲明說，「Facebook根據適用的法律和我們的服務條款回應政府
的數據請求。我們收到的每一個請求都經過仔細審核，以確保其在法律上充分性。對於過
於寬泛或模糊的請求，我們可能會拒絕，或要求提供更多細節」 。

數據隱私專家卡沃基安曾擔任三屆加拿大安大略省信息與隱私事務專員。她說，在美國等
西方國家，如果執法機關希望從臉書這樣的社交媒體平台獲取用戶的數據，必須有合理依
據，證明調查的必要性，並必須從法官取得法庭的授權令（warrant）。

卡沃基安說：「他們（政府）必須經過這個司法程序，這非常重要」

而在「港版國安法」的規定下，警察在秘密審查和刪除網絡內容時可以繞開這樣的司法門
檻。法律界人士注意到，與其他和截取信息有關的法律規章和國際普遍做法不同的是，在
新的法律規定下，警方毋需獲得法官的批准。

實施細則說，如果警務處長有「合理理由」懷疑電子平台上發布的信息「相當可能構成危
害國家安全罪行」或「會導致危害國家安全罪行的發生」，可在保安局局長的批准下，授
權警務人員要求發布者、平台服務商、主機服務商以及/或者網絡服務商移除相關信息，
限製或停止任何人接達該信息，或限製或停止任何人接達這一平台或相關部分。

人權觀察組織的的王松蓮特別指出這部法律加大了警察秘密監視的權力。她說：「在使用
秘密監視方面，這部法律實際上允許警察在不申請法官司法許可的情況下進行一些這樣的
活動。」

王松蓮還說：「所以我可以想像、我們也希望這些公司正在進行一種全面的評估，以保護
用戶的權利，同時保持讓用戶獲得他們的服務，並採取減輕（影響）的措施，在確保用戶
的權利的同時降低自身運營的風險。」


王松蓮：網絡設限只是時間問題

中國大陸「網絡防火長城」外的香港一向秉承言論自由、通訊自由，而「港版國安法」則
首次以立法形式明文規定，當局有權以網絡平台信息危害國家安全為名，要求主機服務商
或網絡服務商（ISP）禁止任何人接達境內外網站。

王松蓮說，香港從未有過任何要求審查的法律和製度，所以任何開此先例的做法都會受到
廣泛的關注和反對。

她特別提到香港「反送中」大規模抗議活動發生之後，警察對通訊軟件越來越嚴格的管制
：「在過去的幾個月裡，香港讓警察實際上刪除了一些人的Telegram頻道內容，所以我可
以想像，他們開始採取行動切斷對某些網站的訪問，只是時間問題。」

這意味著，即使臉書、谷歌這樣的外國科技公司將業務遷移到香港之外，或者將服務器移
出香港，香港當局仍然可以要求本港網絡服務商切斷對某些網站的訪問。

《金融時報》說，谷歌、亞馬遜雲計算服務（Amazon Web Service）以及微軟都在香港設
有數據中心，臉書和推特則沒有在香港的本地數據中心。

王松蓮還指出，外國公司考慮離開香港，擔心的不只是「惡法」的使用，還是對美中關係
大環境惡化的擔憂。

「美國因為（中國）侵犯人權或不正當競爭而製定的任何懲罰中國政府的措施，都可能意
味著這些美國公司成為報復的目標，只是因為它們是美國公司。」

另外，雖然中國大陸禁止普通網民使用谷歌和臉書的搜索功能和社交媒體服務，但這兩家
公司都在中國經營廣告業務。臉書廣告幫助中國公司和政府部門的海外推廣，大部分業務
是通過香港辦公室完成的。諮詢機構Pivotal Research的統計表明，臉書在2018年通過中
國大陸廣告客戶完成了50億美元的廣告銷售額，中國成為了臉書在美國以外的最大廣告市
場。

2020-07-12 02:07 經濟日報 / 記者陳昱翔／台北報導

https://imgur.com/hzVJIiO.jpg
三星下半年新款旗艦機Galaxy Note 20預計8月初發表，目前已要求台系零組件供應商積
極備貨，期盼在疫情趨緩下，新機銷售量能力挽狂瀾，彌補年初Galaxy S20銷售不佳狀況
；據悉，大立光、GIS-KY、雙鴻、超眾、泰碩等三星供應鏈，本月起出貨量將逐月提升。
由於今年初受新冠肺炎疫情影響，用戶消費力大幅下滑，三星上半年旗艦機Galaxy S20在
南韓、中國大陸，甚至歐美市場銷售均不如預期，尤其自家市場表現不佳，銷售量比去年
同期減少四成，讓三星相當無奈。

根據市場調查機構Strategy Analytics統計資料顯示，三星今年首季手機出貨量僅6,000
萬支，較去年同期7,200萬支下滑20%；其中Galaxy S20系列5G智慧手機在南韓銷量僅達上
一代Galaxy S10系列的六成。

不過，隨亞洲新冠肺炎疫情略有趨緩，三星將希望全部寄託在下半年新款旗艦機Galaxy
Note 20，原先6月對供應鏈仍釋出保守展望，但7月開始恢復信心，待8月銷售狀況出來後
，若優於預期，供應鏈有望獲得一波追加訂單商機。

據悉，Galaxy Note 20鏡頭模組由大立光供貨，GIS-KY負責屏下指紋辨識模組，散熱模組
由雙鴻、超眾、泰碩等三家業者供應，上述業者為國內少數的三星供應鏈。法人認為，雖
然疫情讓消費力下滑，但用戶上半年如果沒有換機，下半年很有機會換機，有望推升
Galaxy Note 20銷售量。

根據外電消息指出，三星將推出Galaxy Note 20與20+兩款高階手機，並且支援120Hz螢幕
更新率，同時也會搭載108MP（1億8百萬畫素）相機，屏下指紋辨識將全面升級，可能導
入高通新一代超聲波指紋辨識技術，預估感測面積是S10系列17倍，還能同時讀取兩個指
紋，解鎖時間更快更安全。

法人指出，雖然今年全年智慧手機需求可能低於去年，但最壞時機已過，預料下半年手機
需求在5G帶動下，有機會比上半年好，三星、蘋果都會在近期發布新機種，用戶若未在上
半年換機，就很有機會購買新機，台廠業績則有望雨露均霑。
