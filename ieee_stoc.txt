2年燒掉40億人民幣 馬雲無人超市宣告失敗

4.完整新聞內文:

https://i.ntdtv.com/assets/uploads/2020/01/1506090121501664.jpg
在大陸，無人超市儼然成了互聯網經濟的又一個泡沫。（網絡圖片）

【新唐人2020年01月13日訊】阿里巴巴創始人馬雲看好的產業——無人超市，儼然成了互
聯網經濟的又一個泡沫。過去兩年來，無人超市在中國多地落地。然而，在無數資本湧入
這場燒錢大戰之後，馬雲的無人超市，在2年損失掉了40億人民幣，最後卻宣告失敗。

在傳統零售市場開始逐漸飽和之下，電子商務巨頭和零售超市巨頭都聚焦於產業轉型，中
國掀起了一股新的零售浪潮，在這種情況下，無人超市應運而生。

2020年未到之前，幾乎所有的無人超市，已  经陷入了商店關門、裁員和資金削減的漩渦之
中。

然而，無人超市當初曾是一股新的零售浪潮，是馬雲最為看好的一個產業。

2017年7月7日，號稱「不用掏錢包、不用掏手機」 就能買單的無人超市「淘咖啡」在杭
州正式開業。作為馬雲的首家無人超市，「淘咖啡」一時間門庭若市、風光無限。各大媒
體也爭相報導，鼓吹起新零售：

無人超市不只是酷炫，更是零售業的新機會。
無人超市來了，再不努力，你將無工可打！
無人超市是一場技術革命，未來3至5年，實體百貨店將面臨前所未有的大衝擊！

杭州第一個無人超市上市後，據市場機構統計，僅2017年，全國無人零售貨架就累計落地
25000個，無人超市累計落地200家，無人零售市場累計融資超40億人民幣。

這場新零售改革由阿里、京東、蘇寧等進入，十幾家初創企業競相進入這個新產業鏈中，
各大媒體公司爭相報導，這可能是一場顛覆性的技術革命，馬雲在打破傳統行業以及支付
行業同時，還是將手伸向了零售業。

無人超市曇花一現

然而，好時光並不長久，無人超市就像曇花一現，開的快，去的也快。在僅僅不到半年時
間，盛極一時的無人超市相繼面臨倒閉。

2017年7月，上海首家無人便利店「繽果盒子」，開店僅一個月就宣布關門大吉。而繽果
盒子在剛出現時，曾受到過諸多資本的青睞。

作為西南首家無人超市，成都的GOGO無人超市宣部營業，同屬一家公司的無人貨架項目「
GOGO小超」也宣告停業。與此同時，該公司還曝出拖欠員工工資的消息。

GOGO無人超市成立於2017年9月，而它僅僅只運營了4個多月。

2018年3月，長春市首家無人超市，在吉林大街兒童醫院附近落戶。僅過了1年的時間，
2019年5月，這家無人超市已停止營業並低調出兌。

不論是無人超市，還是無人便利店、無人貨架，面臨運營困難的企業遠遠不止上面這幾家
。

即使少數幾個無人超市仍在苦苦掙扎，但在專業人士眼中，剩下的無人超市，只不過是拖
拖拉拉耗費時間而已，最終的結束將會是一無所獲。

在人們眼中，無人超市比普通便利店的勞動力成本低，意味著利潤更高。也的確因為這樣
，這個新零售行業才會受到資本家的青睞。

然而從剛「出道」時的萬眾矚目，到現在無人問津，只用了不到2年時間，無人超市們的
問題，究竟出在哪？

無人超市，敗給了中國大媽？

在馬雲推出無人超市不久，一個新聞曾引起市場廣泛關註：「無人超市，敗給了中國大媽
」。

新聞主要是說，由於天氣炎熱，一群大媽跑進無人超市避暑，席地而坐聊家常，幾乎把超
市圍個水泄不通。

儘管這則新聞看上去顯得十分搞笑，但這也足以反應出無人超市作為一種新零售，其商業
模式存在的種種問題。

有人在無人超市拿東西不付帳

「無人超市」允許顧客以現金或「支付寶」付帳，因超市里沒有店員，顧客可自行購物並
付款，甚至不付款也沒人管。有民調顯示，大陸的信用環境還不成熟，這種無人收款的商
店肯定行不通。

據陸媒報導，個人徵信機構「芝麻信用」在北京與杭州試驗這種「無人超市」，在北京試
驗一天的結果顯示，有些顧客從「無人超市」中取走商品卻不付錢，也有一名顧客抱走昂
貴的菸酒，卻只付人民幣10元。

而在杭州試驗的結果顯示，此「無人超市」出售人民幣16,700元的商品，卻只收到13,700
元，損失了大約3,000元，單日付款率為82%。

根據新浪網的民調，25%的受訪者表示，理解少付款或不付款的行為，因為人難免有貪念
。至於「無人超市」在中國可不行？有大約73%的人認為，不可行。

也有觀點認為，「無人超市」在國外並不罕見，但目前大陸的信用環境還不成熟，這種無
人收銀模式「肯定行不通」。

中國大媽點評無人超市

馬雲的推出無人超市後，網路上還盛傳一段一位中國大媽接受媒體採訪的片段，記者問大
媽對無人超市的看法，大媽脫口而出：「超市不需要養員工了，那東西是不是更便宜啦？
」

記者沒有正面回答，繼續發問說：「無人超市的推出將會改變我們傳統的購物方式嗎？」

大媽立即說：「改變啥喲？買東西不花錢啦？刷刷支付寶那也是花錢吶！」

最後大媽直言：「我們要的不僅僅是改變，而是帶來幸福的改變。現在很多改變不僅沒有
增添我們的幸福，還增添了許多煩惱！這才是你們記者應該關注的問題。」

記者無言以對。網友則大讚大媽肺腑之言表達民意，「老百姓就是應該這樣關心自己的生
活，『無人超市』如果東西不便宜，對老百姓有啥意義？」

: 蔡總統曾說，最低工資3萬元是她的夢想數字。而今，人民再給蔡四年任期，外界預期，
: 未來四年將會持續大幅調升基本工資，朝3萬元目標邁進。
: 蔡總統自2016年上任後，基本工資連四年調升，月薪調幅達18.95%。時薪調幅更高達
: 31.67%。
: 目前基本工資為23,800元、時薪為158元，若要在2024年達到3萬元目標，每年調升幅度恐
: 要超過5%。外界關注，蔡總統未來究竟會下多少力道調升基本工資，以兌現3萬元的目標
: 。
: 早在2016年總統大選期間，蔡陣營就承諾要訂定《最低工資法》，將取代現行基本工資審
: 議機制。經過三年多討論，勞動部於去年5月將草案送進行政院審查，至今仍未完成審查
: ，足見勞資雙方歧異仍大。隨著蔡連任，兌現最低工資法將是箭在弦上。
: 依勞動部預告草案，最低工資審議應參採消費者物價指數年增率（CPI）擬訂調整幅度，
: 其他如勞動生產力指數年增率、勞工平均薪資年增率、經濟成長率等，都屬參考指標。工
: 商團體看重經濟成長率，並未列入「應」參採指標，勢將面臨一番角力。
: 攸關全台千萬勞工的《勞工職業災害保險及保護法》已送政院審議。職災保險法單獨立法
: 最大變革在五人以下企業納入強制投保對象，連打零工的臨時工，其自然人雇主都可到超
: 商等繳費投保；新法也打破打工族等部分工時投保薪資級距，一律改依基本工資投保，最
: 高投保薪資上調到72,800元。
: 雖然職災保險費率全數由雇主負擔，因為職災保險給付可以完全抵充雇主在《勞動基準法
: 》上的職災補償責任，且開辦前三年保費不會調整，雇主反彈不大，甚至部分業別雇主更
: 主張投保薪資不要訂太低；料勞工職災保險法有望獲勞資雙方支持。
: 蔡總統連任要面臨最大燙手山芋是勞保年改議題。根據勞保財務精算報告，勞保50年潛藏
: 負債持續增加至9.11兆元，破產年限由已提前一年至2026年。
: 蔡政府提出勞保年金改革方案，主要有三大內容：一、加快費率提高速度，由每二年調升
: 0.5個百分點，修正為每年調升0.5個百分點；二、平均投保薪資採計期間由五年拉長到15
: 年；三、政府每年撥補200億元，並負最終支付責任等。
