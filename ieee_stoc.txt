武漢肺炎（COVID-19）大流行重創商業活動，包括百年老店潘尼百貨（J.C. Penney）都
因此宣告破產，但據《路透》分析證券申報與法院記錄文件，全美逾40家在疫情期間聲請
破產的大型企業，近3分之1在破產前竟然先向高管大發獎金！

2005年美國通過的《破產法》明令禁止企業在聲請破產期間向管理層發放慰留獎金（ret-
ention bonus），但上有政策下有對策，企業紛紛利用法規漏洞，在進入破產程序前先行
撥付獎金。

例如，包括潘尼百貨和租車公司赫茲（Hertz）在內，有8家企業在聲請破產保護前的5天
內批准向公司高管發放獎金；德州油氣業砂材供應商Hi-Crush Inc.，則在聲請破產前僅
僅2天向高管支付了獎金。

在《路透》檢視的45家企業中，接近一半是在聲請破產前2個月發放了獎金，約3分之2甚
至是提早在聲請破產前半年內發放。

在疫情衝擊下，潘尼百貨先前不得不暫時關閉全美846家門市，讓多達7.8萬名員工休無薪
假（總員工數約8.5萬人）。潘尼百貨為此在5月15日向法院聲請破產，但該公司董事會卻
在5月10日先行批准了向執行長Jill Soltau等高管發放近1000萬美元（約2.96億元台幣）
獎金的計劃。

法院文件顯示，Jill Soltau在5月13日當天領到了170萬美元（約5030.81萬元台幣）長期
工作獎勵及450萬美元（約1.33億元台幣）慰留獎金。相形之下，索爾托上周三才宣布潘
尼百貨將永久關閉152家門市、裁員1000人。

歷史同樣超過百年的精品百貨Neiman Marcus在3月關閉全美67家門市、4月讓逾1.1萬名員

gy）也於5月向自家高層和較低階員工發放了2500萬美元（約7.4億元台幣），就在聲請破

人才，高階主管紛紛走人，最終可能危及業務前景、傷害債權人和員工的利益。現在，一
些企業主張若非因為這場疫情，營運狀況不會惡化至此，在股票獎勵變得不值錢的情況下
，趁破產前發獎金慰留是必要的，這些獎金可確保在疫情危機中領導團隊的穩定。

有幾個觀念

1. 用所得來當費率計算基礎叫做稅，而非社會保險

2. 稅率和避稅成本永遠都是一個恐怖平衡，稅率越高避稅相對成本就越低

3. 避稅門檻和一般打工仔和股票玩家無關

所得稅機制其實之前討論過就不提了，來談談健保費

健保其實現在的狀態叫做社福，而不應該稱作社會保險

因為一點保險精神都沒有

保費本身是分散風險的功能，但保費必須依照不同的風險係數去計算

如此才有正向的風險循環

社會保險要做的應該是擴大保險費來源讓高風險和低風險保費級距彼此拉近

而不是用社會保險名義去做課稅之實

假如要用社福的型態，應該直接去成立健康社福的法規然後對此立法課稅

2. 企業劣幣驅逐良幣: 守法企業人力成本極高，違法企業賴皮爽爽過

3. 高階生產力人口負擔過大: 低階繳極小稅率，高階資本家會避稅平衡

4. 扼殺新創: 台灣資金本來就不如中美那種市場，分紅機制稅率高吸引力變低

: 台灣這種老鼠會就是壓榨生產力，當然假如你的生產力可以一直維持一定比例
: 那自然老鼠會機制撐得過去
: 但你今天生產力占比越來越小的時候，那這老鼠會只能加大力度去把生產力榨汁
: 有幾個觀念
: 1. 用所得來當費率計算基礎叫做稅，而非社會保險
: 2. 稅率和避稅成本永遠都是一個恐怖平衡，稅率越高避稅相對成本就越低
: 3. 避稅門檻和一般打工仔和股票玩家無關
: 所得稅機制其實之前討論過就不提了，來談談健保費
: 健保其實現在的狀態叫做社福，而不應該稱作社會保險
: 因為一點保險精神都沒有
: 保費本身是分散風險的功能，但保費必須依照不同的風險係數去計算
: 如此才有正向的風險循環
: 社會保險要做的應該是擴大保險費來源讓高風險和低風險保費級距彼此拉近
: 而不是用社會保險名義去做課稅之實
: 假如要用社福的型態，應該直接去成立健康社福的法規然後對此立法課稅
: 那自然看是要用所得級距課徵還是直接提高所得稅稅率去支出都合情合理

: 1. 勞動力價值會嚴格被低估: 勞動力薪資和實際企業成本差距大

: 3. 高階生產力人口負擔過大: 低階繳極小稅率，高階資本家會避稅平衡
: 4. 扼殺新創: 台灣資金本來就不如中美那種市場，分紅機制稅率高吸引力變低
