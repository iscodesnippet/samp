核心面積超巨大！AMD 新一代 Big Navi 顯卡規格曝光

傳言 AMD 新一代 Big Navi 架構 Radeon 顯示卡將於今年問世，如無意外將採用台積電（T
SMC）7nm 製程技術，並被命名為 Radeon RX 5950XT，外媒《Wccftech》披露更多相關規格
。

內部消息指出，Radeon RX 5950XT 將是 Big Navi 旗艦顯示卡的重磅炸彈，直指對手 Nvid
ia（輝達）的 RTX 2080Ti，資料顯示 Radeon RX 5950XT 擁有核心面積 505mm 的巨大晶片
，外界均認為該顯示卡效能將會是中階的 RX 5700XT 的兩倍以上。

AMD 將 Big Navi 架構分成 Navi 21、Navi 22、Navi 23 三種規格，其中 Navi 21 就是 R
X 5950X，而 Navi 22 與 Navi 23 的核心面積則分別為 340mm、240mm，後者被認為是 RX 
5700XT 的繼承者（RX 5700XT 核心面積為 251mm），將成為 AMD 新的中階顯卡。

基於 Navi 21 的 RX 5950XT 將具備 80 個 RDNA2 CU 單元，擁有 5120 個流處理器，比目
前的 RX 5700XT 擁有的 40 個 RDNA CU 單元強悍兩倍以上，考慮到 RDNA 架構從一代到二
代的進步，效能可能比預測強上不少。

消息指出，如果 AMD 希望在 Big Navi 增加跟對手 Nvidia RTX 系列顯卡一樣的光線追蹤
技術（Ray Tracing），必然得減少 RDNA2 上的 CU 計算單元，不過也有消息認為 AMD 可
能會透過增加硬體的方式來做到光追效果，例如 Nvidia RTX 顯卡上就有使用 Tensor 晶片
來實現光追。

無論如何，《Wccftech》認為 AMD 即將推出的顯示卡，無論在效能還是功耗都會有顯著進
步，這次 AMD 將透過三款 Navi 架構來一次達成旗艦、中高階與中階三種市場定位的產品
，這些顯示卡都會隨著 Navi 與 RDNA 架構的成熟，在遊戲市場上造成巨大的迴響。

根據韓國 RRA 認證機構與歐洲 EEC 資料顯示 AMD 即將推出的顯示卡有 Radeon RX 5950XT
、Radeon RX 5950、Radeon RX 5900 以及 Radeon RX 5800，最快在九月以前公布。傳言 N
vidia 之所以還未推出 RTX 20 系顯卡的繼任者 Amper 架構，就是還在等待 AMD Big Navi
 顯卡的發布，因為這將影響 Nvidia Amper 顯卡的上市價格。

分析師警告：美股漲勢只是熊市反彈 仍有崩潰風險

〔財經頻道／綜合報導〕武漢肺炎抗疫藥物試驗出現進展，激勵週三美股上漲，但市場研
究專家警告，4月份的美股漲勢將崩潰，因為投資者過於樂觀。

金融研究機構Bianco Research總裁James Bianco週三向《CNBC》表示，他明白股市自3月
低點漲回許多，但在他看來這是回撤上漲，就像是長時間熊市後遇到的第1類反彈。

報導指，Bianco上月警告武漢肺炎的經濟動盪將大於金融危機，他在3月初把所有資產換
成現金，儘管股市出現反彈也不換回。

據報導，本月迄今，標普500指數上漲近14%，若此一趨勢維持下去，這將會是該指數自
1974年以來最佳表現，與此同時，道瓊工業指數漲逾12%，有望實現自1987年1月以來最佳
月度表現。

但Bianco週三指出，他們將重新評估標普500指數2200點的低點，未來有可能再下探，並
指現在是市場的關鍵時刻。Bianco認為，人們終將意識到，即使經濟重新開放，生活也不
會恢復正常，就算聯準會採取積極政策救經濟，復甦仍是漫漫長日。
