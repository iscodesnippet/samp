※ 引述《minazukimaya (水無月真夜)》之銘言：

: 這是另一個本文要討論的重點：
: 高階晶圓製造，是第四次工業革命的基礎。
: 這個觀念在產業先驅中已經是共識，在未來五年會慢慢普及到大眾身上
: 所謂的工業革命，是指人們找到了某種
: 1. 有效率利用能源的技術   從而
: 2. 在某個生產階段/生活應用中，大規模取代了人力/獸力  形成
: 3. 社會生產力/生活型態的決定性改變
: 從機械革命、電力革命、資訊革命，都是以不同的技術來達到上述工業革命的條件
: 那麼所謂第四次工業革命是怎麼回事呢？
: 實際上是人類要追求
: 1. 高效率使用能源的資訊元件 (包含了運算/讀取/儲存)
: 2. 在自動化生產、智慧化生產、社會資訊整合方面 大規模取代人力
: 3. 最終讓人類社會進入「超人類時代」或者到達「技術奇異點」
: 其中 1.這點是第四次工業革命的必要條件
: 也就是積體電路的資訊處理/學習能力，終於在能源使用的效率上接近人類的表現
: 過去的兩次AI浪潮失敗的原因，其實說到底就是運算力不夠
: 而正是像台積電這樣的積體電路製造公司，對於製造技術的不斷追求
: 隨著摩爾定律的進展，終於接近了觸發第四次工業革命的前夕
: 所以總體來說，高階晶圓製造不只是一個產業問題
: 實際上是一個關於國家命運的政治問題
: 任何國家都希望掌握工業革命的關鍵領域
: 根據歷史經驗
: 這種悠關國家命運的議題
: 絕對不會是用什麼政治道德、國際主義之類各種理想化的意識型態來解決
: 而是赤裸裸的武力、利益交換、國際政治博奕
: 台灣政府如何最大化的利用自身產業優勢
: 在未來的多國政治博奕中最大化國家利益
: 是為政者要仔細思量的問題

台積電與格芯之間的爭議光速和解
但後半部份的議題，以比原本預想更快的節奏被提上議程了

我原本是覺得需要一點時間發酵的
不過這幾天密集的在公開場合或傳媒上被highlight
整理一下各方資訊
首先是10月底美國國防部提出的希望台積電去美國設廠的議題
然後是前天台積運動會時，Morris突然拋出的「兵家必爭」說
會後劉董又多作了一些補充
最後是這兩天FT和Bloomberg輪流當謠指部放風向球

以上的癥兆，可能表示博奕的各方開始為下一個階段作準備了
這其實是一場多方博奕，以利害關係群體(包含各國的商業實體和政府)來分的話
美方三個，兩岸合起來算有三個，再加台積自己
所以是一場七個主要玩家的博奕，再加上一大堆敲邊鼓看戲的

這篇文章主要從張董和劉董的公開發言，分析一下台積電目前的博奕策略
我把我認為是重點的幾個消息列在下面

首先美國設廠這件事，劉董的意思就是不去了
成本是其中一個原因(半導體產業要形成聚落才能壓低成本和提高生產效率)
打算透過其他跟蹤技術來解決美國國防部的安全顧慮

再來就是Morris說了一句話
「真的有具體的東西可以保護你嗎？沒有，我想只有價值能夠保護你。」
這其實很符合Morris一貫的信念，就是誠信正直的核心價值

劉董在之後又補充了一些重點
「平衡很重要，作錯決定會影響平衡」
「在優先順序上，確保我們的客戶得到最大的助力」
「光靠一家公司是不足的，也得將台灣自身的政治納入考量，確保企業能在競爭中和政府配
合，保持台灣的最大利益。」
「地緣政治不會就此結束，會長線發展」

台積電的策略看起來很符合一直以來的公司文化和價值
「正直誠信」「客戶優先」
另外就是打算要「維持平衡」

這個策略的難題也很明顯，就是有兩個玩家現在想翻桌了
剩下的其他本來想維持平衡的玩家，也在各自打算盤

因此對台積來說，找出一種「不同於現有平衡的新平衡」
說服所有玩家把遊戲玩下去，這是最優策略

但因為被翻桌的可能性不小，也要為新局面作打算
也就是在平衡被打破之後，要優先拉攏哪些玩家來重建局面

正如我之前所說，走到這個地位的台積電
未來的風險都存在於政治面而不是經濟面
因此這場博奕的結果，會是影響股價走勢的最主要因素

滿倉GG或者全力押空的各位都是這場博奕中賭外圍的
賭け狂いましょう~

美股多頭不止 台股力拚11600點

（中央社記者韓婷婷台北5日電）美中貿易協商傳出好消息，科技、金融
及能源類股同步大漲，激勵美股4大指數全刷歷史新高，分析師認為，在
美股領頭下，台股將可朝11600點進攻。

美中貿易協商出現進展，美國可望解除美國企業出口華為的禁令；道瓊工
業指數上漲114.75點、漲幅0.42%、收在27462.11點，那斯達克指數上漲
46.8點、漲幅0.56%、收在8433.2點，標準普爾指數上漲11.36點、漲幅
0.37%、收在3078.27點，費城半導體指數上漲36.99點、漲幅2.19%、收
在1726.92點。

台股在電子權值股表態下，站上11500點再創新高，分析師表示，外資在
現貨市場買超再度超過百億元，自9月初正式轉為買方以來，已經連9週買
