選擇權有分週選和月選

週選是每個禮拜三結帳

日盤開盤時間：08：45-13：45
夜盤開盤時間：15：00-05：00

左邊call是看漲，右邊put是看跌

中間的12xxx是履約價

橘色的12700叫做價平（意思是跟現在台指大盤點數最接近）

其他的分價內跟價外

假設以履約價來看，call方12500代表已經漲過了，所以12500是價內

put方法12500還沒跌到，所以是價外

價格要看成交價那一格，需要保證金買方最簡單計算就是*50

格變成0.1，就是一口5元

: : 【買買權(Buy Call)與賣買權(Sell Call)】
: : 現在豬肉一斤 100 元，小明與小華約定好 8/1 小明要用一斤100元的價格跟小華買豬
: 肉

: : 這筆交易背後的概念是：
: : 小明視角：
: : 小明認為未來豬肉價格會長，所以現在先跟小華買了未來可以用100元買豬肉的權利
: : 這就是選擇權的「買買權」
: : 假設 8/1 豬肉上漲到200元，小明就能根據合約，用100元跟小華買到一斤豬肉，賺
: : 到價差100元x10斤=1000元

: : https://i.imgur.com/8Ejtffr.jpg

: : 小華認為未來豬肉價格會跌，所以現在先賣小明買進的權利
: : 這就是選擇權的「賣買權」，什麼都不用做，賺取保證金
: : 假設 8/1 豬肉上漲到200元，小明就能根據合約，用100元跟小華買到一斤豬肉，等
: : 於小華要賠給小明1000元

由於華為受到美國禁令影響，華為目前轉移往中國國內市場前進並使用自己的晶片，但已
影響高通在中國的市占率的流失。預計，9月14日之後台積電已無法為華為旗下海思代工
生產晶片，因此，華為必須另尋出路。或許，華為改變策略，先乖乖付專利授權費換得高
通站在同一陣線，一起說服美國政府同意賣高通晶片給華為，以突破禁令關卡。

2020年7月29日，美國晶片大廠高通(Qualcomm)公布2020第三季會計年度財報，宣布已經
與華為(Huawei)簽訂新的長期專利授權協議，受到此消息激勵高通股價飆升12%。至於協
議細節未揭露。

到底這一筆新的長期專利授權金是多少？

根據高通財報資料，今年截至目前9月底，若不包括某些項目的收入約在55億至63億美元
，分析師平均估計為57.7億美元；若包括來自華為的未付款在內，則營收將在73億美元至
81億美元之間。因此，估算這筆華為未付款，可能就是新的長期專利授權金，應該在18億
至23億之間。

高通預估公司在第三財季，由於新冠疫情大流行，重創手機的銷售，高通第三季的營收較
去年同期幾乎大減50％，達到48.9億美元，凈利也同時下降61%，降至8.45億美元。高通
代表美國電信業景氣指標，它若不好，美國經濟會回溫嗎？

同時，高通也透露期待下一代蘋果iPhone之5G旗艦手機，可能會在今年稍晚時將首次亮相
。因而，帶動高通的5G晶片成長。之前，蘋果支付高通專利和解金45億美元以支付兩年專
利費，有此可見每年約20億至22億美元之間。這與華為支付的新授權協議很接近。

高通反壟斷案與專利授權金計算

由於，高通公司在通信晶片行業的優勢地位，既可從專利授權而獲得大筆授權金，這些專
利涵蓋了許多通信系統的基本專利技術，使得眾多手機製造商必須按每部手機售價支付一
定比率的費用給高通當做是過路費。同時，高通也賣自己設計的行動通信晶片給業者放入
產品中。高通表示，已與所有主要手機製造商簽署了專利授權協議。

高通與與華為曾於2019年6月先簽署短期授權協議，華為須向支付高通每季1.5億美元(每
年約5~6億美元)專利授權使用費，藉此達成暫時和解協議。

當然，企業營運若過於利用行業的優勢地位，很容易在不知不覺中落入反壟斷困境。不例
外，高通的知識產權的覆蓋範圍及其所產生的利潤已經在全球招來了法律和法規方面的挑
戰。

高通在美國反壟斷案，2018年法官已裁定高通必須授權通訊晶片給競爭對手，包括：英特
爾、三星等。也就是說，高通必須遵守FRAND承諾，以公平合理且無歧視的條款提供授權
條件給競爭對手。至於，專利授權金部分，法官說5％授權使用費不符合FRAND原則的。因
為，每家業者狀況皆不相同，如何每家就以5%來定價。所以，法院對高通的專利授權金判
定仍未決。

結語

高通與華為達成的長期授權及和解協議，主要針對高通無線技術專利授權供華為使用。至
於，高通能否出售驍龍晶片給華為，目前仍然不清楚美國政府的意向。如今高通已與華為
達成和解，並簽下一項長期專利協議，預期下一季將可進帳近 18 億美元的和解款項。這
對華為而言，或許是先放下身段示好，同時換得高通為取得這筆高額授權金，應該會協助
華為突破美國政府關卡而努力。
