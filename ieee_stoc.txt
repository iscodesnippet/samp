:  簡單分析
: 在疫情爆發之前,即為少數有獲利的SaaS股,去年自IPO上市時曾經被炒過一波
: 而股價破百,但在炒作之後在年底又回到第一天的價格(60初)
: 而直到疫情爆發之後此SaaS才較為常人所知,但股價從2/1疫情爆發時75漲到212.
: 足足漲了近三倍.股價要這麼膨脹必須要有足夠多的燃料(題材or實績)來支撐.
: 這隻的題材相當多,不管是一開始的居家辦公、居家上學、或是封城、防疫等等
: 都有他的影子.這造成了話題性相當高,人們講到遠端就會想到zoom.
: 有人會想說:這不就只是遠端軟體嗎? 軟體最容易被抄襲了.
: 大廠當然也看中商機,有的改善自己的服務(如Microsft Teams),有的直接硬幹(FB/Goog
le)
: 都想分一杯羹.優劣見仁見智,唯一相同的是大家均把zoom當主要敵人.
: 除了大廠想分食這塊大餅以外,內憂則是資安問題.當初爆出資安問題時,中國色彩濃厚的
: zoom第一時間成了人人喊打的落水狗,股價還從水漲船高的態勢瞬間跌落神壇,從波段高點
: 157跌到110左右.
: 當時有人說:資安問題?我只是上課要用,並不在意這種問題.不能用的話要用什麼?
: 但不管如何,各國政府甚至大企業如FB都對zoom祭出封殺令.逼使zoom用聘請一堆資安專家
: 的新聞來幫自己洗白.或許這招對形象及技術都有效,於是資安再也不成問題.
:  內憂解了,外患依舊
: Microsoft Team挾著office整合/軟硬體服務/資安的能力,企圖左打Slack右踢zoom,
: Google 推出 Google Meet
: FB 推出 Rooms
: 在疫情當下要與大家分食這塊以前只有自己吃的餅,雖然2020H1應該輕鬆超越預期,但隨著
: 前陣子zoom宣布IaaS方面與Oracle合作,跌破眾人眼鏡.但仔細一想也並非毫無道理.
: 三大雲端商(aws/azure/gcp)有兩家產品與自己有競爭關係,而aws最大,可能
: 獲得的優惠並不多.於是找上了也想發展雲端的甲骨文.這算是各取所需.
: 從競爭的角度來看的確是個正常的選擇,但沒有更好的選擇嗎?
: 在平台的營運上面甲骨文並非最好的選擇,因為隨著zoom人數爆增,對IaaS也造成沉重的負
: 擔,而zoom的服務對IaaS的品質尤其吃重.這點如果結盟aws/azure,或許可以取得較好的技
: 術服務.
: 近日有國際用戶反映zoom的連線不如以往順暢,進而轉移至teams等服務.有人即推斷是I
aaS
: 負擔過重的關係(當然也可能是該地連線不佳)
:   疫情過後如何?
: 正如同多數的疫苗股,疫情過後總是會回歸平靜的時刻.zoom也是如此,但會回到哪裡?
: 股價起於封城,解封過後相信不會回到封城前,但過度膨脹的股價要透過什麼來支撐?
: 有人說:許多企業已經允許員工永遠遠端上班. (如twtter/fb/google..)
: 有人說:人們已經習慣在家,不會改變習慣了.
: 有人說:已經習慣用zoom跟親友聯繫了.
: 在這波疫情之下,有些工作被迫停擺,少數的工作可以居家辦公,而這類少數的工作,
: 也不是都允許員工不進公司(聽到比較多允許的是科技業等),這類用戶在疫情過後勢必會
: 減少.

留.

跑去研

中芯國際公布上海科創板上市計畫，
招股說明書中暗示在美國的限制下，
日後無法向華為供貨。
圖為華為位於廣州的門市（歐新社檔案照）

公布上海科創板上市計畫 將籌資840億

〔編譯盧永山／綜合報導〕中芯國際週一公布在上海科創板（STAR）的上市計畫，
將籌資二百億人民幣（以下同；約八四○億台幣），
主要用於製程為十四奈米的十二吋生產線。
但中芯招股說明書提及美國出口政策調整所帶來的風險，
可能使該公司為若干客戶提供的晶圓代工及相關配套服務受到一定限制；
暗示該公司可能像台積電一樣，在美國的出口限制下，日後無法向華為供貨。

中芯表示，
這次售股籌措的二百億元資金，
其中八十億元將建設一條製程為十四奈米、月產能三．五萬片的十二吋生產線；
四十億元將做為先進和成熟工藝研發項目儲備基金，其餘八十億元為補充流動資金。

「美許可前 無法為若干客戶生產」

在招股說明書中，中芯也提及美國出口政策調整所帶來的風險，
尤其是去年五月，美國商務部將若干中國公司列入「實體清單」；
今年五月美國商務部又修訂直接產品規則，
據此修訂後規則，
若干自美國進口的半導體設備與技術，
在獲得美國商務部行政許可前，
可能無法用於為若干客戶的產品進行生產製造。

中芯指出，
上述修訂規則中，仍有許多不確定的法律概念，其具體的影響程度目前仍不能準確評估；
美中貿易摩擦等相關外部因素，
可能使該公司為若干客戶提供的晶圓代工和相關配套服務受到一定限制。

中芯表示，
未來美中貿易摩擦若持續升級，限制進出口或提高關稅，
該公司可能面臨設備、原材料短缺和客戶流失等風險，
進而導致公司生產受限、訂單減少、成本增加，
對公司業務發展和經營業績產生不利影響。
