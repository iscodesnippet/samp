因為顯然這位根本沒看過這部作品
所以我就來列看看  這部作品是否真的如他所說

※ 引述《forsakesheep (超歐洲羊)》之銘言：
: 我很好奇
: 沒人覺得這部的數值設定很智障嗎？
: 表面只有標SABC這種，隱藏實際數字就算了
: 升級居然還會把數值變成隱藏值然後重新累積？
: 我看主角居然有三小各項能力破千的隱藏值，就連看都不想看了
: 不是說這樣的設定是YY還是怎樣，而是這種亂七八糟的設定比七龍珠還爛
: 看情報後來那些數值也沒啥意義
: 就是一個表現『主角好強、成長好快』的指標而已
: 根本是把對故事設定認真的讀者當智障吧
: 而且會把戰力數值化的作品根本就是作者懶到極點
: 連怎麼表現實際戰力差異都不會才會這樣搞

結論來說  這段話對其他有數值的作品來說  或許可能是對的
但對這部作品來說  就是錯誤的

因為  作者確實詳細的描寫了等級有差距的人  在實戰表現上會有多大的差距
主要描寫這點的章節中存在能讓人暫時升級的魔法
於是就能看到一個角色在暫時升級前後的實際表現差距多少  都是有文字詳細描述的
而我想有讀過的人都知道  作者這段文字是表現出戰力差距  還是只說等級提升所以變強


這部跟一般大量用數值的作品最大差異應該是
數值出現次數不太多  但出現的時候  除了上述的等級是魔法的效果之外
主要是用來表現主角這次冒險或前次冒險經歷量化後大概是多少
因為數值設定是要找主神更新後  才會更新
故小說一集幾乎都只有最後面才會看到  尤其小說中段開始一個事件好幾本
主角可能平均兩三本才有機會更新一次數值吧

要我說的話  這部的數值設定可以說是用來補足用的
因為小說主文中  其實已經將各個戰力差異跟成長都描寫的非常完善了
並沒有因為有數值在所以而對任何描寫有偷懶

故大部分讀者都沒有對此作品有數值有意見吧


對我個人來說  一部作品使用這種數值
只要沒有膨脹到其他人的數值都沒意義的程度(很多YY作當範例  隨便就幾億)
那這是一個很方便的  記錄進度用的東西

: 這遊戲主要抽的是角色跟Orb，角色沒有分稀有度每隻角色都是從1星開始，但有分Fes限
: 定角，每個月月底固定會有角色抽率加倍的池(平常5%月底10%)，因此存石都建議在fes的
: 時候再抽，有幾隻角色只有限定在fes時候才抽得到，但他們每個月都會副刻跟輪流up，
: 目前沒有季節限定的角色生肖角色。
: 首抽:
: 色都有他獨立的功能因此不太有強弱的問題，除了參考攻略網首抽推薦以外，其實也可以
: 挑自己喜歡的角色開局就好，畢竟連主線跟活動送的角色都很強力而不含糊。
: 活動:
: 72目前的活動很單純就兩種，第一種是劇情活動，官方目前很固定每個月推一個新活動跟
: 副刻一個舊活動，劇情活動就是邊推劇情然後拿石跟拿活動角，拿活動角的難度不高通常
: 只要推完前幾關就可以拿到角色了，前面有提過這遊戲送的活動角也都很強，因此如怎麼
: 樣都最好拿到角色。
: 推完所有關卡後有挑戰關可以給玩家挑戰，然後可以農活動掉落物去商店換取養成道具。
: 另個活動是稍微新一點的共襲活動，這活動開的時候會出BOSS讓玩家合作討伐，每隻BOSS
: 特性不同就得帶不同的角色上場戰鬥，官方很貼心的讓你沒練的角色也能在這活動時短暫
: 提升到一個中高的練度(雖然會有一點獎勵懲罰)，打完王後可以拿到賓果點數去戳賓果，
: 每一格都是獎勵(養成資源、體力藥水、SKIP卷、抽抽卷等很豐富)，算是目前這款遊戲少
: 數會跟人互動到的地方。
: 好像介紹的差不多了就稍微統整一下至今遊戲體驗的優缺點，以下比較主觀大家參考就好
: 遊戲優點:
: 1.角色模組很精美:立繪畫風這邊每個人接受度不同不特別提，但是角色的3D建模真的不
: 馬虎做的很精美，戰鬥結束時每個角色都會有段Ending pose，這邊都做得栩栩如生，

: 3.遊戲劇情:除了主線以外還有每月個活動劇情，而角色們各自也都有自己的專屬劇情，
: 質量都滿不錯的，懂日文的話很推薦閱讀
: 4.戰鬥系統有趣:由於每個人的角色庫不同，因此戰術很多元，推過難關時的醍醐味算
: 遊戲賣點之一。
: 5.遊戲步驟休閒:度過前期拓荒後，平常就是用SKIP卷耗完體就可以關APP了，官方活動
: 都開兩個禮拜左右，因此也不會太有壓力，而遊戲除了共襲跟PVP以外跟人是零互動，
: 所以也不太有來自別的玩家的壓力。
: 遊戲缺點:
: 1.角色聲優比較多新生代:可能是成本考量(?)這遊戲除了主線角跟幾隻角色以外的聲優
: 比較廣為人知以外，大多都是比較沒看過的名子。不過每個聲優配的角色最多兩隻，因
: 此不會有那種遊戲裡10幾%角色都是某位聲優配的情形...
: 2.送石量中等:新手入門後登入禮可以領個30抽，成就跟主線大概可以挖100~150多抽左
: 右後就剩登入跟每月活動能拿石，一個月大概就20~30抽左右，應該在手遊裡算中等的量
: (?)，不過角色很保值加上也不會有農具之類要抽了才能打活動的狀況，清心寡慾應該是
: 可以活得很好啦。真的想要的角色就等每半年一次的超得就可以花錢入手囉。

 作者  kiderself (冬之雅興)                                      看板  C_Chat 
 標題  [討論] 東立版 鑽石王牌3 翻譯問題                                       
 時間  Sun Dec  1 12:45:07 2019                                               
───────────────────────────────────────


本篇是第三集的部分，
圖數量也有十幾張，有興趣的可以到專頁看。
https://www.facebook.com/boosernocomicandgame/

以下是文字版：

---------------

原文：
あのクセ球を三年がどう攻略するか、見るものですよ！

東立：
可以好好地來看看三年級如何來打他的變化球！

修改：
可以好好地來看看三年級如何打他那顆會位移的直球！

クセ球＝ムービングボール，這我在第一篇裡面就有講過了，
這球雖然會位移，但它依然是直球。
別忘了第一集裡澤村親口說過他只會投直球，
到這個當下他並不會投變化球。
印象中是看過了藥師真田的投球之後，澤村才起念想學變化球。

---------------

原文：
粘り強いピッチングでチームを引っ張る投手の姿。

東立：
反而用強勁的投球來帶領整個球隊。

修改：
反而用極具韌性的投球來帶領整支球隊。

「粘り強いピッチング」是指雖然不斷地讓對方上壘，
但總是能在關鍵時刻鎖住不失分。
翻成「強勁的投球」，「粘り」的部分就消失了。

---------------

原文：
打ち損じたわけじゃねぇ。

東立：
我並沒有漏打那一球。

修改：
那一球我並沒有打不好。

「打ち損じた」是指打中了但沒有打到球心，
簡單來說就是打偏了，不是漏打。

---------------

原文：
ボールが手元で動いたような…

東立：
球竟然在接住後，亂竄了起來…

修改：
球似乎在快接進手套時，忽然亂竄了起來…

你沒有接住！你沒有接住！你沒有接住！
「手元」是指手邊，意即很靠近自己的意思，
在這裡並不是指手套內，而是接進捕手手套之前。

---------------

原文：
打ち取られていたのは俺の方だった…

東立：
我可能就會被三振了…

修改：
敗下陣來的人可能就是我了…

「打ち取られていた」並非特指「三振」，
凡是打者被投手解決掉，不管是滾地球、
高飛球或是三振，其實都可以用這個詞。
考慮到澤村投球的特性，他反而是讓人打不好，
而不是打不到。所以還有比翻成「三振」更好的詞彙。


原文：
あの浮かび上がると錯  覚
するほど威力のある高めの剛球。

東立：
那顆讓人產生錯覺的上飄高速球…

修改：
那顆飽富威力，足以讓人產生上飄錯覺的偏高快速直球。

這句話東立版完全翻錯。
現實中沒有人投出來的球會往上飄，即使是「直球」，
實際上從投手手中投出，一直到捕手接到，
球都是會往下掉的。
但有些投手可能是因為球的轉速快，或是其他的什麼因素，
讓他們的球「掉的幅度比一般直球還要來得小」，
所以從打者眼中來看，跟其他球相比，
就像是「往上竄」一樣。但這都是錯覺。
教練在這裡所說的，就是降谷投的球具有這樣的特質，
而不是他的球真的會上飄。
當然我覺得漫畫在這裡故意把球畫成真的往上飄了

---------------

原文：
左のムービングボールは貴重な存在かと。

東立：
左邊的變化球是非常稀奇的。

修改：
我認為會投位移直球的左投手，是很珍貴的。

前面已經提過了「ムービングボール」不是變化球，
而「左」不要這麼直白的就翻成「左邊」好嗎？
雖然也不能說錯，但應該是指「左投手」才對。

---------------

原文：
勝負カン  冴
える不動の４番、


修改：
在決定勝負的關頭時，十分靠得住的不動四棒，

「勝負カン  冴
える」應該寫作「勝負勘  冴える」。
「勘が  冴
える」就是指「直覺很好」。
因此整句話帶有
「一到緊要關頭，直覺就會敏銳起來」之意。
單純翻成「決定勝負」意思似乎不太足。

---------------

原文：
長い伝統のあるウチには口うるさい外野も多いですから。

東立：
擁有長久棒球歷史的本校，也培養出好幾位講評尖銳的外野手…

修改：
本校擁有漫長的歷史，因此也有不少意見很多的局外人。

日文的「外野」有數種意思，
一種是指棒球的「外野」或是「外野手」，
另一種是指「不相干的人士」。
這裡從語意來看，翻成「外野手」非常不知所云。
很明顯就是在講非棒球社的人士，
或是已經畢業的校友在對學校棒球社指指點點。

---------------

原文：
中途半端な投球するぐらいなら、
思い切って投げた方がいいと思って。

東立：
所以與其投自己不擅長的變化球，
倒不如盡全力投直球來得好。

修改：
所以與其投出半吊子的球，
不如使勁全力投球來得好。

再說一次，澤村不會投變化球。
而且這句話的原文裡根本沒有「變化球」的詞彙。
前面一格是澤村自承自己控球不好，
所以這裡的「中途半端な投球」指的是
「求控球降球速」的半吊子直球，而非變化球。

---------------

原文：
バッティングにはむらがあるが、

東立：
打擊方面雖然有點小瑕疵，

修改：
打擊方面雖然狀況起起伏伏，不夠安定，

雖然翻成「瑕疵」也不算錯，
但是「むらがある」在意思上偏向指「不均一」，
在打擊上則是用來形容可能一下人來瘋，
一下又好幾場低潮的不穩定狀態。
