Xbox Series X 的一些硬體情況早已公布。雖然有著超強性能的 SoC 這枚心臟，但官方
欽定的次世代 Xbox 主機靈魂——「Xbox 高速架構」（Xbox Velocity Architecture，
簡稱XVA）一直有點猶抱琵琶半遮面的感覺。今日 Xbox 項目主管傑森羅納德（Jason R
onald）發表長文正式揭開其神秘面紗。

現代遊戲需要海量數據來創造出具有真實感的遊戲世界供玩家探索。為了讓處理器發揮出
最大性能，通常所有的數據都會預先從硬碟載入到內存當中。愈發廣闊、生動的開放世界
就愈發需要更多的數據，比如環境網格數據、高多邊形的角色模型、高解析度材質、動畫
數據、音頻和視頻資源文件等等。

儘管現代遊戲所使用的遊戲引擎和中間件能讓遊戲數據從本地硬碟載入到內存，關卡設計
師仍然經常被迫設計一些狹窄的通道、過道或電梯之類的場景來配合性能受限的傳統硬碟
和 I/O 流水線。這些小場景就是用來在後台不斷載入新素材並讓玩家保持沉浸感的方法
。微軟在與開發者討論次世代作品和現今作品遇到的阻礙後，Xbox 高速架構應運而生。

Xbox 高速架構由四部分組成：定製 NVMe SSD、硬體加速解壓模塊、直通存儲 API 層、
取樣反饋流（Sampler Feedback Streaming，簡稱SFS）。在這種架構下，開發者能夠迅
遊戲文件通常是經過壓縮後提供給玩家下載，縮短下載時間以及減小硬碟占用。這樣在調
用時自然就需要一定的性能做解壓操作。硬體加速解壓模塊就專門為此而設計，節省了大
約 4 顆 Zen2 CPU 核心的性能。該模塊支持業界標準 LZ 解壓，以及一個全新的、專為
材質數據設計的 BCPack 算法。兩種方法可並行使用來使得效益最大化。假設壓縮比為 2
:1 的話，Xbox Series X 就能為遊戲提供 4.8GB/s 的 I/O 速率，接近本世代主機的百
倍速度。

直通存儲API

電腦系統標準文件 I/O 的 API 已經沿用了超過 30 年之久，即便存儲技術已經發生了天
翻地覆的改變，但在這一層仍未有著多少變化。通過對遊戲數據獲取方式以及 SSD 技術
的分析，微軟意識到他們需要對底層 API 進行大幅度的翻新，由此誕生了直通存儲 API
。它能把更多控制權交給開發者來使用，比如用於建立多條 I/O 隊列、優先權控制和最
小化 I/O 延遲等等。直通存儲 API 封裝在 DirectX 標準當中。

取樣反饋流

基於以上三個技術，取樣反饋流這項技術才得以誕生。遊戲材質通常會隨著與玩家角色距
離遠近來動態改變其細節和解析度。然而存儲各種級別的材質數據需要消耗大量的內存空
間，即便是遠處低質量的物體同樣要把高質量的數據載入到內存當中。

Xbox 團隊很早就注意到了這個問題，在本世代的 Xbox One X 的 GPU 中內置了一枚特殊
然後微軟的新技術吹起來好像比索尼強耶....

比較可惜的是《力量》跟玩家的關係從這段文字裡面看不太出來，
頂多從語氣推測大概是朋友那樣的關係吧。

然後，除了文字內提到的，內心的勇氣才是力量以外，
其實有一個部分是中文沒有翻譯出來的，以下引用日文版：

よろしく代わりの咆哮、聞いててよね！せーの、がおー！
ふふっ。どうだった？あたしは《力》、司るは《活力》！
強くなりたい、強さがほしい！その願いに呼ばれたんだ！

腕っぷしだとか、最新の武器だとか、頭  数の多さだとか、
そんなのぜーんぜん強さじゃない！上っ面だけの偽物よ！
強さは勇  気
！逃げず最後まで  戦い  続ける、  揺るがぬ信念！
挫けそうなら、一緒に叫んであげるわ！がおー！ってね！

雖然意思上沒有沒翻出來的部分，但格式上中文沒有像日文這麼整齊
日文之所以整齊是因為他每一行都用到字數上限，
可能也是想用這個來象徵飽滿的精力正是《力量》的正體吧
然後，SV卡牌故事一行最多就是26個字，也剛好對應到力量的體質2/6
雖然這應該是巧合就是了


那麼，正位這麼有精神的《力量》，逆位可想而知就是無力、虛脫的意思
這裡就聽聽倒吊人是怎樣介紹《力量》的：
「《力量》所示為《活力》與《乏力》──
又或者是指給予、剝奪的存在。
如果看到祂戴上獅子的面具，可就要提高警覺囉！」

帶上面具指的就是這個形態：
虛脫的鋼腕（token） https://i.imgur.com/TE8aBIv.png
幫友方+0/+2的效果語音是：後よろしくー，就是自己不上場所以buff別人教別人上場
而且象徵原本剝奪力量的-2/+0要三回合後才會發動，可以想見逆位的《乏力》多沒幹勁

故事也很簡單：
毫。
無。
《力量》。

然後這裡也要看一下日文版本：
無。
  気
。
《力》。

雖然意思是一樣的，但日文版本有另一個藍色窗簾：
含標點符號共8個文字，對應到偉特版的8號。中文多了一個字，比較可惜

一道巨大的紅光閃過黑暗，它異常高的破壞力瞬間劈開了食死人小隊，被正中央擊中的三
人立刻昏死了過去


「什.....」


在那紅色閃光出現的瞬間，戰鬥就已經開始了。從每個走廊都閃出人影，學生伴著成人。
小仙女東施，金妮和張秋從右側殺出，剛才的強化擊暈咒大概就是這位年輕的正氣師發出
的，張秋和金妮在她身後瘋狂的施放屏障咒，東施舉起魔杖擺起架勢似乎打算在來一發


「是正氣師！！」食死人們一陣騷動，數道詛咒往他們的方向射過來，被金妮和張秋施放
的屏障給擋住，東施露出冷笑

「你們不應該出現在霍格華茲！」


「阿哇呾嗑呾拉阿！」同時三個食死人朝她射出索命咒，然而東施的身影在他們念完前就
已經憑空消失。金妮和張秋像是有第六感一般正好趴下躲過了一劫。下一秒東施的身影就
出現在不遠處，魔杖閃出紅光


護著綴歌的魯休斯馬份看準時機舉起魔杖


「噴噴障！」


巨大的紅光撞擊在空氣中，東施嘖了一聲後再度進入隱身



與此同時另一邊也打了起來。榮恩，路平和妙麗組隊從另一側殺出，三人一齊朝著走廊旁
的巨大石像射出紅色光束，瞬間爆開的石像碎片剛好招呼在食死人身上


「散開！」魯休斯大吼：「快散開！按照計畫行動！我們走，綴歌！」


眾食死人開始相繼使用幻滅咒，一個個消失在視野中。路平立刻舉起魔杖使出了某種反制
魔法，一道金色的光線照亮整個走廊，瞬間所有使用幻滅咒的巫師全部現形，但突然被照
出來的東施立刻就被三個食死人給包圍


「我們要趕快離開這裡...吉朋！卡羅！」魯休斯大吼著拉住一個矮小的食死人：「跟好
我！」


綴歌仍然不知道該怎麼辦


吼聲，尖叫聲，吆喝聲還有咒語與物體，咒語和咒語碰撞的聲音在走廊中雜亂地迴響，綴
歌依稀似乎還看見了剛趕到現場的麥米奈娃，史拉轟和浮立維。他們依序都被預先安排好
的食死人給纏住，魯休斯沒有停下腳步，而綴歌也沒有。


我到底該怎麼做？


「哈哈哈哈！更多！更多青春的肉體阿啊啊啊啊！」焚瑞灰背在戰場中央大吼，他的腳下
是昏厥的露娜羅古德


綴歌倒抽了一口氣，她....死了嗎？


「灰背你快過來！」魯休斯怒吼：「別忘了你的任務！」他吼完轉頭施放了兩個屏障，擋
下從浮立維那邊飛來的擊昏咒


灰背看著昏過去的露娜，一臉無辜的抬起頭：「可是她看起來很好吃欸！我好不容易才打
中她的說！」


「等我們完成任務再說！」魯休斯用咒語喚醒了身旁一個被史拉轟一個詛咒震飛到他腳下
的食死人，轉頭回去繼續怒罵：「黑魔王會給你獎賞的！你想要幾個少女都行！」

綴歌難以致信父親會說出這樣的話，但她仍然一臉蒼白地被魯休斯推著前進


灰背像隻(長得極醜的)小狗低下頭來，然後快步跟上魯休斯的隊伍


艾米克和艾朵倆兄妹在前面開路，而灰背則在最後面，一行人就這樣衝過了岔路口，抵達
了天文塔底部



「吉朋你從這裡釋放黑默標記。」魯休斯指著一旁的窗戶：「艾米克，艾朵，你們準備好
最強的障礙咒，我們要到塔頂等鄧不利多過來。」


矮小的食死人走到窗戶邊舉起魔杖，射出一道蜿蜒的綠色光線


「綴歌...？」他們身後響起一個聲音


奈威隆巴頓瞪著仍然被父親摟著的綴歌，他瞪大的眼睛裏滿是不解：「是妳.....放他們
進來的嗎？哈利他明明.......」


綴歌盯著他，她想要開口辯解，但是卻不知道從哪裡開口


我到底該做什麼？


如果她現在不做點什麼，這個戰場上會死多少人？會有多少學生被焚瑞活活吃掉？


但是她如果現在決定做什麼，她得先打暈父親，然後帶著昏迷不醒的父親逃出卡羅兄妹和
焚瑞，還有一個後面那些食死人的魔掌


我....我為什麼當時沒有立刻打暈父親，然後照原訂計畫在萬應室裡放火呢？


我為什麼沒有舉起魔杖？


綴歌說不出口，說不出這些人不是她放進來的。不管是誰修改她的記憶，修好了消失衣櫥
，通知父親鄧不利多今晚會不在，綴歌本來都有機會阻止這一切，但是她沒有


「青春的肉體啊啊啊！」灰背大吼著朝奈威撲了過去，他甚至沒想要使用魔杖。奈威奮力
一跳，彷彿不受重力影響般騰空而起，越過綴歌等人的頭頂，一邊用魔杖指著下面的魯休
斯


「去去武器走！」


魯休斯像揮蒼蠅一樣的甩了甩魔杖，擋下奈威的繳械咒。騰空的奈威落到吉朋的頭頂，正
好一腳踢開了他正要使出惡咒的魔杖


「你這混帳啊啊啊！」魔杖被踢開的吉朋又被奈威當成凳子，一腳踩在頭上。奈威將魔杖
對準窗戶邊，而吉朋則一個飛撲撿起自己地上的魔杖，立刻轉身大吼：


「阿哇呾嗑呾拉！」


但奈威對牆壁使出的咒語快了一步，將空中的自己立刻往後推了數米，剛剛好躲開了吉朋
的索命咒，還有魯休斯射出的擊昏咒


「劈啪！」


兩人的咒語在空中撞擊，索命咒偏離了軌道後集中天花板反彈了回去。奈威從另一側還未
落地，就被艾米克的擊昏咒給轟的老遠，但吉朋被反彈的索命咒命中，立刻倒在了地上


「該死！」魯休斯咒罵了一聲，但眼下沒有人在乎他們又損失了誰：「總之沒人了，我們
快走！」


他們踏上了樓梯，換做灰背走在最前面(他已經四肢著地的衝了上去)卡羅兄妹則在後面設
下一道一道的屏障。吉朋那無神的表情還烙印在綴歌的雙眼中，那是她此生第一次這麼近
距離的目睹死亡


「等一下就叫給妳收尾了。」魯休斯的聲音像是隔著一層薄膜：「由妳來給鄧不利多最後
一擊吧！女兒！」


「欸？」綴歌


「本來就是妳的任務不是嗎！」魯休斯低頭看著她的微笑看起來跟平時沒有兩樣，除了他
的臉比平常更白了些以外：「由妳來做個完美的收尾吧！讓馬份家再次成為黑魔王最鍾愛
的家族！」


「砰」地一聲，塔頂的門被打開了。魯休斯跟在灰背後面進去，而迎接他們的是一道藍色
閃光


「嘎阿！」


被咒語擊中的灰背瞬間雙腳離地，魯休斯沒有放過灰背幫他擋下這一發的機會，一個側身
躲開讓他被咒語的推力震飛出去，一路滾下樓梯，同時手一抬，魔杖對準施咒者


「去去武器走！！」




鄧不利多的魔杖飛了出去


「噢，晚安，魯休斯。」鄧不利多的語氣就如往常一樣一派輕鬆，好像是很普通的接見客
人。但是綴歌聽得出他的聲音極為虛弱，一隻手似乎已經完全焦黑了，而他正靠在天文塔
頂樓的牆壁上，雙腳不住的顫抖，似乎光是維持住不要倒下就傾盡全力了


「沒想到你也有今天阿，鄧不利多。」魯休斯得意的繼續將魔杖指著鄧不利多：「準備好
面對你的死亡了嗎？」


「我隨時都準備好的。」鄧不利多說的像是他「準備」的只是一趟旅行一般輕鬆：


「不過在那之前，我很好奇你們是怎麼帶著這麼多朋友進到霍格華茲的？」


「我的女兒修好了萬應室裡的消失衣櫥，從博金博克商店直接送人過來，就在你的鼻子底
下！」魯休斯像是在炫耀自己的女兒學校成績一般


「阿，厲害....厲害...」鄧不利多挑了挑眉毛：「不過綴歌小姐，我記得妳不是已經親
自來跟我匯報，妳決定放棄佛地魔給你的任務，並且已經摧毀了消失衣櫥嗎？」


他的目光直直射進了綴歌眼中，綴歌張開嘴，想要說什麼


我應該說什麼？


「什麼？綴歌....」魯休斯低頭疑惑的看了綴歌一眼，但是仍不敢把魔杖移開鄧不利多：
「妳......這是真的嗎？」


綴歌看著鄧不利多的眼睛，她想起鄧不利多的話

「我也要為妳的勇氣，至上最高的敬意

妳很聰明，如果妳真的能夠修好消失衣櫥，確實能夠繞過學校的防護措施，從外面帶人進
來

妳很優秀，綴歌。妳繼承了馬份家的魔法天賦，但是妳不需要繼承馬份家事奉黑魔王的歷
史。妳已經很強大了，而強大的人不需要透過傷害他人來彰顯自己，相反的，強者的價值
在於保護弱者。」


綴歌很願意相信鄧不利多的話，但是此刻的她似乎沒有這個選擇。


「鄧不利多！！」艾米克和艾朵跟了上來，艾米克大聲歡呼：「沒有魔杖的鄧不利多！被
逼到絕境的鄧不利多！」


「晚安，艾米克。」鄧不利多仍舊是那個像在歡迎賓客加入派對的態度：「你還帶了艾朵
來了嗎？真不錯....真不錯.....」


「你以為開開玩笑就能讓你逃過死亡嗎？」艾朵嫌惡的瞪著鄧不利多


「玩笑？」鄧不利多說：「不不，這是禮貌。我們回到剛才的話題吧，綴歌。」


他再度和綴歌眼神交會：「這是真的嗎？」


綴歌很想大叫：不是我！！


這一切都不是我幹的！


綴歌很想立刻就抬手擊昏父親，轉頭再送卡羅兄妹兩個擊昏咒，然後下去給不知道有沒有
意識的焚瑞兩個擊昏咒再把他丟下天文塔


但是她知道自己做不到，她攻擊父親的瞬間，卡羅兄妹就會將她制服。而鄧不利多再怎麼
法力高強，也已經對這個場面束手無策


「.....是阿。」


綴歌顫抖地說出幾個字：「我是騙你的......教授......好讓我不被懷疑，讓我的計畫順
利進行。」


卡羅兄妹放聲大笑


「你這蠢老頭，以為你那套愚蠢的教育方針還能用在現在的孩子身上嗎？時代早變啦！」


「真是愚蠢阿！黑魔王何必這麼懼怕你？我看我來宰了你就好啦！」


「不准！」魯休斯怒吼：「黑魔王已經指定了，由綴歌親自動手，就算她下不了手，那也
是我來！」


綴歌閉上眼睛，努力不讓眼淚潰堤


已經沒辦法了吧？我沒有選擇

馬份家世代都只能臣服於權力，只能服侍更強大的黑巫師。


很遺憾，我不是什麼有巨大勇氣對抗命運，即使失去生命也在所不惜的人


我只是個馬份。



「妳覺得哈利會希望妳怎麼做呢？綴歌。」鄧不利多緩緩地說



綴歌睜開眼睛，她的眼淚還是流了下來


但她仍然舉起她那不斷顫抖的手，將魔杖對準了鄧不利多，而鄧不利多也沒有移開視線，
他就只是看著她。



短暫的沉默降臨，綴歌紋絲不動，而她身後的食死人們也只是看著


「怎麼了？綴歌？」魯休斯低頭：「下不了手嗎？」


綴歌確實下不了手，但是她腦中現在想的不是這個，而是....自己的口袋熱熱的



綴歌左手伸進口袋裏，她摸到了那枚金幣，那個哈利和她對話用的金幣。而此時它正瘋狂
的發熱，上一段文字已經消失了，另一段文字正在出現，綴歌的手指不斷在金幣上來回摩
擦，總算理解了上面寫的是什麼



「Im here」



綴歌瞪大眼睛，往四周掃視了一遍。她當然什麼都沒看到，但是她想起來了


哈利在這裡！他一定在這裡！！！



彷彿一條迴路終於接上，綴歌的大腦開始狂飆了起來。哈利當然在這裡，他今天本來和鄧
不利多一起在研究分靈體！手上的金幣的文字是凸出來而非凹進去，說明他還能自由使用
魔杖。


「算了，沒事的，女兒。」魯休斯摸了摸綴歌的頭：「還是讓我來吧。」


「不！」綴歌的聲音回來了，她堅定的表示：「由我來解決鄧不利多吧，我可是黑魔王指
定的刺客。」


「哦哦？！」艾朵站在綴歌的左邊：「小姑娘終於長大了？」


綴歌露出微笑


計畫迅速在她的腦中成型，綴歌舉著魔杖的手停止了顫抖


你應該知道怎麼做吧？哈利。


鄧不利多看見了綴歌的表情，他非常輕地點了點頭



『哈利，只要有你在我身邊...』



綴歌深吸一口氣，手指將魔杖往前推，直到握著魔杖的幾乎只剩下食指和拇指



『....我就什麼，都不怕！』


綴歌瞪大眼睛，所有的思緒都集中了起來，集中到一句咒語


『去去，武器走！』


將魔杖反握，身體往右一斜，綴歌的魔杖朝著左後方的魯休斯射出藍色閃光


就在魯休斯整個人被震飛到牆上的瞬間，一隻手憑空出現在鄧不利多的右邊


哈利在我的左側！


看見哈利的綴歌立刻轉身，手中魔杖一轉，指向她右後方的艾朵，同時哈利指向綴歌左邊
的艾米克


「咄咄矢！」
「咄咄矢！」


同時出招，同時倒地


脫下斗篷的哈利將魔杖指著魯休斯


「什 ....什麼？」魯休斯完全沒有搞清楚狀況：「綴歌？....妳怎麼....」


「綴歌，這是怎麼回事？發生了什麼？」哈利劈頭就問



「還不能大意。」綴歌往前兩步往樓梯下看：「焚瑞灰背不知道還有沒有意識...」


「他們到底是怎麼進來的？！」哈利氣急敗壞近乎質問地問綴歌


「哈利，你也記得消失衣櫥被摧毀對不對？」綴歌的聲音異常冷靜，完全沒有被哈利的態
度影響到


「呃....對啊。」哈利回答


「是用噴火咒燒掉的對吧？」綴歌確認


「是。」哈利聽起來有一點疑惑


「我也是，但是我們的記憶被竄改了。」綴歌說：「消失衣櫥沒有被摧毀，甚至還被某人
修好了，於是我的父親才帶著食死人闖了進來。」


「什麼？！」哈利不敢置信：「我們....什麼時後被下咒的？！又是誰？！」


「不知道。」綴歌皺眉：「明明除了你和鄧不利多，沒有人知道消失衣櫥的存在才.....
」



「整整石化。」



綴歌話音未落，哈利的身體就突然僵住無法動彈。他只能瞪大眼睛，看著從綴歌背後現身
的石內卜
