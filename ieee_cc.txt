 標題  [推薦] 繼母的拖油瓶是我的前女友                                        
 時間  Wed Apr 15 01:22:14 2020                                               
───────────────────────────────────────

https://i.imgur.com/Uec8HVa.jpg
「過去是不會再回來的，那個時候的幸福，也是不會復甦的。」




















網誌好讀版:https://home.gamer.com.tw/creationDetail.php?sn=4749871

　　本來是不會碰這種書名充滿杏仁味的作品，但是意外看到了推薦文，才鼓起勇氣翻閱
這部作品。意外的，這是一部進到我好球帶的正中央作品，我一直都很喜歡這種探討人際
關係、尋找自己的定位，還有交雜的思緒與情感的拉扯，像這樣子的作品。

　　本作的作者在後記提出了一個很有趣的比喻：「人是像雪一樣的東西，一開始很容易
留下腳印，但是會逐漸地被踩硬，變得光滑、圓滑。高中時期大概是開始變得光滑的階段
。」跟我的想法不謀而合，高中就是有點想法、有點固執，卻還對他人保有依賴性的時期
吧？就是這種不確定的樣貌與形狀，才有這麼多的故事可以寫，也成為群像劇最喜歡的題
材。

　　「時間的齒輪將永遠轉動下去，停下的總是自己本身。」（櫻子小姐的腳下埋著屍體
），故事中的男女主角：結女、水斗在國中時期，因為一些誤會鬧了彆扭，卻因為不夠成
熟而成為分手的導火線，兩人的時間也在這時候停止了。這樣子的兩人，卻因為父母的再
婚而成為了家人，因為顧慮重要的父母，而不得不在同一個屋簷下扮演和樂融融的家人。
　　隨著故事的推進，對對方的了解也慢慢加深，沒用的、可笑的、懦弱的這些熱戀時觀
察不到的部分，也漸漸的展現出來，慢慢從熱戀時的完美超人，轉變成為一個有血有肉的
人，更加理解對方的兩人，其中一人的齒輪慢慢地開始轉動了。

　　「人類是一種會順著周遭的環境和期待扮演各種角色的生物，有人感到痛苦、有人毫
無知覺、有人認為是長大的表現，也有人認為這是對世界的妥協。」這是作者提出的一個
很重要的想法，作品中的角色們，都有著自己扮演的角色，當自己的角色面臨崩壞的時候
會感到恐懼，害怕不再是大家喜愛的那個角色的話會失去自己的容身之處。但是，作者給
他們一個溫柔的回答：「人不是只會喜歡別人帥氣的地方。」這大概也是作者在人生中所
找到的答案吧？

　　這部作品很甜，不坦率的冤家姊弟和另一對青梅竹馬，明明很在乎，卻總是賭氣般的
言語帶刺，然而在重要的時刻總是義無反顧地照顧對方；這對主角很幼稚，死要面子不願
意表達自己的歉意，滿心的好意卻總是要包著帶刺的包裝才敢遞出。
　　但是啊，他們還年輕，繞遠路不就是孩子們的特權嗎？這個時候他們還有犯錯的餘地
，還可以憑著一根筋去碰撞、去了解，可以在衝突中理解對方，可以在碰撞中塑造自己的
樣貌，就讓我們用更加溫柔的目光去守望他們吧！
　　「雖然可能會失敗，但沒有正面沖突一下，也很難成為真正的朋友。該怎麼說呢？要
是只選擇安穩的方法，可是會失去真正重要的東西。」（心連‧情結）


　　雖然這部作品是一顆會炸出砂糖的閃光彈，但是比起閃光的部分，更讓我喜歡的是那
些情感、人際關係還有價值觀的碰撞。作者用細膩的文筆，將那些複雜的思緒一一地攤開
出來，讓我可以很直接的觸碰到角色們的內心，了解他們的恐懼、他們的顧慮，還有他們
內心真正渴望的事情是什麼，會讓人心中一緊，同時期待故事的發展。
　　另外一個很有趣的地方，在書中可以感受到作者文學底子，一些知名的文學作品作者
都有較為深入的理解，但卻不是淺顯的引用名言佳句，而是將那些作品的概念、想法融入
故事裡面，用自己的方式重新闡述，為作品增添風味。我想故事中所謂的濫讀派，還有那
些角色們所喜愛的作品，都是他自己的投射吧。


　　最後我想說的是：
　　台版快點出版！
　　我一定第一時間下訂單！還會推薦給別人！
　　而且拜託拜託，不要幹出拿香蕉請猴子翻譯這種蠢事，這種類型的作品，如果沒有好
的譯者是會被毀掉的啊！

基本上跟交不到女朋友的魯蛇八卦肥宅概念很像就對了。

不只是性格內向不善社交那麼單純而以，

給很多反對他的人留下的印象，

不介意承認自己跟病嬌大使一樣變態，
但是宅男最可怕的不是別人歧視你，
而是你自己爛還不願意承認。

病嬌大使的狀況我相信也是這樣，
再怎麼變態，我相信很多人其實是不介意的，
畢竟這是你個人的性取向嘛，
可是影響到你做人處事的態度，
那要不惹人反感都難。

相信大家都知道，
病嬌模擬器就是扮演一個有病的女學生，
把自己愛慕的學長身邊，
所有接近的女生都殺掉的遊戲。

而主角如果在遊戲中靠近學長，
會因為害羞失去大部分行動能力，
然後處於這個狀態太久，
就會直接 Game Over。

害羞所以操控被限制住很好理解，
但為什麼待太久被學長發現就會 Game Over？
不覺得很奇怪嗎？這有三小意義？

結果病嬌大使有在一次留言中提到，
他會這樣設計，
是因為他不想看到一堆玩這個遊戲的人，
故意跑到學長旁邊繞來繞去，
然後不斷喊 Senpai notice me 的梗。

所以你想靠近學長？給你 Game Over。

...是在哈囉？
這種事重要嗎？連這也要管？住海邊？
因為自己討厭這梗就要不讓玩家這樣玩？

從這一點就可以看到，
他在做這個遊戲時，
基本上不在乎金主、不在乎玩家，
各種遊戲裡面的設定，
很多是自己任性而為。

另一個例子是 skirt inventory，
病嬌大使希望能把主角的"物品欄"做在裙子裏面，
然後打開物品欄時，主角就會直接把裙子拉開...

這個概念我相信這邊的變態鄉民大多數都會喊好，
當然也是有人會覺得這樣不行，
重點是，病嬌大使因此辦了一場投票，
看看他的粉絲偏好怎麼做。

投票的選項有3個：

A.拉開！直接看到內褲！
B.拉開，但是內褲要打馬賽克。
C.別這麼做。

投票的結果，
我記得 B 最多票，A 略輸 B，C 沒什麼人。
結果病嬌大使的回應，
是說他覺得還是要做 A 比較好啦怎樣怎樣...

那你他媽的弄投票是弄辛酸的逆？

病嬌模擬器的政治不正確，
真的不是他失敗的原因。
(至少現在還不是...反正大概沒機會了)

雖然病嬌大使還透漏過，
他希望把學校加上女學生被人口販賣的設定，
但就算是這種會招來一片罵聲的成人內容，
病嬌大使還是努力想打擦邊球，
想把病嬌模擬器做成一個非成人向的遊戲。

要知道病嬌模擬器都有把同學綁架回主角家虐待，
將對方轉化為自己奴隸的遊戲機制了，
為什麼不乾脆一點，直接把這遊戲改成18禁？
我不知道，也許是因為病嬌大使的金主裏面，
有一堆人是未成年的孩子吧。

說到遊戲的機制，如果我沒搞錯的話，
病嬌模擬器這 IP 一開始會紅，
主要是可以扮演一個日本動漫女高中生，
在一個校園沙盒內嘗試用各種方法殺同學，
這種新奇感吸引了很多人幫他宣傳才出名的。

問題是，光是放任你在校園裏面殺人，
沒有劇情又沒有目標，
這樣實在是很難算一個完整的遊戲，
於是病嬌大使就開始了加入劇情模式的漫長道路...

現在的病嬌模擬器劇情基本上會是這樣：

主要遊戲故事會有10週，
每一週學校都會有一個新的女孩子出現，
依序是傲嬌幼馴染、烹飪社長、話劇女王、神經病、肌肉腦袋、
色情護士、淫蕩老師、不良少女、學長妹妹、女神會長。

每一週出現的女孩都會接近學長，
所以要在該週之內把那女孩幹掉，
免得她變成學長的女朋友。
(是說第9週的妹妹...算了這其實是能坳回來的)

大原則上我覺得這並沒有什麼問題，
至於實際上到底好不好玩，
就完全看遊戲作者怎樣去設計並執行了，
結果嘛...

目前經過這麼多年的開發，遊戲主要的故事模式，
還卡在第1週的傲嬌幼馴染這邊，
遲遲沒有發佈。

照病嬌大使的說法，
是說幼馴染其實已經開發完成了，
只是他還在修遊戲中的 bug，
等到一切完善以後就會推出來給大家看。

當然過了這麼多年了，
許多人看著常常在直播玩其他遊戲的他，
已經很難接受這個理由了。

可是即使不考慮速度與效率的問題，
病嬌模擬器的設計概念，
現在也是失控火車。

前面說了，這遊戲一開始會吸引人來玩，
其中一個要素，就是能嘗試各種殺人手段的沙盒模式。
那麼，作為第一個 Boss (大概吧) 的幼馴染，
病嬌大使安排了什麼特別的橋段給玩家呢？

他直接替幼馴染安排了一個叫做"對手醬"的跟班，
全程保護幼馴染的安全，
對手醬武力 120%，偷襲殺不死，
沒辦法用任何方式引開，
會提醒幼馴染避開所有陷阱，
簡單來說，對手醬，無敵。

然後他藏了一種可以成功殺死幼馴染的方法，
要玩家找出來用這個方法通關。

這種限制玩家的做法，直接破壞了玩家的自由度，
更別說病嬌大使還喜歡拿刺客任務系列，
來辯解開發遊戲有多困難多耗時，
結果人家的任務裏面各種多樣性可以選擇，
病嬌模擬器卻是作繭自縛。

第一個對手都這樣了，那之後怎麼辦？
是要逐漸開放更多殺人方式給玩家嗎？
可是一開始限制玩家選擇，
後面解鎖更多選擇，
這樣不是越玩越簡單，難道會只剩劇情？

說到劇情，病嬌模擬器也一樣沒救，
其實關於遊戲的設定資料已經很多了，
問題是，都是一些沒用的東西。

很久沒關注的人可能不知道，
病嬌模擬器的遊戲裏面，
現在有 2D 小遊戲可以玩囉！

首先是一個女僕打工遊戲，
主角要透過這個遊戲賺錢，
然後可以買遊戲中要用到的道具，
雖然主線沒完成，但遊戲又變得更複雜了，
真是太好了呢。

然後，主角還可以回自己的房間裡面打電動，
其中一個是直接致敬惡魔城的橫版遊戲，
叫做病嬌城：月下的學長。
還有一個叫魔法少女美幸的彈幕射擊遊戲，
遊戲中的魔法少女不但又是另外的角色，
他們還有自己的故事，
劇情據說會像 EVA 一樣超展開喔。

這他媽的重要嗎？

就問問你們，如果你是出資的金主，
看到他主線故事遲遲不完成，
先是推出一堆五四三的，
什麼巨人模式、伊波拉模式、絆愛模式、SANS 模式，
已經很讓人不耐煩了，
結果又去搞附加小遊戲跟衍伸設定，
會讓人作何感想？

據說主角的媽媽當年也是病嬌喔、
學生會副會長好像很忌妒女神會長喔、
學校裡面好像藏了什麼不可告人的陰謀、
神秘社的神經病社長一直在跟蹤某同學喔、
那個被跟蹤的同學長得跟附加小遊戲裡面登場的魅魔一樣喔、
有不同的時間線，會出現追跡者喔...

還好病嬌大使還有一些鐵粉，
要是一般的遊戲公司這樣搞，
讓人等了這麼久還不幹正事，
早就被消費者給放生了吧。

或許是病嬌大使太過眼高手低，
想做的東西太多，但有能力實行的太少，
才會變成現在這種慘況。

能力不夠其實也沒有關係，
除了把目標縮小以外，
讓其他人來幫自己的忙也是可以的，
把程式跟美術發包出去，
自己專心做遊戲企劃，
我想正常人都會覺得這樣最合理。

結果病嬌大使是怎麼搞的呢？

程式，他堅持要自己來，
儘管他的程式能力完全不行，
已經有很多人批評過，
遊戲程式碼亂七八糟，
還只會用 if / else 這種簡單語法，
把程式搞得過度冗長累贅，
結果每幀都要執行成千上萬行程式，
導致遊戲延遲得跟垃圾一樣。

在開發期間，有間叫 Tinybuld 的公司，
看上了病嬌模擬器這個 IP，
跑來跟病嬌大使合作。
他們派來了一位工程師，
幫病嬌大使把程式從 Java 改成更適合的 C#，
然後整個重寫一遍，
結果本來打開要等好幾分鐘的遊戲，
一下快到幾十秒就能打開了，
好棒棒！

然後呢？病嬌大使不喜歡他的程式，
就跟對方解除合作回來繼續用自己的程式了。

如果你是遊戲中的病嬌醬，
聽到這邊你有沒有一種想跳樓的衝動。

程式說完了來說 3D，
其實我一直覺得很奇怪，
病嬌大使到底哪裡覺得自己適合開發遊戲，
企劃不行、程式不行，
那 3D 呢？也不行啊。

病嬌模擬器裡面粗糙的 3D 模型被罵很久了，
當然，有些人可能會覺得這樣的風格還好，

別人做好的素材包。

之後會改用自製的模型，

說真的 3D 軟體雖然不好學，

遊戲資料夾裏面的拌愛模型，
