: 請問,

: 此排除BB或兔王或禮裝等等,機率重算的機制~

第一回合的牌是從15張抽5張，所有組合為C15取5=3003
抽到某指定從者3張以上的組合是
C5取3*C10取2+C5取4*C10取1+C5取5*C10取0=450+50+1=501

->501/3003=16.68%

因為三個回合是全同可以互換，
不過為了多騙點P幣就繼續算下去

第二回合的牌是第一回合抽剩的=C10取5=252
扣除已經抽出3張以上的組合
第一回合抽到2張=C5取2*C10取3=1200
-> 第二回合抽到0張=C3取0*C7取5=21  ->第三回合3張
-> 第二回合抽到1張=C3取1*C7取4=105
-> 第二回合抽到2張=C3取2*C7取3=105
-> 第二回合抽到3張=C3取3*C7取2=21

第一回合抽到1張=C5取1*C10取4=1050
-> 第二回合抽到0張=C4取0*C6取5=6   ->第三回合4張
-> 第二回合抽到1張=C4取1*C6取4=60  ->第三回合3張
-> 第二回合抽到2張=C4取2*C6取3=120
-> 第二回合抽到3張=C4取3*C6取2=60
-> 第二回合抽到4張=C4取4*C6取1=6

第一回合抽到0張=C5取0*C10取5=252
-> 第二回合抽到0張=C5取0*C5取5=1   ->第三回合5張
-> 第二回合抽到1張=C5取1*C5取4=25  ->第三回合4張
-> 第二回合抽到2張=C5取2*C5取3=100 ->第三回合3張
-> 第二回合抽到3張=C5取3*C5取2=100
-> 第二回合抽到4張=C5取4*C5取1=25
-> 第二回合抽到5張=C5取5*C5取0=1

->(21/252)*(1200/3003)+(66/252)*(1050/3003)+(126/252)*(252/3003)=16.68%

第三回合直接把第二回合的計算拿來就行
->(21/252)*(1200/3003)+(66/252)*(1050/3003)+(126/252)*(252/3003)=16.68%

結論就是某個從者在第一回合打出EX攻擊的機率是16.68%
一輪發牌的三回合中能打出EX攻擊的機率是50.05%
