※ 引述《attacksoil (第三方程式)》之銘言：
: 最近想開始玩麥塊
: 看了一些網路上的文章
: 似乎有三個版本 手遊 Java Win10
: 看比較以前的討論比較推薦Java版 但不知道現在2019年也是嗎？
: 希望能用XB手把在電視上玩 一開始應該只會單機玩 但有機會也想看能不能玩其他人架的
: 伺服器
這有滿多可以說的
麥塊版本的主要兩個大類別是java版和基岩版兩種
底下提到的基岩版主要為win10版

首先是mojang最早開發的是java版
也就是一般人稱呼的PC版
後來微軟用很多很多很多很多很多很多很多很多很多很多很多鈔票把mojang買下來後
就自己再用C++重新寫了一個基岩版

基岩版和java版基本上遊戲內容是相同的
只有在小部分地方有差異（像是紅石行為不太一樣）
但兩邊的每次大更新的主題都是一樣（發布的進度也接近）

關於使用資源的話
基岩版因為是用C++來寫所以有不錯的效能
java版則是掛著一個java來執行所以效能會較差
但我也是看過有人提出基岩版其實是用偷吃步的方法所以效能看起來好像很好
也看過有人提說java版其實也可以調整出很好的效能
不過這兩個論點我都沒有再深入研究過所以就不細說了

遊戲的擴充部分的話
java版有著大量的MOD而聞名
但由於mojang並沒有開放java版的api給大家使用
所以基本上java版的mod是需要另外再外掛引擎的
這種外掛引擎需要像是破解遊戲一樣弄出api給moder們使用
所以當java版有進行大更新的時候
外掛引擎是沒有辦法及時跟著反應的
也讓各mod在遊戲版本上有很大的隔閡（最新版本的遊戲絕對沒有大型mod能立刻跟上）

像是看過有人宣言自己是永遠的1.7.10住民
也因此出現了一些奇妙的現象
史萊姆球是在1.8時實裝進遊戲裡
原本是只要把遊戲升級到1.8就能立刻使用史萊姆球（java可以自由選擇遊戲版本來遊戲）
但為了要繼續停在1.7.10玩這版本限定各種mod
結果變成有人特別寫出1.7.10版的史萊姆球mod加入遊戲之中
讓不升1.8的人也能藉由mod用到1.8的遊戲內容

此外使java版用mod玩遊戲是一定會讓效能變差（尤其在1.9後吃掉的記憶體暴增，謎）
所以需要去確認java的參數設定來提高記憶體的使用上限
在玩大型mod模組包時情況會更為嚴重
光是一開始打開遊戲的載入都必須花上好幾分鐘才能完成
為此還有人寫了這種mod給玩家使用
"當載入完畢完成開啟遊戲後，會發出鈴聲通知玩家"

以前曾經和幾個朋友開伺服器玩（1.9的時候）
在大家討論之下決定試試看mod伺服器（tinker、botania等幾個mod，也沒裝很多）
結果沒想到光是測試時就讓其中一位朋友放棄參加了
因為他的電腦性能完全無法負荷
最後是決定放棄不參加

這種mod伺服器是需要每位加入的玩家都必須要裝入相同的mod才能進入
但還有另一種java版伺服器不需要裝mod也能擴充功能
這種叫做plug-in
擴充的遊戲內容靠遊戲提供的各種指令來編寫並放入伺服端
這種伺服器只要使用vanilla（無mod的原版稱作vanilla版）就能進入來玩擴充內容
plug-in能做到的東西還滿多的
像是寶可夢伺服器就是這樣子的模式
完全不需要裝mod也能夠抓寶可夢或是對戰（曾聽說是用一堆盔甲架寫出來的）

雖然mod提供很多遊戲內容
我自己是建議即使先從java版開始接觸的話
還是先從vanilla開始熟悉遊戲內原本的內容比較好
不然mod一次加入太多東西可能會不知道該怎麼著手
也看過有實況玩家在實況中發現不同顏色的羊毛無法做床
接著他的反應是找這配方是被哪個mod改掉了
其實這是在1.10調整的配方而不是mod提供的配方
不過也是有mod會顯示各物品是從哪個mod來的
這個倒是很便利而不太會被一大堆物品弄混
vanilla能玩的東西其實也是非常多
也看過有玩家玩vanilla就弄出一堆像是大型工業mod規模的裝置

至於win10版雖然無法裝mod但也是有擴充內容叫做add-ons
這個主要是要在微軟提供的市集裡面發布
我自己是沒怎麼接觸所以就不多說了

在連線方面
java版自然是只能和java版連線
連線時需要注意版本是不是正確
如果是mod伺服器更是要注意是不是需要的mod都有裝上而且mod的版本是否正確
基岩版則是能和全部平台的基岩板來連線，除了索尼平台以外
曾看過有人問他的NS版要怎樣和朋友的PS4版遊戲
看到這問題也只能回答索尼平台不給cross play，無解

最後說購買須知
以前微軟曾經有提供過一個活動
就是買java版會送win10版
擁有java版的玩家可以直接在mojang網站內領取win10版序號
當時我都是推薦說絕對不要買win10版
因為買java就會直接送（而且win10還比較貴，謎）
只是仍是偶爾會看到有新玩家在微軟市集看到有麥塊就直接先買下去了......

這個活動在現在是結束了
不過所謂的結束是指在活動結束的那一天後買java版的玩家無法領取win10版序號
在活動結束之前買java版的玩家仍是可以去領取win10序號
另外在活動結束前後其實微軟有分別在便利商店發售"兩種"麥塊的java版序號卡
因此就算是現在贈送win10版活動結束了
但只要找到在活動結束前還沒賣完的舊版java版序號卡
依然是可以再領取win10版序號的

: 麥塊版本的主要兩個大類別是java版和基岩版兩種
: 底下提到的基岩版主要為win10版
: 首先是mojang最早開發的是java版
: 也就是一般人稱呼的PC版
: 後來微軟用很多很多很多很多很多很多很多很多很多很多很多鈔票把mojang買下來後
: 就自己再用C++重新寫了一個基岩版
: 基岩版和java版基本上遊戲內容是相同的
: 只有在小部分地方有差異（像是紅石行為不太一樣）
: 關於使用資源的話
: 基岩版因為是用C++來寫所以有不錯的效能
: java版則是掛著一個java來執行所以效能會較差
: 但我也是看過有人提出基岩版其實是用偷吃步的方法所以效能看起來好像很好
: 也看過有人提說java版其實也可以調整出很好的效能
: 不過這兩個論點我都沒有再深入研究過所以就不細說了
: 遊戲的擴充部分的話
: java版有著大量的MOD而聞名
: 但由於mojang並沒有開放java版的api給大家使用
: 所以基本上java版的mod是需要另外再外掛引擎的
: 這種外掛引擎需要像是破解遊戲一樣弄出api給moder們使用
: 所以當java版有進行大更新的時候
: 外掛引擎是沒有辦法及時跟著反應的
: 也讓各mod在遊戲版本上有很大的隔閡（最新版本的遊戲絕對沒有大型mod能立刻跟上）
: 像是看過有人宣言自己是永遠的1.7.10住民
: 也因此出現了一些奇妙的現象
: 史萊姆球是在1.8時實裝進遊戲裡
: 原本是只要把遊戲升級到1.8就能立刻使用史萊姆球（java可以自由選擇遊戲版本來遊
戲）
: 但為了要繼續停在1.7.10玩這版本限定各種mod
: 結果變成有人特別寫出1.7.10版的史萊姆球mod加入遊戲之中
: 讓不升1.8的人也能藉由mod用到1.8的遊戲內容
: 此外使java版用mod玩遊戲是一定會讓效能變差（尤其在1.9後吃掉的記憶體暴增，謎）
: 所以需要去確認java的參數設定來提高記憶體的使用上限
: 在玩大型mod模組包時情況會更為嚴重
: 光是一開始打開遊戲的載入都必須花上好幾分鐘才能完成
: 為此還有人寫了這種mod給玩家使用
: "當載入完畢完成開啟遊戲後，會發出鈴聲通知玩家"
: 以前曾經和幾個朋友開伺服器玩（1.9的時候）
: 在大家討論之下決定試試看mod伺服器（tinker、botania等幾個mod，也沒裝很多）
: 結果沒想到光是測試時就讓其中一位朋友放棄參加了
: 因為他的電腦性能完全無法負荷
: 最後是決定放棄不參加
: 這種mod伺服器是需要每位加入的玩家都必須要裝入相同的mod才能進入
: 但還有另一種java版伺服器不需要裝mod也能擴充功能
: 這種叫做plug-in
: 擴充的遊戲內容靠遊戲提供的各種指令來編寫並放入伺服端
: 這種伺服器只要使用vanilla（無mod的原版稱作vanilla版）就能進入來玩擴充內容
: plug-in能做到的東西還滿多的
: 像是寶可夢伺服器就是這樣子的模式
: 完全不需要裝mod也能夠抓寶可夢或是對戰（曾聽說是用一堆盔甲架寫出來的）
: 雖然mod提供很多遊戲內容
: 我自己是建議即使先從java版開始接觸的話
: 還是先從vanilla開始熟悉遊戲內原本的內容比較好
: 不然mod一次加入太多東西可能會不知道該怎麼著手

: 其實這是在1.10調整的配方而不是mod提供的配方

: 這個倒是很便利而不太會被一大堆物品弄混

Q：有繁中和日版，該玩哪個？

新手有學院系統保證送你5星角。
日版的角色池比較深可是也比較難抽。

當然日版現在的遊戲內容還是比較豐富，尤其角色多很多。
想好好享受遊戲的人還是建議玩日版。

對遊戲劇情跟世界觀有興趣又看不懂日文的人
可以先玩繁中看看劇情。
想要長期玩再看要不要跳日版繼續玩下去。


Q：需要刷首抽嗎？有沒有推薦的角色？
A：台版完全沒必要，現在角色池太小，開場還有學園送你五星角。
台版真的想刷就刷小紅帽跟灰姑娘吧。

日版首抽範圍就很大了。
原則上角色池列表裡越上面的越新也就越強，要刷就盡量刷到上面的角色。

Q：打開來一大堆東西能打，新手開場要打什麼？
A：台版就乖乖去推主線等活動吧，順便把石頭都存起來。
早期出的一些角色都很廢= =
石頭存著等美女與野獸的活動再去拚美女比較好。

無論台日版新手都應該去刷星期三的每日關卡。
有多少體力就刷多少。
在工房要做武器裝備食物道具的材料全都在星期三打。


Q：簡單介紹一下世界觀？
A：主角是人界的普通人類，意外認識來自童話世界的女主角。
女主角看上主角身上的神聖之力，就把主角帶進童話世界裡了。

童話世界正苦於神聖力大減，各種魔物橫行，連很多童話角色都跟著失控暴走。
為了拯救童話世界，主角一路用神聖力幫下面幾個同伴補了魔力：
灰姑娘、小紅帽、桃樂絲、汀卡貝兒

在依序都快樂地跟主角補了魔力之後，一行人就這樣踏上旅程。
日版二周年已經把主線第一部完結了! 又多了幾個同伴。
對劇情有興趣的可以從台版開始看看劇情。


Q：我看不懂這遊戲怎麼玩！
A：正常。
童話前線的複雜度意外的高。
這遊戲沒辦法光是把強角放上去就輾壓所有關卡。
有新角就直接無腦放上去，基本上就等著被王的特動暴打。

隊伍的配置、技能的先後順序、武器裝備的調配等等，對戰鬥的影響非常大。
不過研究這些大多數人大概都要燒腦了。
喜歡認真鑽研遊戲內容的人，這遊戲的深度絕對夠你鑽研。

只想輕鬆玩的人，童前的活動難度其實也都不高。
在活動獎品幾乎都很好的情況下，童前對每天的遊戲時間要求也不多。
一天清個兩三次體力大概就夠拿完大部分的獎品了。
休閒玩家每天遊戲時間頂多一兩小時。
