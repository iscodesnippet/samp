懇請連署調整稿費免稅額和認定標準：
https://join.gov.tw/idea/detail/62514356-310e-46cd-ad19-c3eba7797547


各位版友好：

許久沒來到這個版了，進來這裡其實有點感傷。

我是一名「前」輕小說譯者，

幾年前因為身體和家計因素離開了輕小說的世界。

雖然我總是在潛水，

很感謝這個版的讀者曾經討論我的譯作，

也沒有人批評過我翻譯不好，不然我可能早就離開翻譯這一行了。



不好意思前言說得有點長又低氣壓籠罩的感覺，

其實這篇文的主要目的，

是為了一項攸關藝文工作者的公共議題要拜託大家。

台灣的藝文工作者包含作家、書籍譯者、漫畫家等等，

領取的報酬是稿費或版稅，政府為了獎勵著作，

給了稿費18萬的免稅額，

但是大家知道稿費的免稅額已經多久沒調整了嗎？









答案是「30年」，不用懷疑自己的眼睛，真的就是這麼久，

久到一個當年出生的小嬰兒現在都可以當爸媽了。


稿費的免稅額定額三十年，早就跟不上物價漲幅，

而且認定標準還侷限在紙本媒體，

沒有納入網路時代的電子媒體，

都是明顯跟不上時代的規定。

因此我在政府的公共政策參與平台發出提案，

希望調整這些不合時宜的法令，

在如今稿費微薄的時代，

讓優秀的創作者可以受惠，多一點意願留在業界，

各位讀者才能繼續欣賞他們的智慧結晶。

提案連署需要五千個附議，政府就會出面回應。

希望各位支持優秀作品的讀者能一起站出來，

連署附議並轉發這個提案讓政府聽見大家的聲音，謝謝！

https://join.gov.tw/idea/detail/62514356-310e-46cd-ad19-c3eba7797547

個人覺得真的是滿有趣的一種感覺差別
明明骨子裡幾乎87%像，但玩起來感覺卻差滿多的

傳說對決：
很久以前玩的，不知道現在環境如何
只記得各種神奇的削弱跟強化，每次出新角就把強的舊角打成廢角
平衡爛得要命，就算我很久沒玩，我想大概現在也差不多吧？
角色出的最快，但技能創意我認為是三款中最低的，很多技能都只是排列組合不同
不過還是最多人玩的一款，排的速度超級無敵快，當然白癡也很多就是
美術....偏大陸風，個人不是很愛
詳細可能要看其他人怎樣補充跟討論了，太久沒玩


決戰平安京：
角色出得很慢，人物風格偏韓國+日本
平衡我認為是三個裡最好的，雖然有幾隻是太扯，但BAN的位置還勉強夠
而且最重要的是那幾隻真的都很難玩...高投資高報酬
技能我認為是三個裡最有創意跟特色的
一堆角色你看完說明文都看不懂他在說啥小，有夠長，還一堆專有名詞
對線期也還滿有趣的，是真的有在對線跟GANK的交流
順風跟逆風的節奏也還勉強平衡，不會很容易一路輸到脫褲
最大的問題應該是連線品質很差，雖然比以前好了
但排到一堆韓國仔時還是會爆PING....



非人學園：
角色出的也還算慢，人物風格偏日本，但僅限圖片，人物模組....差很多
模組本身也不是到很慘烈，但就是跟圖片差超多這樣
技能創意還OK，但平衡方面偏糟糕，很多隻你很明顯能感受出設計者根本隨便在設計的
然後最神奇的就是這款的經濟，非常依靠殺人，認真對線的根本沒幾個
只要殺到幾個基本上經濟就頂天，死都不參戰認真農兵的，比跟團的輔助還窮....
滾雪球非常嚴重，而且裡面還有兩隻扯到爆的NPC，擊殺後幫忙推塔
這個平安京也有啦，但強度完全不能比，如果放著不管，這NPC可以一路從外塔推到快內塔
導致認真對線的更像白癡，塔直接被灌爆
不過很神奇的，相對於滾雪球，這款反敗為勝的機率卻是最高

2. 課時很長，不確定自己會不會懶得把課上完，而且總價小高，需要觀望一番，不過聽
說遇到問題老師都會幫忙回答，不會放生你
3. 某些功能只能用APP操作，課程是電腦手機都可以看，但我個人不愛用手機上課，螢幕
太小惹
4. 雖然他們強調母語式教學，但我們就身在非英語系國家，母語式其實無法100%達到（
？

-Vclass
好像是voicetube今年新出的預錄影片式課程的新品牌，跟jella感覺有一點像，目前專攻
英文領域的課程，我比較的是Lily的英文聊天術。他們的課程因為比較新還沒有朋友有買
過，這裡的比較是看官網和課程示範影片得出來的

優點：
1. Lily老師在加拿大留過學，是語言教育專業，應該是很懂這個領域的教學內容，且配
合真實生活中的情境對話，可以最直接應證上課內容是否與實際相符
2. 一次拿出的錢不會是這麼大的數量，如果課真的很爛的話，也不會超級痛心（但3600
也不是說超級便宜啦ㄏ）
3. 課程可以永久觀看，沒有期限
4. 一堂課時間短，很符合現代人沒辦法長期專注的痛點

缺點：
1. 課長很短，總共只有8小時，且看目錄都是從打招呼開始，不知道是不是真的上完就
能說出自然、有深度的英文。而且價錢除下來其實CP值不算高，即使售價看起來較親民～
2. 好像沒有APP可以看課程，會受限於器材，只能在有電腦的情況下學習
3. 沒有體驗課程，只看課程示範影片的話，課程設計感覺就是看影片，比較沒有練習的
環節
4. 品牌才剛剛成立，沒有足夠的成功樣本數和評價，難以分辨這堂課是否真的有用，或
是課程內容品質高低

-Hahow/Yotta
這兩個平台會放在一起是因為他們開太多種類的課了，而且都是比較小型的，課長大概都
只有四、五小時左右。我自己有看過幾堂朋友買的Yiling開的口說課，其他朋友也買過這
兩個平台的其他語言課

優點：
1. 課程價格平均較低，可以以實惠的價格買到想學的內容，不小心買到爛課的話只會罵
聲幹然後遺忘這件事
2. 平台已經經營一段時間，有足夠的知名度和可信力，且購買前都有課程評價可以先了
解，購買課程後也有問題討論區可以問老師問題和互動

缺點：
1. 課時短（總共4,5小時），如果不想要只學到冰山一角，通常還得買其他的課程補
充說明這個課的內容
2. 課程內容不夠完善、架構太大且模糊，感覺是有很企圖心想要一次給學生很多東西，
但反而課程規劃變得有點虛有其表（這是上完我自己買的課，和聽兩個朋友上完他們買的
課之後得出的共同結論）

[結論]

英文基礎已經很不錯，想練習聽力居多，考試導向 -> Hero
覺得自己好像會但其實都說不出來，追求學習完整性，特別想學習口說 -> Jella
不想一次砸下太多錢、想上小一點的課程，學局部的重點就好 -> Vclass、Hahow、Yotta

因為我個人這次比較想要找的是完整、大架構的課程，而且想要可以直接運用在生活中
還能只買一門課就得到所有想學的內容 不用再花更多錢買其他課
評估完內容完整性、架構編排、CP值和我的需求之後
目前的排名是Jella>Vclass>Hahow=Yotta>Hero
