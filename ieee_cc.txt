: 小萌老師在課堂上提到）
: 半個世紀前，御船千鶴子的數據被用來培養學園都市最初的能力者『發源
: 檢體』『Beginning Child』（星門計劃）。今天的學園都市，是參考了發源
: 檢體的構成構建起來的。
: 隨後世間對超能力持懷疑態度，御船脫離理事長亞雷斯塔的控制逃跑了，
: 培養計劃因而中斷消失。御船被認為已死亡，發源檢體們失蹤。而實際
: 上，御船尚存活著。（進入了冷凍睡眠狀態）
: 生化駭客組織『L.S.S.』出現，首腦名為松尾龍介，目的是想利用心理穿孔
: （蜜蟻愛愉）讓御船千鶴子覺醒，得到學園都市的一切，獲得超越理事長
: 亞雷斯塔的權限。
: 星門計劃是研究古代原石能力者而培養人工能力者，『L.S.S. 』則是希望通
: 過覺醒近代能力者的發源檢體掌握獨家的培養技術。
: 目前『L.S.S.』已經拿到御船的冷凍身體了。接著為抓獲蜜蟻愛愉而派出了
: 飛龍、生化駭客、心理定規獄彩三姐妹。
: 以下為魔禁新約9，電子書158/584頁，小說裡關於御船千鶴子的原文
: 就在眾人鬧成一團時，明明鐘聲還沒響起，小萌老師卻進入了教室。
: 「今天我們要參考跟確立量子論法前就已實際存在的能力者—御船千鶴子

: 以下是三次元關於御船千鶴子的軼事傳說
: 資料來源：https://www.storm.mg/lifestyle/626415
: 標題：
: 日本首位透視超能力者，卻被科學家無情糟蹋、服毒自盡！
: 歷史真實的「千里眼女」悲慘人生
: 內文：
: 如果你小時候看過「七夜怪談」，從電視機爬出來的貞子想必應該是兒時
: 最大的夢靨。在劇情中，貞子就是繼承了母親的超能力，甚至是悲哀的命
: 運，才造成了一連串的慘案。
: 但你知道貞子的超能力母親確有其人嗎？她的原型角色是日本首位具有透
: 視能力的超能力者—御船千鶴子。她在明治時期，因為透視眼的超能力而
: 聲名大噪，甚至被稱為「千里眼之女」，當時日本科學界還一度對她展開
: 了一連串的測試。然而，究竟這名日本第一位透視能力者遭遇了什麼樣的
: 悲劇？才得以成為影史最重怨念厲鬼的母親原型呢？
: 診療身體殘疾，意外讓「超能力者」覺醒
: 御船千鶴子於明治19年7月17日，誕生於日本熊本的御船醫生家。對於御船
: 千鶴子孩提時期的描述極少，只知道她年幼時聽力受損，左耳幾近失聰，
: 但卻有著超乎常人的感官與更為豐富的情感，但身體上的殘疾，卻也使她
: 時常表現出悲觀的情緒。然而就在千鶴子17歲時，出現了一名改變千鶴子
: 一生的人—她的姊夫清源猛雄。
: 清源猛雄曾是一名中學的體操老師，據說他研究過當時日本很流行的催眠
: 術。在了解千鶴子的情況後，便對她進行催眠術訓練，希望能增強她的感
: 官。沒想到千鶴子在被催眠過後，不但感官能力大增，甚至在被催眠進入
: 昏睡狀態時，竟然能準確說出千里之外的景象！這樣的情況是清源當初始
: 料未及的，他立刻了解到這個女孩並非常人…
: 千鶴子真正展現出自己的千里眼能力是在1904年，當時正值日俄戰爭期
: 間，日本軍艦常陸丸號遭到俄軍擊沉。受村民的託付，清原再次對千鶴子
: 進行催眠，並問她原本應該搭乘常陸丸的陸軍第六師是否有搭上軍艦，千
: 鶴子表示第六師團不在軍艦上。三天後，師團的士兵向家鄉親人稍了信，
: 證實了千鶴子的說法。此事件一出，眾人譁然，千鶴子首次在眾人面前展
: 現超能力，就引起極大的關注。
: 在千里眼的能力覺醒後，千鶴子繼續接受清源的訓練，開發了更多未知的
: 能力，也發生了許多令人驚奇的事件。有次她曾表示庭院中的大樹內有蟲
: 寄生，而在家人劈開樹木後果真有數隻蟲寄居在內；另外，一名婦人在沙
: 灘上弄丟了戒指，出動了數十人找尋仍一無所獲，但千鶴子卻精確地指出
: 丟失的位置，果真找到了戒指。
: 平時，千鶴子也會在家中的醫院幫忙治療、看診。離奇的是，千鶴子竟然
: 能夠準確判斷病人的病因，對症下藥，彷彿能夠看透人體，因此來看診的
: 病人也越來越多。千鶴子的名聲逐漸在鄉里之間傳開，然而真正讓千鶴子
: 聲名大噪的，是日本三井集團請託尋找煤礦礦脈一事。
: 當時三井集團在大牟田山區展開調查，準備進行採礦作業，就找來千鶴
: 子、希望她能幫忙。一行人在山裡走了一圈後，千鶴子表示「南方一處有
: 巨大黑塊，不知為何物」，開挖後果真發現大量煤礦，三井集團立刻給予
: 兩萬元致謝（相當於現今的兩千萬日圓）
: 此時千鶴子的名聲到到達了鼎盛，人們越來越信任千鶴子、對她極為崇
: 拜，每日都有大批民眾前往尋求千鶴子幫助。但同時，日本民間也越來越
: 多人冒用「千里眼」的名聲進行宗教或醫療的詐騙與斂財，這也替千鶴子
: 的能力與名聲蒙上了一層陰影。
: 「千里眼」名號轟動日本，卻被遭科學家害慘
: 千鶴子的「千里眼事蹟」，也傳到了當時關西首屈一指的京都帝國大學前
: 總務長木下廣次的耳裡，她對於千鶴子的事蹟感到好奇，便對她的能力進
: 行驗證，實驗不但非常成功，還登上了報紙版面。這讓千鶴子的事蹟逐漸
: 在關西地區流傳了開來，當時幾乎所有人都知道這個「關西異能少女」的
: 存在！
: 而實驗成果，也驚動了在超能力研究領域極為出名的東京帝大助理教授福
: 來有吉，他提議再次對千鶴子展開了一連串的實驗。福來教授將19張用鋁
: 箔覆蓋的名片個別裝入封緘信封寄給千鶴子，要求她隔著信封觀察名片
: 後，寫下內容後寄回東京。
: 然而在這個實驗中，千鶴子最後僅寄回了7封，但其中除了少部分的錯字，
: 內容都大致正確。雖然這個實驗方式有諸多破綻，千鶴子寄回的數量也不
: 太合理，福來教授還是很滿意實驗結果，於是她便前往熊本與千鶴子會
: 面，希望進行幾場公開實驗。但萬萬沒想到，這正是千鶴子不幸的開端…
: 福來有吉帶著千鶴子在熊本、京都、東京進行了幾場公開實驗，這些實驗
: 中雖然有幾場非常成功，但有許多實驗的方式與細節都被批評不夠嚴謹，
: 過程也有許多漏洞與瑕疵，甚至有些人還認為福來教授與千鶴子根本有串
: 通作弊之嫌。
: 最後，真正引起大眾爭論以及令千鶴子灰心的是東京的公開實驗。在實驗
: 當中，福來教授將紙片放入鉛管，並將其封死，讓千鶴子坐在另一間房間
: 透視管內的紙片內容。在這次的實驗中，千鶴子幾乎無法透視以鉛管包覆
: 的物品。許多學者便批評，千鶴子根本是騙子，認為其只是想藉由名聲斂
: 財而已。
: 被世人誤解的超能力者，服毒終結短暫、悲劇的人生
: 其實千鶴子的能力受到質疑與誤解，早已不是一天兩天的事了。在千鶴子
: 22歲時，曾與一位日本陸軍中尉有過短暫的婚姻，丈夫新婚不久便前往前
: 線作戰，千鶴子便與公婆同住。有天，公公找不到自己錢包中的50元，非
: 常懊惱，沒想到千鶴子環顧家中一圈，便直接表示50元就在家中抽屜某
: 處，公婆一翻果真如此。然而，千鶴子的能力卻開始讓婆婆懷疑她偷了
: 錢，甚至跟軍中的兒子報告，最後，千鶴子被休掉了。從那時候起，情感
: 脆弱的千鶴子就開始懷疑，自己所擁有的能力到底是福還是禍？
: 而在東京實驗過後，輿論與謾罵排山倒海而來，支持者與科學的批評也越
: 顯激烈，千鶴子越是看清超能力為自己帶來的悲慘命運。雖然福來教授依
: 然相信千鶴子能力的真實性，然而，當時另一名被發現的異能者長尾郁
: 子，卻認為千鶴子的能力不過是騙人的低級伎倆，進而對她冠上了許多不
: 名譽的批評；另一方面，當時日本深受敬重的物理學家山川健次郎，也同
: 樣不相信千鶴子能力的真實性。千鶴子的心理的壓力漸漸地超出自己所能
: 負荷的。
: 明治44年，千鶴子服下了染料中的劇毒重鉻酸鉀， 終結了25歲的年輕生
: 命，而另一名異能者長尾郁子也在一個月多後，發高燒猝死，而福來有吉
: 博士則被革職。千鶴子究竟有沒有超能力，已無從查證，唯一知曉的是，
: 人們對於未知的抗拒與盲目的崇拜，甚至是不理性的批評，都成了奪去御
: 船千鶴子性命的原因。

2020/03/01 14:00
本月的「Dragalia Lost ～失落的龍絆～」
各位玩家您好

我是「Dragalia Lost ～失落的龍絆～」的執行總監岡田。

大家對料理對決的設施活動「食宴！饗趣美味廚藝巔峰戰」是否還滿意呢？
以下將為各位玩家帶來今後的「Dragalia Lost ～失落的龍絆～」相關情報。

■關於活動「食宴！饗趣美味廚藝巔峰戰」
本次活動中，玩家可獲得讓使用刀和魔杖的角色能力上升的設施。
此外，於召喚中登場的角色「瓦萊里奧」能使用3種模式，是個可透過切換模式使一般攻
擊和技能產生變化的特殊角色。
在單一模式下就已十分強大的瓦萊里奧，根據戰況切換模式能更加地發揮力量。
希望能讓玩家們體驗不同模式下有所變化的操作感。
瓦萊里奧也能以技能使敵人陷入「凍傷」狀態。

此外，瓦萊里奧發動技能2「請享用」時可賦予角色新效果「靈感」，提升至第5階段後發
動下一次的技能必會發生爆擊。

龍族「咕嚕曼」則擁有能使龍化時間延長的特殊技能。
由於龍化時間增長，在龍化期間可造成巨大傷害。

近期將會舉辦第2彈召喚。
精選角色為活動中也有登場的角色「三葉」。
與瓦萊里奧相同，三葉也可切換生魚片船模式以及天婦羅模式，使普通攻擊和技能產生變
化。
詳情請參考日後公開的公告內容。

此外，對於需要特殊操作的角色和龍族，今後將會調整於自動功能時的動作。

■關於瑪納之環第6層解放角色
已解放3名角色的瑪納之環第6層。
闇屬性的「克菈兒（法杖）」與「涅法莉耶」在瑪納之環第6層解放後，爆發攻擊將追加
效果，可解除敵人1個增益效果。
該效果可對「咢牙叛逆戰」的BOSS等等發揮效果。
水屬性的「塞因福蘭德」技能將追加使敵人陷入「凍傷」狀態的效果，與提升陷入「凍傷
」狀態的敵人傷害的效果，能夠對敵人造成更大的傷害。
3名都是於遊戲初期登場的角色，透過本次瑪納之環第6層解放將會大幅強化，請各位玩家
務必嘗試。

另外如同先前公告所告知，關於闇屬性★3角色的第6層解放，在調整上尚需花費些許時間
。
造成各位玩家的不便與困擾，在此致上最深的歉意。

■關於「咢牙叛逆戰」
高難度任務「咢牙叛逆戰」中，光屬性的新BOSS「凱嚴」已登場。
試著讓本次解放瑪納之環第6層的闇屬性角色裝備奇美拉武器或真‧龍族武器來進行挑戰
吧。


本次也可在中級的首次過關獎勵獲得可製作★6武器的素材。
「咢牙叛逆戰」的BOSS從高級開始會在戰鬥途中變身。
在此為各位玩家介紹變身後的模樣。

「咢牙叛逆戰」的難易度比真‧龍族的試煉稍微簡單一些，敵人動作等等也是以此進行設
計。
此外，實裝時也將持續追加各屬性的奇美拉和奇美拉武器。
奇美拉難易度雖高，但在我們的設計下，即使單人挑戰也可過關。使用打倒亡靈所獲得的
素材可製作對奇美拉有特殊效果的武器。

亡靈 → 奇美拉 → 咢牙 → 真‧龍族
玩家可依照上述順序進行挑戰，未能完成真‧龍族的試煉的玩家可以先嘗試挑戰咢牙。

■關於本月的活動
‧3月中旬預計舉辦復刻團體活動「鄉愁夜空盛綻之一輪煙花」 。
角色「蘇歐」預計將會解放瑪納之環第6層，尚未進行本活動的玩家，可以藉由本次機會
讓蘇歐加入夥伴的行列。
此外，同時也預計將會解放2名角色瑪納之環第6層。
其中1名角色為「霍克」。
詳情請參考日後公開的公告內容。


‧主線劇情第11章追加新劇情已於1月公開，第12章的追加新劇情預計於3月公開。
該章節將包含影響今後劇情的重要內容，敬請期待。

■關於今後的「Dragalia Lost ～失落的龍絆～」
‧遊戲將於本月迎接1.5週年。
相關紀念活動預計於3月底舉辦。
此外也預計公開「那姆頻道」，會為各位帶來3月底的版本更新情報，以及今後遊戲的相
關內容。
詳細公開日期將另行公告。
