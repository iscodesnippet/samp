: 先說, 鬼滅之刃已經是這時代難得的佳作,
: 作畫、音樂、劇情、效果各方面都屬中上,
: 但身為JUMP作品的特徵, 真的是一覽無遺。
: 不用主角的鼻子, JUMP的臭味我在一百米外都聞的出來。

其實jump公式還是有他吸引人的地方的

畢竟創作的世界也可以說是讓作者和讀者逃避現實的地方

正是因為現實中熱血和努力總是無法得到回報

所以幻想世界中熱血努力得到回報的故事才能夠激勵人心，讓人心生羨慕

: 1. 毅力論、吃苦論這種老掉牙的劇情梗, 不停地重複
: 雖然比起現代爛大街的穿越開掛作, 這種毅力吃苦的劇情
: 是較合理也更扣人心弦, 但我個人覺得很膩, 努力不夠就
: 再努力、吃苦不夠就更吃苦, 是嫌日子太好過嗎？
: 這種劇情真的、已經老梗到這年頭出現反而顯得新潮了。
: 比起吃苦的方式, 跳過這些吃苦的橋段是不是更好一點,
: 沒人想吃苦, 更沒人想在看動畫的時候看人吃苦, 哪有這麼閒？
: 更好一點的方式是, 比起吃苦, 為什麼不用腦袋克服？
: 當然, 這部作品也靠腦袋克服了很多關鍵, 不過大方向還是
: 要靠吃苦修練、吃苦擊敗敵人...真的是自找蛋疼

這個打點很奇怪，你既不滿意輕鬆開掛，也不滿意吃苦後拿掛

那你要怎麼樣讓主角獲得足夠的力量?

何況劇中吃苦是能獲得顯著報酬的

那不把吃苦這段寫出來，譬如只在開頭跟觀眾講三小強經過艱辛的訓練變得更強

那跟轉生重來系列寫著從嬰兒時期努力使用魔法來擴增魔力有什麼兩樣?

不就又變成你所謂的開掛作?

: 2. 沒有標準的、廉價的強度展現手法
: 鬼的蜘蛛絲很強, 所以刀砍下去反而是刀斷了, 但另一個
: 高手來砍, 就輕鬆斬過去, 就是因為他是高手、他很厲害...
: 標準在哪？道理何在？
: 那下次只要讓高手砍不過去某東西、再讓高高手出來砍
: 又可以顯現出另一層的強度, 這是多麼廉價的手法。
: 當然, 這樣演就整個糞掉了, 但這種手法的廉價性,
: 一次也就夠庸俗了。
: 也是濫用兼老梗到不行, 很典型的JUMP系手法。

但這方法就是很直接啊

不用在那邊量戰鬥力數字，多少JUMP類型作品在戰鬥力數字出來後就變難看了

例如《七大罪》就是一個很明顯的例子

雖說也有像《獵人》這種在氣量出來後用特殊能力和當下狀態來增添戰鬥可看性的手法

不過畢竟這作品本身就是一個很特殊的案例，也不是很JUMP系的作品

: 3. 假掰的中二手法
: 你看那個鬼好厲害, 眼睛都長著一堆奇怪的花紋斑點；
: 你看那個柱, 看上去就怪怪的, 講話一副屌炸天的樣子, 一定很強；
: 你看當主身邊那兩個女娃, 看上去就很奇特, 這個鬼殺隊組織一定很神祕很厲害；
: 這叫什麼, 這叫假鬼假怪。隨便設定一些人連走在馬路上都會被人白眼的設定,
: 用這樣的方式來顯現與眾不同, 大概也只有中二會認同了。
: 這種手法也是早就爛梗到不行, 我也不知道為什麼還能讓觀眾埋單。

吐槽這個跟吐槽二次元高中生每個都染髮一樣

實際上日常番一堆金髮就算了，還紅髮紫髮，也沒啥人吐槽啊

: 4. 典型的扁平式JUMP主角
: 主角, 講好聽叫做心靈純淨, 講難聽叫爛好人一枚,
: 只會拼命熱血、拼命認真, 然後心思單純、為人正直...
: 然後呢？沒有然後了。
: 上上一個大前輩叫孫悟空, 那時候大家覺得好棒棒,
: 這樣的角色很奇特；但上一個大前輩叫魯夫的,
: 玩到最後只讓人懷疑他是不是腦髓全部都換成橡皮筋了。

你舉的兩個JUMP大前輩的問題是很無腦

炭治郎至少算不上聖母無腦，被訓過後就乖乖殺鬼了

在都市時也不是馬上就去砍無慘

(雖說直接衝上去砍還會比較少人吐槽，畢竟無慘後面造成的傷亡大成那樣，

當然當下能不能打得過是另一個問題)

徹底奉行判定他們有沒有罪是上帝的工作，而我的工作是送他們去見上帝的工作態度

: 結論, 我看鬼滅時看到還以為是80年代的漫畫復刻了。
: 也許每個時代都需要一點這種東西,
: 不過我不知道為什麼能在全年齡層帶來風潮,
: 是因為禰豆子夠萌嗎？
: 好吧, 鬼滅之刃儘管存在這麼多JUMP的標配, 確實也沒這麼糟,

那就來理性討論一下鬼滅的優點吧

有雷請慎入


1.戰鬥能力設定簡單明瞭
雖然有人會認為這是缺點但我認為設定簡單更能讓人入坑，回歸最單純的武士斬鬼

鬼殺隊的設定就是一群武士利用呼吸法強化肉體來砍鬼罷了
這也讓訓練的篇章不會過於冗長以及避免後期的戰鬥力膨脹。
之前很多人都講鬼滅都沒有修練呼吸法的鏡頭，忽然用出X之呼吸很突兀
但動畫或漫畫當中砍草蓆其實就是在練劍術了=練X之呼吸

https://www.youtube.com/watch?v=ihWuLMlmKo8&t=317s
這是日本居合道的比賽，有發現跟動畫第三集沒兩樣對吧?

像水之呼吸第七型其實就是牙突-原地板、雷之呼吸第一型就居合斬


2.角色成長很明顯
鬼滅中成長最明顯的不是炭治郎而是我妻善逸，一開始只會哭到後面認清自己
該做什麼、不需要再依賴夢遊、明白想守護某樣事物就得付出努力。


3.黑頁不會冗長
鬼滅作者喜歡在黑頁的分鏡中就把一些配角的故事給解釋完，例如:師兄、村田。
另外黑頁的長度大多都2話搞定，作者也有在附錄提到不喜歡把黑頁畫太長。
但短短的黑頁也能讓觀眾對角色有很深刻的感受像上三、蛇柱、炎柱...等

4.和風感強烈
現今的熱賣漫畫中，鬼滅算是最會結合日本文化了(也可能是我看太少)
蛇柱的故事其實就是八丈島的大蛇傳說、前面提到的居合道(可以注意義勇收刀的動作)
另外利用花語、和服上的圖紋來讓角色更立體

例:https://i.imgur.com/xIUD19n.jpg
戀柱和服上的圖案是梅花，象徵:忍耐的心

https://i.imgur.com/XJV7AaF.jpg
戀雪和服則是雪花，象徵:純潔的心，腳下的花是牽牛花 花語:短暫的愛

https://i.imgur.com/QWE4Zm5.jpg
蛇柱幻想與戀相親，當中紫色的花為杜若 花語:幸福終將到來
在《萬葉集》中，杜若的和歌內容為等待戀人的故事，因此也常被當作等待戀愛或戀人的
象徵。

其他還有很多像是16卷封面天音大人的和服上是蝴蝶 象徵:長壽，仔細看就會發現鬼滅
角色的和服圖紋都不一樣這些都代表一些含義，封面的花也是一樣
這也是為什麼鬼滅能受到年長女性歡迎或日本人喜愛。

打了那麼多其實是希望大家能注意到鬼滅漫畫的好而不是一昧講靠UFO...
UFO確實把鬼滅的打鬥畫面做得很好但原作也有它細膩的地方。況且UFO的劇本就..= =
總而言之希望大家也能來一起研究和服圖案及花語XD，不然總是看對岸或自己餵狗好累
