嗯...

這邊要看哪款MMO啦，不同款不能一起比


下面我以WOW為例子


: 1.嫌補太慢或是補太少
: 通常發生在初中期，通常是單補/群補/驅散/上Buff
: 打到最後腦筋整個打結會被嫌棄

這種要看是真的補不夠力還是輸出太低導致戰鬥時間太長讓補沒魔

前者就摸摸鼻子，後者就嘴回去了


: 2.做最苦的工拿最少的酬勞
: 有些時候掉一些裝備或是好東西時，為了公平起見採平分制
: 你拿你職業的其他人勿動，表面上很公平實際上完全不是那麼一回事
: 打手農野怪平時錢已經夠多了，進副本還要講求公平原則是不是搞錯了什麼？
: 坦補農一小時的效率高
: 還是打手農的效率高
: 我想不用分析大家都知道
: 說難聽點就是沒坦補連團都開不成，要求平分分裝真的公平嗎？


WOW在1-7版時，固定團隊會用DKP(屠龍點數/團隊積分)或是遊戲金幣之類的方式
讓團長可以決定裝備分配問題

到了8版就沒有這問題了，因為都看運氣。
打完王每個人撿到的東西都不一樣，能不能撿到裝全看運氣


: 3.補師練功效率遠不如其他職業
: 想必有人也有遇過，可能初期大家都很仰賴補師的buff
: 都會帶出去農
: 結果久而久之變成人手一台小號補師
: 拍謝我決定自己壟斷農怪市場，補師你就好好加油吧
: 導致補師要改配點，什麼爆牧敏牧將自己強迫變成一個半打手，效率還沒打手高

WOW給的解法是，讓你可以在補/輸出間隨時切換，大多數的裝備也可以通用

: 4.躺在地上喊復活不自己想辦法
: 我自己是有玩過幾款在副本死亡可以重跑回現場的
: 可是總有些人不肯我就是喜歡躺著看自己隊伍團滅
: 之後再嘲諷說早叫你復活我了吧，搞到大家都滅團
: 哈囉
: 別人的MP都不用錢還是技能都不用CD
: 隊伍少一個人已經忙得要死還要詠唱復活你，還不被幹飛天


WOW中，"戰鬥中復活"會有次數限制，所以這種技能多半是由團長指定施放

也就不會有後面輸出嘴人的場面



: 5.嫌走太慢，拖延副本推進進度
: 媽媽咧，你們自己技能加跑速或是你穿跑速裝
: 打個王或小兵都要閃的老遠的補師，你要跟上你們速度是不是搞錯什麼

副本推進速度應該是看小怪/王的戰鬥時間，這種時候應該回應他們打太久了

: 6.要求坦補公開裝備
: 遊戲後期組團要求坦補裝備夠好來應付副本
: 結果自己鎖裝備不給人看
: 對別人說哎唷你裝備才這樣下一位
: 還真是抱歉啊
: 出個團賺不到幾毛錢還要為了你們買優良裝備


WOW的角色裝備隨時可看，而且拿翹的多半不會是輸出....


: 8.遊戲進化成只需要打手的環境
: 如標題，平衡嚴重失衡變成出個團已經不用正統的坦補

: 坦補紛紛失業
