死亡筆記本規則 https://reurl.cc/M79WWk

死亡筆記本的規則太龜毛
光是「操縱死亡時間須於23天以內」這點就讓人賺不了大錢

有些人說親戚的遺產
光是遺產的獲得也是很麻煩的
死亡的順序不能錯，否則拿到遺產有可能是別人
(民法繼承篇)

當然了，你也可以寫說
父方或母方那邊的親戚同時保了大險
然後約定同時旅行，結果同時意外死亡
但這樣的寫法又可能因為筆記本的規則而無效

大多數的人應該記不得所有親戚的名字及長相
加上讓那麼多人同時死亡的機率不大且容易影響到第三者
結果就變成一堆親戚突然心臟麻痹死亡 = =

「寫入的死亡狀況限定於合乎物理學以及那個人做了也不奇怪的範圍之內，
否則無法實現。」

: 文長
: 一、扶羅提亞藝術學院
: 要介紹這間學校以前先認識他的名字，芙蘿提亞也就是Frontier，就是開拓動漫祭那個
: Frontier，所以也被叫開拓學院。另外也能譯作邊境，所以對岸的翻譯常會叫它邊境學院
: 又因為前期劇情沒什麼爆點，腳色強度堪憂，沒什麼人在意這間學校，所以就被稱為邊
: 緣院了，這就是邊緣院這個名稱的由來。為了方便後面稱呼一律統一邊緣院。
: 邊緣院是一間新興學院，以開拓娛樂領域內學生們的才能為宗旨。學校本身非常有錢
: 會送新生平板，舉辦烤肉會等等活動。但是學校本身評價並不是很好，被批評學生隨便
: 、玩世不恭，表現不怎麼樣，只會靠著新技術來炒話題和吸引學生等等。這點也能在邊
: 緣院的故事看到，當蝴蝶等人被選上參加FES的時候，得到「選拔的謎團」這樣的討論。
: 另外邊緣院內有很多科，像劇情中有看到陶藝科、繪畫(?)科，還有舞台表現科等等。
: 補充一點，邊緣院的所有人都是雙手武器喔!

我通常和人介紹芙羅提亞，特別是有接觸BGD的人介紹時，
都會說芙羅提亞是少歌的HHW。
同樣是以歡樂為主基調的一團。
但是比起有弦卷家財力這個超強外掛的HHW，芙羅提亞則是更貼近普通人。

不同於像是不知道不開心為何物的心，艾露露的開心是基於能學習到自己喜歡的事物，
對他而言新的挑戰新的困難，也意味著有新的東西能夠學習。

相較起聖翔、席格菲特和凜明館的悠久歷史，作為新興學校的芙羅提亞呈現出來的是一種
毫無包袱的自由奔放。他們比起其他學校更樂意去嘗試新的舞台表演方式，甚至是各種
高科技設備(就說他們學校真的很有錢)，這種一直以來都被芙羅提亞式的噱頭，
不過新的道路也是這樣被摸索出來的。


這大概也是校名會是「開拓」的關係，比起延續傳統，毫無歷史包袱的芙羅提亞則是
一直往新的邊境衝刺碰撞。

就如同這五人組的代表戲劇「雙子海盜」一樣。揚帆！朝著新的天地冒險前進





: 二、蝴蝶靜羽─邊緣院的全能首席
: 蝴蝶靜羽，邊緣院的首席。有著被克洛和蕉蕉稱讚的實力(主線)，能夠被這兩位聖翔前
: 幾名稱讚實力絕對不可小覷。在校內，蝴蝶一年級就在SCON中，獲得了幾乎所有的獎項
: ，甚至贏過即將離開學校的學長姐。但這也導致出現「只要和蝴蝶一組就能保送優勝」的
: 說法，也因此讓蝴蝶不願參加二年級的SCON，但最後還是被阿露露給拉上台了。
: 從各劇情中可以得知蝴蝶是非常全能的。在復活節蝴蝶羈絆劇情中，蝴蝶製作的草莓餡餅
: 被說堪比外面的店家，在這之後還幫助陶藝科和委員會的人修改企畫書，而且舞台科的會
: 議，和委員會的會議也是蝴蝶在主持，從這樣一個羈絆劇情就能看到蝴蝶的全能。在
