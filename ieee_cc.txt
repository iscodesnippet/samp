獨佔遊戲而受到玩家的吐槽。但是從長遠來看，Xbox總監菲爾‧斯賓塞認為這並不是個問
題。微軟現在擁有15個Xbox第一方遊戲工作室，這些團隊都在專門為Xbox和PC開發新的遊
戲內容。

據外媒Gamespot報導，Xbox總監菲爾‧斯賓塞在最近的GameLab活動中表示：「本家遊
戲的內容對我們的未來非常重要。這對我們的硬體平台也很重要，對XGP（Xbox Game
Pass）和我們在微軟的整體遊戲策略也很重要。幾年前，我們組建了新的領導團隊，開始
制定我們的戰略，並向公司內部闡明我們的戰略看到董事會和公司高層領導團隊給予我們
支援，讓我們的本家遊戲陣營發展壯大，真是令人高興。」

斯賓塞說，他對公司內部眾多的遊戲工作室和他們正在進行的專案感到「非常滿意」。他
暗示說，還有許多工作室的新作還沒有公佈呢。

斯賓塞還表示，他不想模仿競爭對手的本家遊戲開發方法。比如，有些人就談到了索尼
的獨佔遊戲IP《秘境探險》和《戰神》，問微軟能夠跟這兩大IP的遊戲競爭在哪裡。斯賓
塞說，微軟的想法與眾不同。

斯賓塞在談到開發微軟第一方獨佔遊戲時說：「我們知道這（本家獨佔遊戲）很重要—
—我們也知道這是粉絲們想要的東西，但我也要說，我們將圍繞我們認為需要對Xbox平台
做的事情進行一些支持，比如說打造Xbox的本家遊戲。有時候我們會聽到這樣的問題：
『你的這個獨佔大作在哪裡，或是你的那個獨佔大作遊戲怎麼還沒出。』但我不認為我們
的目標是複製其他人的做法。我們對自己的平台、服務或內容做和其他平台完全相同的事
情，對整個行業都沒有幫助。

儘管微軟不會直接模仿索尼或任天堂的本家遊戲戰略，但斯賓塞表示，他相信自己的團
隊正在開發能夠吸引廣大玩家的本家遊戲。

斯賓塞說：「我為我們Xbox遊戲工作室團隊創造的多樣化的內容感到驕傲，比如新遊戲的
類型、藝術風格、平台和遊戲性。單人遊戲、多人遊戲、合作遊戲都有。我認為這種多樣
性還會繼續下去。這將是我們的中流砥柱。我們的團隊領導力來自於他們自己的願景和他
們想要打造的東西，而不會是某種特定的遊戲類型或者特定的故事。」

不過，總體而言，斯賓塞承認，微軟過去在本家遊戲開發方面做得不夠。他說： 「這
是我們需要做更多努力的領域，現在我們已經做到了。」

1. 這篇的代理商或者持有版權的公司，和會播放動畫的平台有啥關係？

簡易的示意圖，並非所有的商業模式都能描述，當作一個參考就好

上游中游下游

╭────╮   授權    ╭────╮  再授權   ╭────╮
│
│────→ ││────→ ││
│日方窗口│
│ 代理商 ││影音平台│
│
│←──── ││←──── ││
╰────╯  保證金   ╰────╯  授權費   ╰────╯
版權稅版權稅


本篇當中＜＞內的公司都是位在中游的位置，一般都簡稱為代理商

也就是從動畫的製作委員會的海外銷售窗口取得動畫的授權

而大家所熟知的動畫瘋、愛奇藝台灣站等等，都是位在下游的影音平台

台灣大部分的平台都不是直接從日方那邊購買放映權的

基本上都是從代理商那邊取得授權，然後在自家的平台上架

至於權利和金流的細節則要端看合約怎麼簽，每部作品的狀況都有所不同


2. 怎麼沒有OOXX這部作品？

有可能沒人代理，又或者代理了還沒宣

對於特定作品有問題的可以在這篇詢問


3. 要怎麼知道哪些作品會在哪個平台上架？

針對台灣的部分的話：

CASE 1：木棉花＆曼迪＆提恩傳媒

基本上木棉花都是台灣「全網上架」的，不過有時候一些作品會有平台搶首播權

使得其他網站必須要暫緩播出（要晚一周或一季度才能上架）

曼迪和木棉花還滿類似的，只是上架的平台比木棉花少一些

提恩傳媒剛發展沒多久，目前合作的平台還不是很多，但有持續在增加

整體授權的模式也都是「全網上架」


CASE 2：羚邦

羚邦針對台灣的授權模式一直在改變

以近期羚邦針對台灣市場的狀況來看的話

這次夏番應該大部分的平台都有很高的機會上架

尤其以動畫瘋、KKTV、friDay影音、中華電信(Hami、MOD)等平台為主

近期也開始做YouTube頻道，部分作品也會於他們的Ani-One頻道上架


CASE 3：Netflix＆亞馬遜

基本上被這兩大巨頭拿下版權的作品，台灣的觀賞管道就只剩下這邊了

不要奢望其他平台有任何上架的可能性


CASE 4：bilibili／童園創意

部分未被b站港澳台獨家的作品，有機會會在中華電信體系上架

但也基本上都是過季好久之後才會授權，當季新番一樣不要奢望其他平台會上架

部分作品也有機會授權電視的放映權給ANIMAX，例如炎炎消防隊


當然上一季也稍微有了些轉機，因為有《格萊普尼爾》這個特例

不過也就投資的作品有機會分售出去給其他代理商，直接買放送權的估計還是無解...

但哪一個人對山王來說更為重要一些呢?

-----------------------------------------------------------------------
1.澤北缺席- 先發調整:C野邊/F大河田/F松本/G一之倉/G深津

失去了澤北這管超強火力雖較難彌補 不過可改由大河田擔綱攻擊主力

失去澤北的影響可能是較輕的

輪流單打的局面 對山王這支王者球隊來說 不確定的因素就變多了

----------------------------------------------------------------------

以上 個人認為三巨頭缺陣的影響 深津>河田>澤北

但無論是缺誰 山王都仍是強隊

※ 引述《dro001 (風雲)》之銘言：
: 最近重看山王篇 真的蠻喜歡山王這支球隊的
: 實力非常強 又不輕敵 除了三分火力稍弱些外 幾乎是完美無缺的球隊
: 尤其是三巨頭 冷靜的後場指揮官-深津 全能的禁區怪物-河田 無敵的單打王-澤北
: 三人都是攻守兼備 相信任一人都有機會能帶起一個球隊 何況山王有三個
: 假設三巨頭有一人受傷缺席 我相信山王仍是具有問鼎冠軍實力的強隊
: 但哪一個人對山王來說更為重要一些呢?
: 個人的看法 由影響最輕到最重如下：
: -----------------------------------------------------------------------
: 1.澤北缺席- 先發調整:C野邊/F大河田/F松本/G一之倉/G深津
: or C小河田/F野邊/F大河田/G松本/G深津
: 儘管書中很少有深津與大河田的檔拆戰術 不過相信這是有機會執行的 而且威力應該不錯
: 失去了澤北這管超強火力雖較難彌補 不過可改由大河田擔綱攻擊主力
: 松本雖不及澤北 但補上側翼火力支援大河田應是足夠 深津若有適時的外線支援
: 山王的攻擊面仍是相當強悍 至於防守面 則看對手而定 對方禁區強就擺上三塔陣
: 大河田的運球及速度打小前鋒應該也可 雖然少了一把王牌大鎖 但整體來說
: 失去澤北的影響可能是較輕的
: -----------------------------------------------------------------------
: 2.河田缺席- 先發調整：C小河田/F野邊/F澤北/G松本/G深津
: 少了大河田這禁區攻守一把罩的大將 只能用小河田+野邊雙塔來彌補
: 畢竟兩人防守攻力再不好 兩個超過197的長人放上場 威嚇力還是在的
: 何況野邊也說過籃板在櫻木之前他也只搶輸過大河田 書中野邊也很少有被直接吃掉的
: 而攻擊方面 會由深津和澤北發動的快攻為主 澤北少了河田後 在進攻方會更賣力
: 而其實小河田的禁區低位要不是遇到櫻木 也沒那麼不堪 擔綱副戰術執行也夠了
: 只是遇到像名朋這種有禁區怪物坐鎮的 可能會辛苦點
: -----------------------------------------------------------------------
: 3.深津缺席- 先發調整：C野邊/F大河田/F澤北/G松本/G一之倉
: 深津缺席可能是小弟我認為影響最大的 儘管無論是哪隊
: 有了河田和澤北兩大怪物 都絕對具備問鼎冠軍的實力 但山王在描述上
: 大部分的進攻戰術都是由深津發動 全隊沒有第二個人有組織戰術的能力
: 可能得由松本或一之倉來代替 甚至是澤北自己來 松本和一之倉的能力也許夠
: 但心理素質沒像深津一樣沉穩 能鎮壓全場 可能山王整體會流於澤北和河田兩人
: 輪流單打的局面 對山王這支王者球隊來說 不確定的因素就變多了
: ----------------------------------------------------------------------
: 以上 個人認為三巨頭缺陣的影響 深津>河田>澤北
: 但無論是缺誰 山王都仍是強隊
