: 我只玩過DS的chrono trigger，音樂真是爆好聽沒話說，且跟遊戲畫面非常的契合，劇情
: 也是我玩過的遊戲中數一數二，神作無誤。續作chrono cross雖然沒有接觸，但水管上很
: 多影片會把兩者BGM串在一起所以我就一起聽了，我覺得兩個作品音樂風格非常不同耶，C
: C的音樂似乎多了點民族風的感覺，也是超級優秀，光田康典先生真的是神阿。

我覺得主要其實是因為 CT 是在超任上，音效音樂能使用的樂器種類沒那麼多，不過像是
Guardia Millennial Fair 其實也是滿多民族風，Corridors of Time 也是。

相較之下 CC 是直接錄製的，所以選擇上更為多樣，也就能夠有光田康典愛用的傳統
凱爾特民俗樂器。

: 印象中雖然超時空之鑰是電玩交響樂的必帶曲目，但一直沒有專門為這系列舉辦的音樂會
: ，因此得知這個消息時非常興奮，請人幫忙抽票幸運抽到クロノシート，這是最貴的票種
: 要13800日幣，有送一組特典音樂盒（是說怎麼lawson的網站也要手機認證了…只能找代
: 抽），價錢有點心痛但因為愛我還是給他抽下去了，去完的感想覺得這錢花的真值得XD

我個人也是覺得滿值得的，不過在現場聽東京場，其實可以說是跟 CD 幾乎沒差別XD
差別大概就是整個交響樂隊在你面前演奏，那個震撼力真的不是聽 CD 可以比的，另外
加上遊戲畫面輔助，真的是很感動。要說遺憾的話，就是交響樂編曲少掉了民俗樂器的
部分。

: クロノ・トリガー》，切入的時間非常剛好，好像真的在玩遊戲一樣。經典曲目風之憧憬
: 、時之迴廊等都有出現，這次大地圖的音樂都改編得比較安靜沈穩，但時之迴廊後半部越
: 來越有氣勢，非常棒。
: 其中我最喜歡的是青蛙主題曲，以前就覺得這首氣勢磅礡的音樂改成交響樂一定很讚，去

めない  宝
石～》，螢幕畫面則是放上演出人員名單，也代表整場演奏會即

: 樂的心情XD男主角跟山貓一直換來換去的，他們是同一個人嗎？山貓感覺很像壞蛋但是有
: 幾幕發現他在戰鬥成員裡？另外我還看到很像CT三人組的小孩，可是衣服又不太一樣。還
: 有一堆龍，不知道他們出現的理由是什麼，不太像boss，可是看起來也很壞。然後男主在
: 沙灘被藍色光影包圍還有女主被刺的畫面重播了好幾次，整體看下來超錯亂的，很難理出
: 來故事的脈絡，是不是應該去補個遊戲。不過男主越看越帥誒，變身裸體那段好香
: talk的部份不多，大概就是講當時製作這兩部音樂時的事情（抱歉日文能力有限無法完整
: 翻譯），有聽到一件驚訝的事情是超時空之鑰是光田康典20幾歲時的作品，有夠猛。

這好像是他在 Square 工作時第一次挑大樑，wiki上寫的故事是他原本是音源編成師，他對
自己的薪資，所以跟老闆說不讓他作曲的話他就要閃了。後來就  坂口博信建議他去作 CT 。

: 總之這是一場很棒的音樂會，非常慶幸有買票來聽，10月在東京有兩場，有機會可以去聽
: 喔，票好像還有。
: 另外11月光田康典先生要來台灣辦音樂會喔，不過是CC的沒有CT，感覺官方沒有什麼宣傳
: ，票還剩很多的感覺，台灣好像還是CT比較熱門，而且說實在票蠻貴的XD不過還是希望大
: 家能來聽，絕對不會失望的。

聽完大阪場 (東京場完全買不到 Orz) 我個人覺得非常值得去聽。由於這是光田康典自己辦

在看了大大們的回覆後，我覺得應該就誠如onnax大所說的。
現階段法律確實不處罰自殺（或者反過來說，自殺不違法），不過其實也就僅止於此而已。
而「自殺法」則是想透過「法律的明文」來承認人民有自殺的權利。
說的極端一點，我將其理解為把「自殺權」視為基本人權之一來處理。

現行法下，自殺確實不違反任何法律。
畢竟，殺人罪保護的是「他人的生命法益」，但是自殺卻是侵害自己的生命。
退一步來說，自殺既遂，死了就死了，但如果是未遂沒死成，那麼去處罰一位想自殺的人
，判他坐牢，其實也完全沒什麼實益可言吧？

不過，也如a204a218大所言：「法律上自由和生命之類的基本人權個人是無權放棄的吧」
自殺確實是「不處罰」，不過在一般情況下的阻卻違法事由中，所謂「得被害人承諾」，
是無法處理重大身體法益及生命法益的。

這也是為什麼，自己自殺不成立任何犯罪，
但是如劇中提到的「教唆自殺」（我國則是刑法275條之加工自殺罪）時，去教唆、幫助
一個人自殺卻是犯罪的原因。
換言之，縱使基於尊重自殺當事人自主意願的角度，因自殺者誠摯而完整的意願來減輕教唆
或幫助者部分的違法性，但仍不得完全阻卻刑法的成立。

追本溯源地說，
國內就有學者（謝煜偉）指出，會有加工自殺罪，就是因為他牴觸了國家不欲助長自殺的政
策方向。
易言之，刑事政策上把將「自殺」視為一種禁忌。


而結合到這部動畫，「自殺法」就是要破除這種「禁忌」，將自殺視為一種「人權」。
從第六集對話中「不再將自殺視為一種『異常』」應該可以佐證這樣的看法。

這樣也能解釋，為什麼明明自殺是不成立犯罪的，而劇中角色也都明白這一點，卻又要另行
推出一部自殺「法」來規範自殺。
同時，從這個角度再來看第六集辯論中，前幾位學者以「經濟」、「道德」等角度來否定「
自殺」，確實是契合主旨的。

這幾位就是站在將自殺視為禁忌的角度，去論述為何要有加工自殺罪的存在、為何要將自殺
評價成「不對」的事情。
更進一步的話，則可以討論的就是安樂死的爭議。


不過很可惜的，上述劇中並沒有「明確」指示說「自殺法」設立之目的或其內容。

或者說，整個來龍去脈其實一時間我沒有搞懂，所以使資質駑鈍如我，陷入了「  诶自殺明明
就不犯法，為什麼要不斷討論能不能自殺？」的疑問漩渦中。


一言以蔽之，本劇目前為止，看起來就是「要讓人民能完全掌握自己的生命」。
而如果能貫徹這點（自殺法），自己燒炭、跳樓的這種自殺法與現在不會有什麼太大的差異
。
但是加工自殺（如教唆、幫助自殺等）及所有種類的安樂死（如劇中提到的積極安樂死及消
