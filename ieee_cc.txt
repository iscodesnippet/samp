───────────────────────────────────────

各位鄉民安安

雖然艦B與艦C的戰鬥方式完全不同，
但是作為船艦擬人化的遊戲，還是有相同的機制可以比較。

(發文者目前狀態:艦C於改HTML前退役、碧藍剛入坑2個禮拜)

因為已經從艦C退役，所以如有與目前遊戲不符的地方，
還請現役的提督在推文提醒，我會盡快修正。

=========================================================

1.建造

艦B:

分為3種池(驅逐&輕巡池、重巡&戰艦池、空母&特殊艦池)

每次建造只需要投入心智魔方(藍方塊)與金錢即可。

優點:簡單、快速，不用想太多，點下去就對了。

缺點:抽選是從池內的全體抽選，
所以隨著池子可建造的艦娘增加，想要出特定艦娘的機率就會越少，
常常會被其他船GANK。(對，就是說妳，聖地牙哥)

_________________________________________________________________

艦C:

建造需要投入特定數值的4種原料(鋼、油、彈、鋁)外加開發資材，來建造想要的艦娘。

缺點:複雜且需看網路上建議的數值表來建造。

優點:雖然一樣要從建造池裡抽選，
不過可以調整特定艦娘的出貨率，使其機率變高。

===========================================================================

2.戰鬥

艦B:

沒有轟沉機制(不用擔心因恍神而讓艦娘沉沒的悲劇)

出擊只需消耗石油，戰鬥結束後不用補充彈藥與燃料

沒有入渠機制，回港後自動回復為100%生命。

有心情值與親密度機制，心情可隨時間回復。
_____________________________________________________________

艦C:

有轟沉機制，因恍神導致與心愛艦娘永別的提督大有人在。

出擊完需手動補充出擊艦隊的燃料與彈藥(常常發生沒有補充就派出去的慘事)

有入渠機制，艦娘受傷後不會自動恢復，需要入渠回復。
(入渠時間根據損傷程度及艦種決定)
(可以妄想是艦娘在洗澡)

有疲勞值機制，疲勞可隨時間回復。
配合上述的入渠機制，可以做到疲勞後直接入渠+丟水桶來瞬間回復心情值。

========================================================================

3.立繪與語音

艦B:

可以自由切換改造前、改造後、特殊立繪。

額外的立繪需要用鑽石購買、部分由活動與結婚贈送。

部分艦娘尚未實裝語音(實裝預定中)

語音基本上綁定立繪，想要聽老婆的其他語音請購買立繪。

因為遊戲機制的關係，所以沒有中破立繪。

秘書艦可以安排多位且不須指定旗艦位置。
_____________________________________________________________


艦C

無法切換改造前後的立繪，改造就回不去了
---------------------------------------------------------

因為這個原因，當年我特地造了第二台俾斯麥維持原本的立繪，
時至今日還是覺得不能切換自己喜歡的立繪這點，實在是很愚蠢。

還在任的時候，上K島的艦娘版抱怨過這點。
島民的回覆是"你懂什麼，這樣才有歷史感!"

...可是我不想要歷史感，只是想要老婆穿我喜歡的衣服(′・ω・‵)

---------------------------------------------------------

額外立繪與語音是免費的，不過期間限定，
像是泳裝立繪到夏季的時候，
(如果她有泳裝立繪的話)人物會(強制)替換掉原本的立繪變為泳裝立繪。

有中破立繪
(讓祕書艦維持色色的中破立繪不給修理的鬼畜提督大有人在)

固定第一艦隊的旗艦為秘書艦。

===================================================================

4.遠征

艦B

在艦B稱為委託，
不用占出擊艦隊的格數，
但如果有想派出遠征的艦娘，每次都需要手動選取。
委託出現的項目無法自己指定，所以無法自行安排時間。
__________________________________________________________

艦C

占出擊艦隊格數，
因為是固定的，所以不需要每次都手動選取艦娘。
遠征可以自行指定想要的項目，便於安排時間。

===================================================================

艦B

基本上為等級壓制，等級練高就能過。
(因剛入坑，故待驗證)

: 各位鄉民安安
: 雖然艦B與艦C的戰鬥方式完全不同，
: 但是作為船艦擬人化的遊戲，還是有相同的機制可以比較。
: (發文者目前狀態:艦C於改HTML前退役、碧藍剛入坑2個禮拜)
: 因為已經從艦C退役，所以如有與目前遊戲不符的地方，
: 還請現役的提督在推文提醒，我會盡快修正。
: =========================================================
: 1.建造
: 艦B:
: 分為3種池(驅逐&輕巡池、重巡&戰艦池、空母&特殊艦池)
: 每次建造只需要投入心智魔方(藍方塊)與金錢即可。
: 優點:簡單、快速，不用想太多，點下去就對了。
: 缺點:抽選是從池內的全體抽選，
: 所以隨著池子可建造的艦娘增加，想要出特定艦娘的機率就會越少，
: 常常會被其他船GANK。(對，就是說妳，聖地牙哥)
: _________________________________________________________________
: 艦C:
: 建造需要投入特定數值的4種原料(鋼、油、彈、鋁)外加開發資材，來建造想要的艦娘。
: 缺點:複雜且需看網路上建議的數值表來建造。
: 優點:雖然一樣要從建造池裡抽選，
: 不過可以調整特定艦娘的出貨率，使其機率變高。
: ===========================================================================
: 2.戰鬥
: 艦B:
: 沒有轟沉機制(不用擔心因恍神而讓艦娘沉沒的悲劇)
: 出擊只需消耗石油，戰鬥結束後不用補充彈藥與燃料
: 沒有入渠機制，回港後自動回復為100%生命。
: 有心情值與親密度機制，心情可隨時間回復。
: _____________________________________________________________
: 艦C:
: 有轟沉機制，因恍神導致與心愛艦娘永別的提督大有人在。
: 出擊完需手動補充出擊艦隊的燃料與彈藥(常常發生沒有補充就派出去的慘事)
: 有入渠機制，艦娘受傷後不會自動恢復，需要入渠回復。
: (入渠時間根據損傷程度及艦種決定)
: (可以妄想是艦娘在洗澡)
: 有疲勞值機制，疲勞可隨時間回復。
: 配合上述的入渠機制，可以做到疲勞後直接入渠+丟水桶來瞬間回復心情值。
: ========================================================================
: 3.立繪與語音
: 艦B:
: 可以自由切換改造前、改造後、特殊立繪。
: 額外的立繪需要用鑽石購買、部分由活動與結婚贈送。
: 部分艦娘尚未實裝語音(實裝預定中)
: 語音基本上綁定立繪，想要聽老婆的其他語音請購買立繪。
: 因為遊戲機制的關係，所以沒有中破立繪。
: 秘書艦可以安排多位且不須指定旗艦位置。
: _____________________________________________________________
: 艦C
: 無法切換改造前後的立繪，改造就回不去了
: ---------------------------------------------------------
: 因為這個原因，當年我特地造了第二台俾斯麥維持原本的立繪，
: 時至今日還是覺得不能切換自己喜歡的立繪這點，實在是很愚蠢。
: 還在任的時候，上K島的艦娘版抱怨過這點。
: 島民的回覆是"你懂什麼，這樣才有歷史感!"
: ...可是我不想要歷史感，只是想要老婆穿我喜歡的衣服(′・ω・‵)
: ---------------------------------------------------------
: 額外立繪與語音是免費的，不過期間限定，
: 像是泳裝立繪到夏季的時候，
: (如果她有泳裝立繪的話)人物會(強制)替換掉原本的立繪變為泳裝立繪。
: 有中破立繪
: (讓祕書艦維持色色的中破立繪不給修理的鬼畜提督大有人在)
: 固定第一艦隊的旗艦為秘書艦。
: ===================================================================
: 4.遠征
: 艦B
: 在艦B稱為委託，
: 不用占出擊艦隊的格數，

: 委託出現的項目無法自己指定，所以無法自行安排時間。
: __________________________________________________________
: 艦C
: 占出擊艦隊格數，
: 因為是固定的，所以不需要每次都手動選取艦娘。
: 遠征可以自行指定想要的項目，便於安排時間。
: ===================================================================
: 5.活動難度
: 艦B
: 基本上為等級壓制，等級練高就能過。
: (因剛入坑，故待驗證)
: ___________________________________________________________
: 艦C
: 出了名的田中難度，
: 在打之前必須先讓前線攻略組(其他玩家)探路，
: 等攻略出來在上，不然只是白白浪費資源。
: 甲級難度喪心病狂，連當年最簡單的丙級也一點都不簡單。
