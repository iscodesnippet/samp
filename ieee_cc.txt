 時間  Wed Apr 22 22:47:13 2020                                               
───────────────────────────────────────

首先不管<咒術迴戰>的作者芥芥到底能不能解釋清楚五條悟的能力，
五條悟本人都已經說他自己的能力是「阿基里斯和烏龜」了，
也就是「芝諾悖論」這個哲學家還有一個著名悖論「飛矢不動」。

當然我們都知道阿基里斯不可能追不上烏龜，就算阿基里斯讓了999m給烏龜也一樣，
實際上阿基里斯只要跑個101秒就會追上烏龜了，只是追趕的距離在悖論中被限制了。

所以敵人的能力永遠都打不到五條悟，如同阿基里斯追不上烏龜。



叭叭！這就是悖論。



我還記得一個在巴哈連載的漫畫，但我找不到了，不知道有沒有洽民知道？
好像是一個惡魔女孩跟其他惡魔一起應徵女僕，
女主角有著跟米菲兔一樣的嘴巴，能力是「特休斯之船的惡魔」，

「如果不停補修一艘船直到所有舊的零件都被替換了，這艘船還是原本的船嗎？」

女主角可以藉由裝替物品修補自己損壞的四肢，被女僕長稱為新世代的惡魔。
（不知道記憶有沒有錯誤）



除了悖論，還有一些不可能的理論被用在動漫中，只是我一時想不到誰有這種能力。

比如說著名的「拉普拉斯的惡魔」容易被表現為全知全能、未來預測。
如果有一個智者能知道宇宙中每個原子的狀態，
那麼這位智者也能同時知道宇宙的所有過去與未來。

（當然除了非是上帝不然不可能有如此強大的計算能力）
（僅僅只是一小顆電子和偉大的量子力學也讓拉普拉斯惡魔等於不存在）

還有一個著名的「馬克斯威爾惡魔」被表現為無限的能量、永動機、最強之力。
如果有個惡魔能控制、選擇把高能量和低能量的分子分類，
那麼就可以利用兩邊能量的差異獲得無限的能量。

（當然的，馬克斯威爾惡魔違反了熱力學第二定律所以等於不存在）



拉普拉斯和馬克思威爾，都是歷史著名的偉大學者，
這些悖論只是他們提出的有趣理論罷了，他們在人類科學上的貢獻是極大的。



我自己想了個角色如果能使用「羅素的茶壺」這樣的能力，

大概有點類似幻術吧。

「我以羅素的茶壺假定你已經死了，除非你證明你並非死亡的狀態，不然就去死吧」

我覺得很強啦，只是我這個應用的例子好像有點爛，抱歉。

要懂這些 請先搞懂

組合語言 (assembly code)
執行檔格式 Windows 是 PE 檔, Linux 是 ELF 檔
系統函式 (Win API, Linux system call)
