雖然說是七日遊，但是晚去早回的關係

實際上真正有行程的只有五天
機場 -> 入住唐津
https://i.imgur.com/62L3JOI.jpg
因為是晚去早回的飛機，所以到機場大廳時已經快晚上9點了
有事先預約ORIX的接機服務，本來以為會有接機的人舉牌
看來是我太天真了，到了之後再撥電話給營業所，他們會開車來載你
一些手續弄一弄，正式上駕駛座已經是晚上9:40了
第一次在日本開車，一開始就被雨刷洗臉...
之後一路開到唐津，入住"河畔の宿からつキャッスル"
相當推薦這間住宿，河畔風景、日式客房，和風早餐，完全超乎我的期待
老闆相當客氣，離開時還會幫你拍照


第二天 (2019/9/14)
虹之松原(車  内
欣賞) -> 鏡山展望台 -> 山崎  団地  児童公園 -> 唐津車站
-> 唐津市鄉土會館Arpino -> 舊大島邸 -> 舊高取邸 -> 唐津城
-> 唐津市  歴
史民俗資料館 -> 七ツ釜 -> 名護城遺跡 -> 波  戸岬
-> 濱野浦梯田->ドライブイン鳥 -> 入住松浦"アミスタホテル"
https://i.imgur.com/ttZdxKi.jpg
早上吃完早餐後直衝鏡山展望台，中間會開過虹之松原
下山之後稍微繞去山崎  団
地  児童公園，一個社區小公園
最後沿著北海岸，邊開車邊逛景點

晚餐吃ドライブイン鳥
到的時候發現人超多
我們等了一個多小時才順利入座
最後入住松浦"アミスタホテル"


第三天 (2019/9/15)
田平平戶口車站 -> 平戶大橋 -> 平戶城 -> 松浦史料博物館
-> 三浦按針の墓 -> 六角井  戸 -> 鄭成功記念公園
-> 鄭成功兒誕石 -> 鄭成功廟 -> 鄭成功記念館
-> おさかな  広
場 小佐々店 -> 神崎鼻公園 -> 展海峰展望台
-> 佐世保市 ->   稲
佐山山頂展望台 -> 入住"長崎オリオンホテル"
https://imgur.dcard.tw/jB5ZajR.jpg
先去日本最西端的車站 - 田平平戶口車站 拍照留念
再往平戶市前進，中間有平戶大橋可以欣賞
逛完平戶城和平戶市區後，便去朝聖仁王主角的墳墓
再來便驅車朝向鄭成功的出生地前進
這趟旅程讓我知道了原來大家都不知道鄭成功的出生地在日本...
我都回答我要去朝聖鄭成功的出生地
然後每個人都露出不可置信的表情
"原來鄭成功是日本人!!!!!"
"呃....."
鄭成功有一系列的景點能逛
但是我們這群傻逼沒發現鄭成功記念館是可以進去的Orz
沒進去覺得超可惜，平戶大概也不會來第二次了QQ

離開平戶後，先去日本本土最西端之地拿取極西點的證書
再去展海峰展望台欣賞九十九島的海景
本來晚餐是想去佐世保吃當地的特產漢堡
結果當天剛好有艦娘的活動跟當地的漢堡店合作
導致所有店的漢堡都被提督們買走或預定
我一個漢堡都買不到!!!  真的很誇張!!!
最後只剩連鎖漢堡店LOTTERIA能吃
認真說，LOTTERIA有夠難吃......

再來就開夜車直奔  稲
佐山
自此我終於湊完日本三大夜景!!!
個人覺得函館比另外兩個好看就是了

平和公園 -> 長崎原爆資料館 -> 眼鏡橋 -> 長崎新地中華街
-> 長崎水濱森林公園 -> オランダ  坂 -> 哥拉巴園 -> 鍋冠山展望台
-> 祐  徳
  稲
荷神社 -> 武雄溫泉樓門 -> 武雄圖書館 -> 入住"湯快度假村 嬉野館"
https://imgur.dcard.tw/RQ4QSFA.jpg
長崎一日遊兼"來自繽紛世界的明日"巡禮
因為我沒看過這部，所以是陪朋友拍照der
中間還遇到一個中國來的留學生
看我們在鬼鬼祟祟的對照場景就被發現我們是來聖地巡禮的
稍微聊了一下幾個聖地巡禮的地點

結束長崎的行程後便前往祐  徳  稲荷神社
來的時候已經有點晚了，所以人很少
再加上風鈴一直響，其實有股詭異的氣氛XDD
但是這裡真的很漂亮

結束後前往武雄，武雄圖書館非常有設計感
不過怕打擾到使用者，所以只允許在特定的位置拍照


第五天 (2019/9/17)
神社 -> 足湯 -> 公眾浴湯嬉野之湯 -> 吉野里歷史公園 -> 佐賀城跡
-> 入住"アークホテルロイヤル福岡天神"
https://imgur.dcard.tw/n6LsA08.jpg
因為前幾天太累，所以今天想要放縱自己一下
難得來到溫泉旅館，早上想再泡一次溫泉
所以離開飯店時已經快中午了

其實蠻推這次入住的溫泉旅館
CP值很高，因為是平日入住，2人一晚不到3500台幣
還有提供免費浴衣跟早餐，溫泉也是泡免錢
不過因為中秋連假的關係，一堆台灣人XDD

比較可惜的是，因為行程時間關係(早上在耍廢)
吉野里歷史公園蓋個百大名城章就離開了，覺得難過


第六天 (2019/9/18)
舞鶴公園 -> 福岡城跡 -> 太宰府 -> 太宰府政廳跡
-> 天神水鏡天滿宮 -> 逛街 -> 回飯店
https://imgur.dcard.tw/DlR5yuI.jpg
因為昨天把車還了，今天都是走路跟大眾運輸行程
福岡城跡在舞鶴公園裡面，基本上只剩斷壁殘垣(!?

再來前往太宰府
由於季節不對，所以基本上沒有任何花朵
不過遊客還是非常多，也非常多台灣人

晚上就逛逛福岡市區
感受一下大城市的氛圍
居然還被看起來像是牛郎公關的搭訕
我朋友說他一定以為我是OL = =


第七天 (2019/9/18)
因為晚去早回
這天基本上是拿來買土產的
買完就準備搭飛機回台灣



想要的行程太多，但是假太少
所以有些景點幾乎都是走馬看花的跑完
在日本鄉下開車幾乎沒什麼壓力
只是有時候google會亂帶路而已....
