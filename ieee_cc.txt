還有音game類的遊戲難度是沒有上限的...

三星譜差不多已經不會像easy或normal那麼無聊了
就先停在這邊慢慢練吧

: 小弟2011年到現在9年，一路斷斷續續玩，就5000pp左右的一般玩家
: 稍微分享一下心得
: 1. pp計算的方式修改過很多次
: Osu早期是可以透過把次難甚至次次難的長譜打到高 ACC S 甚至 SS 來衝排名
: 不過後來 peppy (製作者) 也針對這些方面重新設計過，變成注重星數跟ACC的形式
: 而在這種公式下，玩家發現開 DT 的效率遠高於 HDHR，因此開啟了大 DT 時代
: Meta 變了所以讓很多讀譜力驚人但手速也許沒那麼頂尖的玩家被淹沒了不少

osu玩完一張譜面的成績計算有pp跟分數兩個系統
(分數是算每個譜面的排名，pp是算玩家的綜合排名)
兩個算法不同，但是都有很奇怪的一點
就是combo占比非常重，分數的算法甚至是跟combo數的平方成正比
以至於同一張譜面除非拿到full combo不然就拿不到pp
所以後來很多人喜歡打短跳譜就是因為這些相對好拿pp，排名才衝得上去
但是在特定類型有所專精的玩家還是不少
比如說最近剛改ID的-GN
可以看這個影片感受一下 https://www.youtube.com/watch?v=L8PLaGsXFt4
(但是這個也拿不到pp...

我覺得pp系統也是對新玩家是否能持續玩下去的阻力之一...

(分數很吃combo這點其實某台灣廠商出了幾款音game全部都是...
不知道設計的人是怎麼想的

: 2. 台灣區的頂尖玩家知名度減弱
: 全盛期時台灣的 Top 玩家是可以穩定肛進世界前 50 的 (Ranking 第一頁)
: 很多名字當時是全世界玩家聽到都會肅然起敬的，維克、Rucker、小K.....等等
: 前兩位都曾殺進世界第 1 與第 2 位，當然每人都有自己的因素終是離開舞台
: 不是說現在玩家的實力不好，但這方面明顯歐美進步的幅度比我們大得多

最近歐美玩家的表現真的恐怖
早期osu還是亞洲人的天下，現在已經沒有優勢了

: 3. 粉絲自成小圈圈
: Osu 作為只要你有譜，即便網路線拔掉都可以打得很開心的遊戲
: 長期會進多人房連線的玩家本就不多，多是自己刷譜刷累了才會去消遣一下
: 玩家之間的連接性不夠強，加好友方面也不是強制雙向性 (A加B好友 但B不會知道)
: 我自己屬於邊緣玩家，沒有參與論壇的 Line 群或 DC 群
: 但長期觀察玩家的生態，你會發現宅宅相輕這現象非常容易發生
: 時不時有人在戰排名、戰人虛偽、戰人跪舔高玩
: 因此新手玩家要想進來這個圈子，阻力是非常高的

這種事情在哪裡都會發生orz

: 4. 7星以上的譜速度已經逼近反應極限
: 承第 1 點，在開 DT 的情況下各類譜的難度都會大幅提升
: 在星數超過 7 星的情形時，譜面的速度已經會快到超過大部分人的極限
: 以享受音樂的角度來說樂趣也不高 (1.5倍速、而且 overmapping)
: 如果你是個已經22.23歲以上的玩家，那很抱歉你有很高機率沒辦法再繼續突破
: 因為單純的手速或眼睛就會比不上國高中的玩家，這點在職業電競很常見

像我是從大三開始比較認真在玩，說真的起步就比別人慢了
現在我大概極限就是六星出頭，感覺不太可能再進步多少
而且常常玩到肩膀不舒服...
不過這遊戲的好處就是一直會有新歌新譜面，可以挑自己適合的難度打
而且有些低星譜還蠻有趣的
套我朋友說的話，三四星附近一堆坑....

: 這些是我幾年遊玩下來的一些小心得，當然人外有人、天外有天
: 相信一定有我概括的這些狀況的例外，就請大家分享囉。

其實音game好處就是只要有新歌就有東西可以繼續玩
加上osu本身是社群在管理，整個生態已經非常成熟了
而且全球知名度也夠，不怕沒有新玩家加入

osu還有很大一個爭議就是版權的問題

但披著悠閒的外皮

其骨肉卻是相當硬核

用可愛的老鼠和溫吞到讓人放鬆甚至想睡的音樂

包裹著需要精打細算謹慎抉擇的內涵

如果以為是輕鬆小品就隨意遊玩

那死亡機率我覺得高達8成

一開始沒有特別為任務留需求道具

而是選擇升級你的船艙

那你將會面臨著缺錢買油買糧

最後會導致老鼠在野外被襲擊或是老鼠把你生產出來或留著有用的物品給吃掉

不得不重玩或是讀檔重新再來

一步一步皆需謹慎

野外遇到事件

可以當好人也可以當壞人

救老鼠可以收下對方當夥伴

也可看機器人值錢而出手抓來賣(我沒選過，是爬文看到這個情況)

野外事件甚至有遺跡探索

一個不小心你的老鼠是會死的Orz

到了第一次換船時

雖然僅僅需要2千元

但你原本的船做過得升級都會不見

船艙需要重新建造並且重新升級

沒有留存一定物資的話

你就要重新花時間去蒐集製作來升級

這時沒多預備錢

那可能又會導致你沒錢買油買糧

而陷入惡性循環

如果說第一次換船還只是新手教學

你靠累積輕鬆度過而沒遇到困難

那第二次換船就真的要千注意萬注意

第一次更換還在安全區域

不會遇到敵人襲擊

但第二次換船卻是深入敵營

我就是在第二次換船遇到挫折

沒注意留存物資不夠

讓我先去規劃其他船艙導致機槍船艙無法直接造出來

還好原料還夠

糧食也有大量儲備

讓我可以在原地等待機槍船艙造好

然後出發重新蒐集材料

雖然到現在我也還沒把船艙弄回第二艘船那個等級

甚至也沒再接到獎勵油的任務

但勉強還能維持下去

只是要乖乖回初級地區重新撿材料順便解任務存錢存糧

真的就是暫時勉強支撐

感覺沒啥用

但因為回去蒐集資源暫時不需要才沒蓋第二個

只是空間有保留

雖然有入手高級一點的火砲船艙

只是也缺資源蓋不了

現在超頭痛不知道該如何規劃

才能保證能生產每種資源且能安全進行遊戲

好像下一次換船需要電池

資源超缺啊Orz

會。沒有，就繼續投，投到有，這應該是7成的漫畫家出道的方式

少，你永遠不能指望自己是one那種天賦異稟的人，就算你知道跟著其他人的道路走希望

講回主題，我覺得閃靈是一個浮動戰力，這不是指它的戰鬥階層不穩定
而是角色很容易因為多種原因變強變弱，主角也是。

劇中的設定又有少年時期雷帝是前期最強，所以在爬無限城時，銀次受到影響
變回了少年雷帝，隨便就丟落雷，一直到雷帝任務結束，這階段銀次
某種意義上才超越雷帝，有點像另一個角色統合者的概念。

那麼，赤屍有多強，全劇強而穩定的人，除了一半算是搞笑漫畫的不死之身
他的死亡聽起來都像是打嘴砲，作為讀者，很難確定他是
真的想像不到自己死亡，還是單純隨便說說

因為他完全沒有"復活後的虛弱"，隨手一抬又砍出一條血線，轉頭把蟬直接打爆

蟬說 你比我還像是怪物。

美堂在蟲篇反而打得很艱辛，鬼蜘蛛要開七成(當然美堂可能也是隨便說說)，打
毒蜂第一次完全沒占到便宜，第二次最後作者以"暴力"來解釋，毒蜂也說自己更方面都占
優勢，但是比不上這種暴力，這種暴力不是力量。

我想就是跟無限城、世界真相，某種契合度，也就是意志之類的。

但主角群並不是常常都能這樣發揮，包含爬無限城，一開始被游利暴打，老爸輕鬆虐，
同時間的赤屍怎麼爬塔的，帶著拖油瓶海溫，直接揮一刀走上去，甚至
一刀還可以逆轉無限城崩潰。


到了美堂蠻阻止少年雷帝失控，再次使用惡魔之手，對戰老爸後變成天使翅膀
這時候的美堂才真正戰勝全力的赤屍，當然赤屍又很快地更新外掛，丟出了刀
阻止了美堂因邪眼而消失的詛咒。

我記得馬克貝斯與無限城意志(倉庫)對話中有提到，他和赤屍都是未知數
而博士(馬克貝斯老媽)也調侃過赤屍像是另一個勢力

赤屍就是無限城天花板，他也不太會因為什麼原因變弱，即便有時候劇情讓他變弱，
像是早期被銀次大磁鐵吸出刀子而敗北(早期是金屬刀)，或是我想像不到死亡，
下一刻就又站得直挺挺，刷刷刷A怪，其他角色再怎麼樣也得
有一些情緒變化，也可以說赤屍在變成超越者的那一刻，就已經領悟到
有意志就夠了，然後超級單純地相信它。

才會不斷說出 我只要任務有趣。

也這因為這種單純慾念，在閃靈世界才特別強。
反觀主角一行人都有各自放不下的羈絆，造成他們容易浮動。
赤屍也說過蟬想復活兒子是愚蠢的事

在無限城中，美堂爸跟三騎士也早就死掉，靠著執念活下來，卻離不開無限城。
如果赤屍那句死掉是真的，那麼赤屍便是幾乎唯一一個不在無限城內也能靠"執念"
不死的人。
　
相對來說倉庫被提拔的"眼睛"超越者，鏡形而，或許正因為執念複雜，
像是我要證明給美堂阿嬤看、我要取代誰誰誰，遠沒有比
"因為興趣而擔任搬運專家"的赤屍還強大
