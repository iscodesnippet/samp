───────────────────────────────────────

原文恕刪

因為有很多可以說的，所以多開了一篇應該比較好

我也是剛接觸 MediBang Paint的人

實際上工具有很多跟電腦繪圖差異蠻大的地方

先從工具開始說起

https://i.imgur.com/9ZS21II.jpg

這些是我比較常用的工具，儘可能的標示清楚

筆刷基本上只用 (畫筆) 跟 (畫筆(起筆/收筆)

而畫筆(起筆收筆)
這個我把(筆壓不透明)取消跟(強制起筆收筆)取消

主要是拿來填色用的，因為我的線稿不是密封的

要注意的是，圖層的順序：由上到下應該是

線稿、顏色圖層(顏色能分得細會比較方便)

如下圖：https://i.imgur.com/SVE1jcd.jpg

Line資料夾下面各部位的線稿、Color資料夾下面則是顏色的區塊

只要能做到這點，之後上更細部的色彩會很方便


-----------------------我是分隔線-----------------------------

至於說到怎麼畫圖

我雖然不是專門的，不過可以跟你分享一下我的路途

https://i.imgur.com/l484jy5.jpg

這個可以看到2010年開始的圖

因為一直沒有正規的學習過畫，處於缺啥補啥的情況

所以進步緩慢

不過總體來說，我覺得學畫可以分成幾個部分

1、毅力：這條路會不斷地被自己的圖噁心
所以首先要先有心理建設
一開始畫得不好很正常，請不要灰心

2、先從掌握物體相對位置開始，
比如說：
頭髮到眼睛的距離 是 眼睛到下巴距離的多少
眼睛之間的間距該是多少(以眼睛為單位的的話)
她身體下巴到肚臍的距離大概是幾顆頭   之類的

8.用戶忠誠度也是我們的優勢
https://i.imgur.com/7of5jyF.jpg

他們沒有這類專業知識，這也是我們的優勢之一

如果你考量到我們只有60人，就很容易理解優勢在哪裡了

※ 引述《sb8888 (V5)》之銘言：
: 問題:
: 1.線條很醜 導致我要描線描很久(幹我一開始不知道有防抖動這東西)

1.線條不好看可以選擇不畫線

我是認真的，完成草稿之後直接從色塊開始動手也未嘗不是種不錯的選擇

其實你現在的線條已經算很好的了，沒有太過毛躁，也有在注意輕重的節奏變化

  如果想要練習的話，我也只能提很老套的建議：

多觀察：

別人是怎樣用線條概括物體的形狀/材質/光影的？他們對線條的取捨可能出自什麼需求？

練速寫：

我對畫面有怎樣的需求？想畫怎樣的畫面？要怎樣施力、怎樣控筆才能畫出理想的效果？

觀察→練習→反思→練習，確實老套，不過也確實有用

: 2.骨架
: 沒學習過 我畫這個小莫的時候一邊查女性肌肉一邊畫...
: 然後就變成肌肉女了(太凸顯肌肉? 這個網路上好像是骨架臨摹學習的
: 是否用人偶方塊+畫上肌肉來做快速練習?

2.網路上各式各樣練骨架的方法都有，其實你挑順手的去練就好

但我個人建議練習人體骨架與肌肉一律認準醫用/藝用解剖書

坊間現有的所有骨架概括方式終歸還是從解剖學出發，練久了早晚都要面對它

既然如此還不如直接從它開始，少繞一點彎路

順帶一提素描人偶或網路上的一些免費3D人體模型，建議初學者少用，

它們不但通常動態僵硬，有時還會出現非常嚴重的錯誤

多觀察與速寫現實人體(不要修圖修很大的那種)帶來的幫助永遠是最多且最好的


: 3.我是否要完整的畫完一張圖
: 我花了一堆時間才畫好形狀 但是衣服配件背景這些我也都沒畫過
: 要一次練好一張圖感覺花的時間不知道划不划算
: 這次我可能會畫完 我想問的是之後練習都要畫完一整張圖感覺進步會很慢?
: 要怎麼做比較好

3.當然要！完整完成一幅畫帶來的幫助永遠大於沒有目標的練習

衣服、配件、背景這些細節永遠服務於整體畫面的效果，

為提升單一元素的完成度而放著平衡整體的能力不管，實在得不償失

拿應試素描來說，有些學生會非常用心在學如何畫好各種特定物件，比如說蘋果好了

結果到考試時，畫面明暗關係混亂、透視糟糕，只有一顆顆精細的蘋果躺在紙面中央，

這樣就是細節上合格，但整體非常差勁的畫

一切練習都要先有了目標才有意義，

比如說我正在畫一張畫，發現衣物皺褶很不自然，對畫面造成不良的影響，

此時我就可以針對衣物皺褶練習，去畫素描、畫色稿，

我練習的目標是練好衣物皺褶，而練好衣物皺褶是為了整體畫面平衡美觀

如果單純為了練習而練習，往往會落入窠臼

完成一幅畫增加的是你的綜合能力，練習是增加你特定方向的能力

對於剛起步的人來說，完成一幅畫的經驗與能力比甚麼都重要

: 4.上色
: 完全不會,看那些上底色的流程 感覺不通 不過我也就嘗試上底色而已
: 可能畫得差不多會再來問問看

4.有個挺偷懶的方法，就是填好物體的固有色(在均勻白光照射下物體本身的顏色)後，

另外開一個圖層畫陰影(用色彩增值等屬性)，這樣就可以把上色簡化為素描

色彩這個領域要細講能說的東西太多了，先給你一句重要的話：

色彩的根基是明暗

把畫面的明暗關係顧好，基本上顏色就比較難變得難看(雖然還是有可能很難看)

: 5.筆刷
: 筆刷我實在不知道該怎麼用怎麼學  我從頭到尾都用圓筆

5.與細節一樣，筆刷服務於畫面效果

使用特定筆刷的理由，只會是為了較為省力地得到某種特定的畫面效果

講難聽點，畫得難看問題不是出在筆刷上，是出在畫的人身上

這種問題不是換筆刷就能解決的，換個筆刷就會畫畫不過是幻覺
