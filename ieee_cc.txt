: 世界衛生組織（WHO）去年在第72屆世界衛生大會宣布通過「國際疾病分類」第11次修訂
: 版（ICD-11），正式將遊戲障礙（Gaming disorder）認定為疾病，給了它一個疾病代碼
: 「6C51」，將於2022年1月1日正式啟用。
: 遊戲障礙是什麼？專家告訴你
: 台北市立聯合醫院忠孝院區精神科主治醫師李旻珊說，簡單地比喻就是玩家的大腦被遊戲
: 「綁架」了，它讓玩家的大腦無法去思考或執行其他重要的功能，就算想要掙脫也不輕易
: 讓玩家掙脫，停玩遊戲幾小時就渾身不對勁、心癢難耐、坐立不安。同時遊戲障礙也可能
: 會和注意力不足、憂鬱、焦慮、睡眠不足、社會適應、生活品質問題互相連結。
: 「遊戲障礙」強調的是玩家的失控行為模式，而不是玩遊戲本身這件事；WHO官方也曾表
: 示，從過去的研究顯示，在所有參與數位或影片遊戲活動的人中，只有一小部分人受遊戲
: 障礙影響；李旻珊認為WHO將遊戲障礙列入疾病，是希望增加政府醫療單位和醫護人員對
: 這個問題的關注，進而去發展預防措施和治療方法。
: 李旻珊指出，其實許多事業成功人士或優秀學生也都曾經或是現在都有使用遊戲來調劑身
: 心壓力，只要時間分配得當，讓遊戲適度幫助自己紓壓，可以不用過度緊張，將遊戲列為
: 拒絕往來戶。
: 如何防止遊戲成癮？亞洲國家這樣做
: 事實上，以亞洲舉例，南韓於2011年就通過法案，規定要進入線上遊戲需要有個人身分認
: 證，並且禁止16歲以下青少年在午夜至凌晨6點間玩網路遊戲，預防青少年過度沉迷遊戲
: ；中國於2017年明確規定網路遊戲採實名制，並限制未成年玩家的遊戲消費額和遊戲時間
: ，並增列「防沉迷」設置，像是騰訊就推出「成長守護平台」，家長可以透過登入微信去

: 效管控未成年子女線上遊戲使用時間。

 標題  Re: [問題] 兩神聯手的龍絆是怎麼涼掉41%營收的？                         
 時間  Fri Jun 26 10:28:48 2020                                               
───────────────────────────────────────


玩過比較久的遊戲有CGSS、

白貓、戰女


開服玩到現在+手機裡只剩

龍絆的感想

A.缺點

1.活動劇情內容及活動類型單調

不是WAVE就是RAID

但是像上次的美猴王跟李弗

意外的獲腐女眾喜好，據說

是跟美猴王的胸肌有關

2.劇情子供不子供要黑不黑

看起來很黑的背景跟劇情走向

但卻搭配了很子供向的人物智商

就像這次的14章劇情。

　雖然我是可以接受，畢竟這

只是手遊不是什麼次世代主

機３Ａ大作。

3.卡面不夠香(主觀)

雖然對我來說是很夠用的，

但要跟艦B、CGSS、明日方舟

這種明顯賣香的卻又明顯不足。

4.人物及搭配變化少

雖然主打動作遊戲，但模組、

護符到武器甚至龍，其實能

選擇的變化有限，組合不外

乎技能傷害符＋狀態特攻符

＋技傷龍就是王道。

更不要提玩家間或野團

生態也有一部分的人帶有不知所

以然的優越感或歧視

例如超風牙非3隻馬爾斯+瑪斯

不打的出狀態特攻或技傷龍還

會吃個狗妹捏

　狗妹捏貼圖連發，更不要說出

非頂尖一線角，但必須說這只是

一部分玩家間的心態。

5.高難度副本讓手殘退卻

雖然我也經歷過這段時間，現在

也沒說操作多風騷，觀念多好

但這遊戲的高難副本

（真龍、咢牙）畢竟有一個

門檻在。

要攻略就是得先看個影片

知道流程或應對法，或有

人帶，但躺在地上過的話

相信你遊戲體驗通常不會

太好玩得下去的人畢竟有

限。除非就是打定主意要

一直打活動劇情。

6.遊戲生態斷層

以前有統計打真龍的玩家可能只

佔了整體10%，這裡面有些人也

已經全武器全建築滿等制霸所以

都改為AUTO單機模式，所以環

境變成熟悉關卡但還有素材需求

的人+剛不熟關卡且進入高難副

本的玩家，所以當在面對超龍

超牙的時候，野團的攻略難度

大幅上升，因為最高難度的副本

建立在大家都知道機制且火力充

足。變成在沒有群組親友的幫助

下，無論是哪種玩家都會有攻略

困難的感受（雖然我覺得這是樂

趣之一，但要建立在即使有人在

武器不足或流程不熟的狀況下

　有溝通管道）。

7.課金體驗差（看人看運氣）

免費仔跟重課長的差別可能只有

武器數量我自己只買過新手優惠

包，大概100多塊，但各種高難

關卡也都能湊出2-5隻應對角色

這遊戲課金就只是讓你的龍跟武

器可以早點滿突，

最划算的應該是超得

讓你補足自己沒有的普池龍或

角。

但只要你不要亂升級、無謂的耗

費資源

基本上也都能讓你弄出1.2隻角色

+龍去應對其中一種副本，然後

農出武器後開始五屬循環攻略。

再來就是抽卡沒天井，這點我是

覺得無所謂，但對不是財富自由

又有課金抽角需求的人來說就不

是那麼回事了，不過手遊抽卡就

是那麼回事，我覺得

還是適度+心理調適會比較好。

B.優點：

1.操作難度跟動作遊戲遊玩感
　
就動作類的遊玩體驗對我來說是

不錯的，

　遊戲群有人沒拓荒過以及共同

討論如何攻略的感覺很不錯，

進度比較前面的玩家

　甚至AUTO黨只要能在新手拓荒

時共同參與或建議，大家就能

有不錯的遊戲體驗。

2.不用重課也能玩得下去

當然這一點見人見智，反面解釋

的話，就會變超糟的遊戲體驗。

每個關卡你只要有

1隻角色跟龍就能入門檻，在現

在連鎖EX以及角色成長、傷害降

低，非一線角也能

出頭的環境下，門檻確實降得比

以前低了，所謂的高難度操作對

我個人是熟悉度問題，當然也有

人天生操作不靈光那沒辦法，

真的玩不下去不要勉強自己，如

果真的很有愛想玩下去，那我建

議加群找人帶會好一點

3.按部就班就能玩得下去

農素材、做武器、提升建築、龍

絆日加倍的時候抽抽補一下角

色，只要你慢慢地提升

進度，基本上角色戰力就會變

強，不用靠臉來決定。當然有人

權角+龍可以打得更快，

但這遊戲好處就是並不是一定要

頂級角才能過關。



當然現在玩家間也有一些人堅持

要全關卡AUTO才叫玩得下去，

但自己的隊伍配置卻又沒有達到

標準，於是就開始覺得這遊戲爛

的要死那我也只能說那是你選擇

的，像我自己不能全AUTO也還

是能得到樂趣。

遊戲要怎麼玩得開心取決於自

己。


總結：我會推薦給喜歡動作類

型、喜歡跟其他人

互動討論怎麼一起共鬥攻

略的玩家。


額外心得：這遊戲玩的人真的已

經不多了， 拜託玩家

之間少點無謂的優越

感，在群組裡多帶帶

剛接觸高難副本的新

手，或偶爾去野團救

世一下（野團觀念不

差的前提下）

https://ec.ltn.com.tw/article/paper/1382384
禁「全球同服」，中國玩家鎖國自娛

中國自二○一八年起強化監管線上遊戲，不論是海外遊戲或中國自行研發，必須宏揚社會
主義核心價值觀及樹立中國文化自信等原則，且限制宣揚西方普世價值、民主思潮及言論
自由等，更不斷限縮發聲權，近年來要求中國專屬伺服器、不得「全球同服」，「讓中國
玩家鎖國自娛」。

一九九○年代，外國遊戲進入中國，當時並沒限制外企發行及內容審查，多透過電子工業
部（現為工信部）旗下中國電子物資總公司，或經新聞出版署旗下的中國圖書進出口（集
團）總公司提出申請，依圖書出版物流程，出版社自審即可上市，只是對照說明書，打開
一、兩個頁面象徵性審查。

隨著線上遊戲興起，文化部在二○○三年施行新規，線上遊戲定義為「互聯網文化產品」
，要求提報文化部進行內容審查；除了新聞出版署、電子工業部外，又多了個文化部「交
叉管理」。


二○一三年，中國國家新聞出版總署、國家廣播電影電視總局合併為國家新聞出版廣電總
局；二○一八年，又將國家新聞出版廣電總局一拆為三，國家廣播電視總局屬國務院直屬
機構，新聞出版管理職責、電影管理職責劃入中共中央宣傳部，改掛國家新聞出版署、國
家電影局，「由中宣部監管線上遊戲」。

台灣業者說，早年除了涉疆、涉台及六四、四九年前歷史等敏感內容，得「自我閹割」外
，其餘監管相對寬鬆，但近兩年來，題材內容也不可宣揚西方普世價值、自由民主思潮等
，甚至透過AI限制玩家發聲權；例如今年六四期間，中國多款手遊自前一日起關閉聊天功
能，避免誤觸審查紅線。

中國知名遊戲媒體「遊戲葡萄」之前披露，中國官方曾在三月中舉行會議，對遊戲監管訂
出更嚴格規範，包括不得宣揚「全球同服」功能，在中國營運線上遊戲得替中國玩家打造
專屬伺服器；實名制也從線上遊戲，擴及到任天堂Switch等主機遊戲。
