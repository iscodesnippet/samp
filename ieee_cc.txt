在預售組合中，效果怪獸的體質普遍比同級的白板怪來得差勁。

擴充包出了之後，可以看到更多在設計上的思路。





https://i.imgur.com/OUm4wdU.jpg

舉例來說，這張亞古獸是很基礎的等級3效果怪獸。


登場 Cost = 3    代表他直接從手牌上出場，需要耗費3格行動刻度。
進化 Cost = 0    紅色代表顏色，代表從紅色等級2怪獸進化，耗費的刻度是0。


登場之後 DP（戰鬥力）= 2000，具有效果。




https://i.imgur.com/triu16t.jpg

而紅色等級3的白板怪獸是這樣的


登場 Cost = 3 
進化 Cost = 0

DP = 4000沒有效果，出場限制相同但是戰鬥力是亞古獸兩倍。




登場 Cost = 2
進化 Cost = 0


DP = 3000直接登場的代價更小，而且戰鬥力還是比效果怪獸高。




登場 Cost = 3
進化 Cost = 1

DP = 5000直接登場的代價一樣是3，進化雖然要1，但是戰鬥力5000，已經比一般的等級4效果怪獸還高。






https://i.imgur.com/tcMbfUe.jpg

再看一張等級4的暴龍獸


登場 Cost = 5
DP = 4000
有一個進化元效果。




https://i.imgur.com/33Tn6Xm.jpg

紅色等級4的白板怪獸


登場 Cost = 3
進化 Cost = 2

DP = 4000
非常弱的等級4，沒有效果但是登場只要和等級3相等的刻度3格
等於說可以當作快速跳到等級5的跳板。






DP = 5000
巴多拉獸，登場代價較小，同時戰鬥力稍高，標準白板。





登場 Cost = 6
進化 Cost = 1

DP = 6000
登場需要更高的6刻度，但是進化只需要1刻度，
以他做為過渡可以省下一個刻度，而且戰力是準等級5的6000





這幾天測牌的時候發現，這遊戲很好的把打點和效果做出區分，

而且還有各種設計，有的是代價較大，有的是能加速你進化，有的是戰鬥力的差別。



這增加了小生物快攻的可能性啊。

除了這次的新異能，直擊的時候不會被戰鬥破壞。可以盡情地摸對手之外。




https://i.imgur.com/7KI3BGV.jpg


紅色這次有新卡Brave Tornado，勇氣龍捲風？

DP+2000 外，Check數 +1，用在小生物身上有奇效，沒頭沒腦就打掉兩三張防禦卡。




感覺白板生物快攻是可以開發的戰術。
