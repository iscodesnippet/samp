: 回應一下推文裡的討論

: 2.建築經費

: 台灣是例外中的例外  台灣的RC結構便宜的有點不可思議....
: 反而木結構比RC貴不少
: 原因有幾個
: a.原料
: 台灣林業限伐導致國產材生產斷鏈
: 國外進口木材通常為裝修材  結構材進口量不高
: 也因此無法把平均價格以量制價
: 反而是RC原料台灣自產充足 除了鋼筋以外
: 骨材 (沙 卵石)  水泥  這都是台灣國內最大規模的礦業類別
: 國外對水泥 石材的開採 有時比木材規範還嚴
: 台灣的規範....  所以原料成本偏低

看到就不專業來回一下，歡迎打臉，用力打QQ

進口木材大多為裝修裝潢用這點沒錯，結構材大多是用松木杉木扁柏之類的
但是這一類的木材很少進口

再來已經不是以量制價的問題了，是國際木材價格大漲
外加幾年前原木出口大宗的馬來西亞提升了200%的森林砍伐稅
導致現在木材進口大多轉移至索羅門或巴布亞紐幾內亞之類的國家了，此外也有非洲跟南
美洲的國家
而且最近連木材進口量也大減，所以相對的價格就，嗯！

而結構材像是梁柱的部分需要大的木材，而且要不彎曲不凹損的，要取得如此漂亮的就更
麻煩的


: e.木結構產業鏈
: 台灣在這部分 各個環節斷的比較厲害
: 要找到會設計木構造的建築師和結構技師就不容易
: 送審也很麻煩
: 木工台灣大多是裝潢木工 而不是結構木工
: 作大型木結構的公司湊不滿五支手指
: 也因此成本就會高出不少
: 資訊和技術協助也充足  甚至有一些人都可以自己蓋木結構小屋
: 所以成本便宜不少

木結構的建築師在幾十年前聽說還有不少，現在可以說是稀有般的存在了
之前有個消防隊的來勘查說我老家那邊有個棚子是加蓋的要罰錢
結果跟他同行的前輩看了一下就說，這東西現在沒人會蓋了，怎麼可能是加蓋的
而那個棚子就是用卡榫一類的工法架起來的，歷經風雨都沒事
而且你說白蟻？其實如果你用扁柏一類的就不太需要害怕這點，但不是絕對的
但是為甚麼會有白蟻或蛀蟲的原因打從一開始就是他們本來就在木材裡面了
而且白蟻其實算是還好的了，好發現，好解決
麻煩的是那種妳就算整隻從原木開始加工到最後的板材都不會發現，某天才突然探出頭的
蛀蟲
但是通常不向白蟻那麼危害就是了

如果現在還想看木造結構的東西，可以去很多還沒拆掉的老工廠(尤其日治時期的)
或是宜蘭的那個中興文創中心，裡面的倉庫也是木結構

但木結構真的怕颱風嗎？其實不然，但是連牆壁都用木頭就不好說了

大家好，我是消。

一直想提筆紀錄一下自己喜歡的作品，
但要寫卻又得重新翻上幾次才能寫的完善所以一直沒能動筆，
某方面也是自己的鑽牛角尖吧。

如果說有什麼故事是會讓人還能鼓起勇氣、
繼續在人生這條痛苦的道路上走下去，
近期來說大概只有上次寫到 [去南極] 莫屬，
但這樣的神作並非能夠天天見到，
今天就來聊聊編劇上有些可惜的 [傳說中勇者的婚活]


(下有雷)









這部作品大概是一兩年前看過的，
整個故事結構以短篇形式構築，
我對這個故事的本質理解會偏向 [尋找自我的存在意義]


[我作為非勇者的存在是否被需要]

主角裕里是一個不懂人情世故的存在，
在設定上 [因為拯救世界而不再被世界需要] 是很重要的一點，
雖然主題設定上是勇者相親，
但實際上在前兩話就有提到其實他是不知道自己 [消滅魔王後該何去何從]
（與狡兔死、走狗烹中的 [走狗] 這一存在有些類似）

選擇尋找結婚對象是一個契機、一個選擇，
最終的目標還是彌補他自幼討伐魔王而不像一個正常人的人格，
進而以 [裕里] 而非 [勇者] 找到自己真正想做的事情、
與之同時找到與 [真我] 相知相愛的對象。

編劇中前五話可以說是還在暖機，
大致上的婚活內容都是以裕里沒有人情世故失敗收場，
但從與武器癡 [布金] 結交好友的篇幅來看，
可以看出作者其實是想讓裕里慢慢學習常識成為一個正常人的，
但也可能是因為這鋪陳太過慢熱沒有高潮點才導致後面腰斬。



[暖機太久惹、高潮時機沒有把握好]

雖是世界觀的構築、篇幅與高潮點沒能分配恰當，
前面劇情內容還算有趣但沒有特別深刻的印象點，
而後面花到比較多篇幅闡述的三個女性卻才像動真格的演出，
面癱村姑、蘿莉公主，綜合前面二位特質的——面癱蘿莉妹妹XD
(也有可能是被下了腰斬通牒之類的……)

若是簡單總結這三個大篇章的內容大意的話，
面癱村姑終於打開心房喜歡上一個人，
見到與自己 [命運] 相仿的裕里完成了使命而得到成為 [公主] 的勇氣，
還有因為中間少了許多鋪陳、硬是直接寫到結局而顯得有些可惜的妹妹篇。



[穿著好土!!]
[聊天好無趣!!]
女角攻略失敗。

我認為這個故事可惜的地方是，
大多數的成長都是予以女角們，
你可以看見她們在兩三話之中獲得了改變或勇氣，
而作為主軸最需要 [成長] 的主角並沒有太過著墨，
讀者樂意見到裕里因為笨拙吃土、
卻也希望他能夠從每次的失敗中學到經驗而成長，
更懂一點人情世故、更了解自己如何作為人，
在下次的婚活中能夠更順利的撩到妹這件事反而是一個驗收，
或許是篇幅不夠，
但裕里沒有足夠成長的模樣也是事實，
故事的走向也只(夠)寫給婚活，我覺得是相當可惜的一點。


[勇者?!]
眾人露出驚訝面容。
準備迎接本篇結局……

既然編劇的開頭就寫明 [這個世界已經不需要勇者]，
裕里也在追求 [若我非勇者、我是否還有存在的意義]，
那一直搬出勇者的力量來解決問題就不是一個好想法，
故事內容中可以看到，
幾乎每一話解決事件都是需要搬出他勇者的身分或實力，
這與他的期望某種程度上相違背，
你可以說世界不如他願，
逼迫他必須表露自己身分 (就像小當家那樣嘛XD?)
但現在觀眾的胃口已經變得刁鑽，
只是露出 [勇者] 頭銜大家就高潮的展開已然一點都不精采。

裕里短期內既沒有明顯的成長，
周遭因為他而長大(?)的女孩卻因為這是短篇故事的關係而快速離去，
高潮自然而然就顯得餘韻不足，
女孩成長如果也能帶給裕里成長、相輔相成，
即使裕里下次依然 [失言] (ㄎㄆ:= =)
也還能夠接受並期待他未來成長的 [可能性]，
我們可以說這是編劇上可惜的地方，
但也可以說這種故事在主流的編劇下沒能獲得足夠的篇幅發揮。

針對主角沒有明顯成長的部分，
我認為這是大長篇的戀愛漫畫中最多的詬病，
藉由優柔寡斷意外重重來展現女角們的魅力，
但在這部作品不能這樣做的原因也很明顯，
一來他並非大長篇，
必須在非常短的篇幅中獲得醒悟，
可是如果得到成長的過程太過簡短卻又覺得膚淺虛偽，
(就連完成連載[魔奇]的大高忍老師在畫Orient的現在都還令人捏把冷汗)
二來作者太過忠於主軸，
當然同上所說極可能是因為篇幅不夠之故，
但這方面的東西我想也是跟客群是否能接受如此的編劇有關係，
市場是現實的，
不有趣的東西很容易被淘汰，
所以最近常常看到一些短篇幅被翻譯的漫畫浮上檯面，
主要就是他們有趣，
無論是具有衝突感、製造話題等等，
都是屬於有趣的範圍。

這部作品開場還算可以，
但暖機到第一次真正令人動容的高潮時間太長了，
若這樣想，
以一個沒有太大名氣的作者來說，
被腰斬大概也是必然吧……
很多作品前期在暖機階段都有這樣的狀況，
而死在這階段的作品也不計其數，
可見業界現實之殘酷。


[尋找真我的過程]

如同開頭所說，
裕里覺得 [找到伴侶] 是找到真我的一個方法，
也就是說編劇上最終的結果是找到真我進而得到真愛，
所以在裕里其實沒有找到 [答案] ，
而在腰斬前硬寫的妹妹結局也就顯得沒有那麼令人動容。

妹妹所喜歡的是一直以來他看在眼中的裕里，
願意為了裕里付出一切、捨棄自己身分，
然而這份覺悟的鋪陳不夠，
裕里也還沒有找到自己放下 [勇者] 後又是誰，

裕里的 [本質] 加上某種程度的 [改變]
妹妹喜歡其本質的 [覺悟] 以及是否能因為裕里改變跟著 [成長]
這兩者之間的衝突其實才是這一路線的精彩之處，

村姑篇之後我很期待接下來會有更澎拜的故事展開，
但公主跟妹篇都顯得尾勁不足，
對於本作除了惋惜實在想不到別的說詞。


[對於佳作的感慨]

在一兩年前我看完還只是有些不過癮，
卻說不上是哪裡不好，
以現在的標準來說也已經沒有那時候的感動，
只能說一個好的故事真的太難寫了。

我喜歡漫畫，
更能理解漫畫故事受限過大的難度，
從他這部作品裡可以看出他想寫些什麼，
自從腰斬後我偶爾都會看他還有沒有新作，
直到剛剛去看了一下WIKI，
他這兩年作畫的作品，
畫風變了點、也似乎不是他自己想寫的故事，
若是我大概會相當不得志吧，
如果老師還能有個好一點的機會發揮就好了。

對這部作品如此收尾遺憾的感想就是，
或許在哪邊走對了，
就能達到 [勇者赫魯庫] 或者 [惑星公主] 之類的高度，
但這一切都是或許罷了。

一個好的題材、一個好的機會，是不可一而再、再而三重來的。
時間很殘酷，
而正是因為有時間的限制，
磨鍊與成長才顯得有意義。
若人類都有無窮無盡的時間，
那花十年跟花十萬年達到登峰造極的境界並沒有任何差別，
成長這件事也就顯得一點價值也沒有，

無論何時開始、磨鍊自己變得更強，
才能在關鍵時刻抓住讓自己發光的機會。

抽到某指定從者3張以上的組合是
C5取3*C10取2+C5取4*C10取1+C5取5*C10取0=450+50+1=501

->501/3003=16.68%

因為三個回合是全同可以互換，
不過為了多騙點P幣就繼續算下去

第二回合的牌是第一回合抽剩的=C10取5=252
扣除已經抽出3張以上的組合

-> 第二回合抽到3張=C3取3*C7取2=21

-> 第二回合抽到2張=C4取2*C6取3=120
-> 第二回合抽到3張=C4取3*C6取2=60
-> 第二回合抽到4張=C4取4*C6取1=6

-> 第二回合抽到0張=C5取0*C5取5=1   ->第三回合5張
