我也抄了一副，試過的感覺是這副真的非常強。

跟暴力純紅來打，基本上也可以打個五五波。



命名：攻守萬能型  XDD



https://i.imgur.com/9gybcyn.jpg

這一副主要價值在

美美+光子郎兩張人物卡



美美的效果

共通：回合開始時，刻度低於2時，增加至3。

主要：自己場上有等級五以上的綠色怪獸時可以發動，橫置這張人物卡。
可以在空的育成區翻出一張數碼蛋卡，
或是將育成區一隻等級3以上的怪獸移置戰鬥區。


光子郎的效果

主要：自己場上有等級五以上的綠色怪獸時可以發動，橫置這張人物卡。
翻開自己牌堆最上面的一枚卡片，如果是怪獸卡則加入手牌，反之送入牌堆底。




如果你有看上面那個影片，使用綠牌的主持人
靠著這兩張卡，累積資源的速度就快對面一倍以上。應該是現在系統最完整的顏色。


光子郎每回合穩定的翻一張，高機率將怪獸卡加入手牌。
而美美加速育成區的養成，而且利用這個效果登場的怪獸不會有戰鬥暈眩，可以直接攻擊


只要兩張卡到齊，你一個回合當兩個回合在打，根本就是金科拉




https://i.imgur.com/HgYvv6p.jpg

再加上綠色牌組本身就有快速抽率能力



登場時
翻開牌堆上方三張，其中等級四的怪獸一張加入手牌，其它送入牌堆底。


仙人掌獸

進化時
翻開牌堆上方三張，其中等級五以上的怪獸一張加入手牌，其它送入牌堆底。可以抓五和六



他們倆個的效果都很像閃刀姬的那個場地。翻三張選牌。

再搭配光子郎，基本上手牌會進得非常爽。



再加上薔薇獸（當自己橫置，對手回合可以攻擊時可以橫置一張）
和究極古加獸（自己回合 DP + 4000，貫穿）



透過上面的抽濾系統，你基本上怪物卡住的機率會比對方低很多


究極古加獸 DP 11000 + 4000 = 15000 這是奧米加獸的水準。
                                           今天真的有一場Check到奧米加輸掉 XDD
然後他有貫通，破壞對手怪獸時可以翻一張防禦卡。


薔薇獸橫置時有在對手回合發動的能力，
對手宣言攻擊時可以橫置對方一張怪獸，這能力可以拿來保護究極古加獸。

然後等到自己的回合，究極古加又可以 +4000。



https://i.imgur.com/ngEAsod.jpg

因為投入四張花仙獸，有高機率可以發動花仙獸的進化元效果。

攻擊時
橫置對方一體沒有阻擋異能的怪獸。


有很多的用法。
上面那張 Dimension Scissors，次元剪刀臂，

這回合一體怪獸，戰破對手時可以重新恢復直立。


他是這回合喔，不是一回合一次，可以連續觸發的，
上面那個影片最後也是用這張收頭的。



用古加獸 + 次元剪刀臂

不管是花仙獸還是花仙砲，總之橫置對手怪獸。



攻擊對手橫置怪獸 -> 花仙獸效果 -> 橫置對手怪獸一體。

戰破 -> 古加獸效果 Check 防禦卡。

次元剪刀臂效果 -> 恢復直立。



攻擊對手橫置怪獸 -> 花仙獸效果 -> 橫置對手怪獸一體。

戰破 -> 古加獸效果 Check 防禦卡。

次元剪刀臂效果 -> 恢復直立。



只要對手場上還有非阻擋的怪獸，這個連招可以一口氣把對手場上打空。

而且每破壞一隻，就削減一張防禦卡。


真的是很強 ~"~







這一副唯一的問題是，馴獸師卡上手的速度快不快。

美美 + 光子郎很強，
但是因為各投入兩張而已，所以其實要抽到各一放置在場上，不是那麼容易。

但是如果你放多，會影響到巴魯獸和仙人掌獸，怪獸系統的抽濾。

來打綠紅牌組，這樣應該能夠加速抽到人物卡。

不過還待開發。應該可以組一副出來。
