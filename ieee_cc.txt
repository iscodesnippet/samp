※ 引述《FayWenWang (FWW)》之銘言：
: 競技作品的主角通常都會有兩種描寫主角不凡的設定
: 一種是雖然初學該競技，只是個菜鳥
: 但某些身體素質卻很強或是充滿天分學得很快
: 劇情多半像勇者打魔王那樣，一邊搭配原有的身體能力一邊逐漸把技巧練起來
: 像灌高櫻木、神拳一步、光速蒙面俠瀨那

這種除了讓讀者能感受到主角的成長外

另一個重要的目的就是解釋規則

就像你舉的例子，都是刊登在大型少年漫畫雜誌上

客群很廣，所以得要預設讀者完全不懂規則作為前提

否則如果光速21開場就讓蛭魔秀一堆玩規則的操作

估計第一次打完皇城就被斬了

但是讓老手一天到晚自己解釋起規則又很智障

所以乾脆讓主角當新手，從頭對主角和讀者做教學

: 另外一種則是一開始就是老手，直接在該競技制霸
: 但通常都因為負面的過去而從競技中退出
: 劇情走向則是開頭拼命的被挖角
: 被挖一陣子後不情願的重回場上，然後最後跟同伴們找回競技的快樂
: 但也常常被吐槽明明一開始強的跟大魔王一樣
: 結果隨劇情進展遇到其他對手卻都要先五五開一下
: 舉例像西住軍神、輕羽綾乃、天才麻將  咲
: 特別的是，第一種類型主角似乎以男生為居多
: 然後第二種類型則以女生為居多

這種就偏向描寫人物間的關係

因為會一度退出，多半是有個創傷或是疙瘩在

所以除了主角新的社交圈外，還能挖原本社交圈出來

讓兩群人馬以主角為中心，上演感情戲(?)

這種類型的競技描寫會比第一種淡一些

相對的就比較多在描寫人際關係

: (目前只想得到這些例子，所以應該還是會有例外?
: 有一部漫畫  "女生混入男子羽毛球部"
: 男女主角這樣的差異更是明顯
: 說到底為什麼會有這樣的差異啊?

如果是有戰鬥的劇情可能就還好，可以用畫面掩蓋過去

但到了日常或比較過度的章節時，就會感覺非常的拖

我不知道是不是因為沒有編劇的關係啦

而且我覺得可以只做23或24集就好

不是很懂為什麼硬要做成26集，搞得前面很拖

如果有大大知道希望可以解釋一下，雖然我看UFO其他的半年番也是做26集



再來談談對劇情的心得吧

一開始只知道是在jump上連載算是王道型的漫畫吧

整體來說中規中矩，沒啥好挑剔的點同樣地也沒啥好稱讚的點

還有一個設定不是很明白

一開始通過入隊測驗的時候，有說全部分為10個階級，然後主角是最下級的癸

我還想說哇靠修練兩年練完水之呼吸才通過，想說其他人應該也不差

要爬上去可能要一段時間，結果蜘蛛山的雜魚隊員直接打我臉

讓人很難相信他們都通過入隊測驗，也沒說他們是什麼級的

然後我知道主角可能會強一點，但沒想到直接開打十二鬼月

柱全體也都亮出來了，讓我感覺那前面10級是分好玩的

然後到了關鍵19集的時候，我原本以為炭治郎會有個神操作

把一到十式打完撐到富岡趕來救場，畢竟刀都斷了

結果一個突然回憶老爸，然後就用鬼之神樂呼吸打贏了

雖然畫面上我看得很爽，後來想想不對啊

前面練那麼辛苦的水之呼吸到底是幹嘛的

也不是說不能這樣，但前面可以先給些暗示吧，不然我覺得這樣轉得很硬



總結來說鬼滅之刃還是值得一看的動畫

畢竟UFO的實力擺在那，作畫的精緻、戰鬥的分鏡、特效的處理

這些到現在我覺得也是業界中一等一的水準，

但讓我比較反感的是一直有人吹神作，你吹漫畫神作我就算了

但吹到動畫我就不認同了，畫面上是很過癮啦，但前面劇情我看得很氣啊

我雖然不是資深UFO迷，但近幾年的作品大概都有看一下

從看不太懂得空境、FZ、UBW、噬神者、熱情傳奇到最近的HF

真要我說大概跟UBW同個等級啦，要吹得話好歹也要到FZ吧

不過最後改編還是很成功啦，畢竟帶起一波轟動和提高了原作的銷量

至於之後的劇場版就看板上的評論再決定要不要看吧

但漫畫我是不會看啦

雖然還有一些想說的，不過首PO就先到這邊吧
