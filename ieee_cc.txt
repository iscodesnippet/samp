跟動畫中一樣的角度拍照，想看專業一點的可以參考這兩篇：

中二病的巡禮點主要分成四區：

１．穴太站（車站場景）
２．日野舊鐮掛小學校（校園場景）
３．大津石山（城市場景）

＃出發

站要前往穴太站，從大阪到穴太車程約1:40，路線如下：

JR新今宮 => JR大阪 (大阪環狀線)
JR大阪 => JR大津京 (東海道、山陽本線 往近江今津)
JR大津京 => 皇子山 (步行)
皇子山 => 穴太（滋賀） (京阪石山  坂本線 往  坂本(滋賀))

下一站是主舞台日野，劇中的校園相關景點都在這邊，從穴太出發的車程約1:40，路線如

JR草津 => JR貴生川 (草津線 往柘植)
貴生川 => 日野 (近江鐵道本線 往彥根)

＃後記

果要去巡禮的話，建議爐石存京都，然後分兩天跑：

第一天：京都　=>　穴太　=>　敦賀　=>　京都
第二天：京都　=>　日野　=>　大津石山（因為場景是黃昏及夜晚）=>　京都

講一點自己的想法
日月因為小智奪冠不少人和跟風雲就吹翻天了
只是這過程覺得很可惜是
有些人沒理解日月真正神的點在哪

整個主題要傳達的資訊就是"家族愛"

這家族愛的範圍很廣
人類角色部分
馬澳母女之情/水蓮一家/
莉莉艾家的八點檔/卡奇兄妹之情/庫庫伊夫婦

寶可夢部分也是在傳達無血緣之愛
木木梟直接被銃嘴大鳥養大
銃嘴大鳥日後還是有持續探望木木梟
火貓被犬類寶可夢照護 岩岩狗被庫庫伊照護
後來小智全部接手這些責任

而主人公小智在這主題中也扮演一個兒子的角色
小智的母親花子和假父親大木博士
在這季是相對很關心兒子的狀況
到最後比賽都有來看
無血緣方面庫庫伊夫婦是無條件把小智當親兒子養
最後對決比起勝負本身倒是比較像親子心意交流

這季的Fanservice除了把小霞小剛叫回
是沒有搞太多其他東西
動畫組也很有自信
因為老觀眾在XY/BW負面聲音太多
他們並沒有因為這件事作出妥協行為
還是專心注重在配角群發展身上

-----------------------

新無印的部分
因為丹帝這角色在原作定位相當庫庫伊在日月
所以會被小智打敗不用懷疑了
當那件事發生新無印就要被吹到太陽系
現階段劇情就是滿滿的Fanservice

但我自己視角新無印是不是神作上面都不是重點
小智打敗丹帝如果是這片唯一可以講的事
那就只是成就至上論的爽片而已

可能我自己就是比較愛看感性的劇情

因為小豪的故事線發展
又會牽扯到小智的良友特質
小豪父母因為長期工作
自己封閉在家裡導致社交模式有問題
對上任何人會馬上展現自傲心導致永遠沒朋友

後面一步步實踐加強對方潛意識(看卡拉卡拉那話)
如果小智最後真的激發小豪想法
小豪到了會自我檢視的地步

新無印才會真正變神作

 標題  [閒聊] Cytus 2結局                                                     
 時間  Sun Apr  5 21:02:35 2020                                               
───────────────────────────────────────

Cytus 2在三月底釋出目前這代的劇情結局，
自2018年1月發行以來，我花在這款遊戲的時間明顯比1代少很多，
大體有幾個原因:

1.太騙錢
雷亞從之前的VOEZ到後來的Sdorica，騙課的意圖太明顯。
如果有從Cytus 1代開始玩的玩家，應該都會想起那段陪雷亞公司一起創業的革命感，
從遊戲領域四處求來的電子音樂、作曲家徵稿、集滿付費玩家就開啟新章節...等等，
在當時，玩家和遊戲公司互相拉拔長大。

到了2代，一開始有2免費角+2付費角的設計可讓人接受，
但隨之而來的曲包擴充、新付費角登場次數過於頻繁，
各種沒有章法脈絡的騙課，讓部分老玩家心裡不太爽。

2.音樂不突出
1代雷亞出現的ice、KIVA等本土音樂製作人，或是國外的M2U、Cranky等老人，
2代貢獻的音樂就少了些，
新的音樂能讓人印象深刻的曲目也遠比1代少，
個人認為Ivy帶來的老曲目還遠勝大多數新曲。
(但免費角Crystal PuNK有收錄可推薦的Ring樂團，等等會講)

3.遊戲性未有突破
除了幾個劇情中的突發關卡和魔王關，
玩法沒什麼特別新穎之處(多了長按和側滑)，
隱藏關卡比1代少了許多，感受不到神秘性。

4.主線支線劇情混亂
一開始強硬地把音樂分類給不同的角色，用各自的視角詮釋主線，
搭配類似巴哈討論板/PTT鄉民題材的表面事件，
以及底下的"事件真相"，
剛開始還算有趣，後來就顯出敘事的雜亂無章及時間感紊亂。

後來為了劇情需求，又把所有角色收束在主線，
當初的角色與音樂分類就顯得毫無必要性，只剩騙課的功能。
(多數付費角幾乎都沒有新曲update)
回想1代的主線劇情，只有主畫面機器人Vanessa在頹敗的城市四處走動的身影，
搭配官方設定和玩家腦補，反而比2代出色許多。

以下有新劇情雷





















新劇情講述1代的主角Vanessa因為被文化復興的人類過度濫用，
吸收太多人類惡，導致黑化爆走。
比起前幾次的更新，這次總算有把故事講清楚，
但黑化的過程以及與2代主角群的決戰，個人認為致敬F/SN HF的小櫻太明顯，
總之就是劇本缺乏創新。

1代主畫面呆呆看著月亮的Vanessa被人類玩壞成這樣，還蠻令人感慨的。

不知道是不是在角色設定的溝通上出了問題...
