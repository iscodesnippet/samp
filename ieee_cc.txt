一直進戲院看但是沒時間，今天終於在電視上看到了，雖然少了臨場感，不過可以反覆播放
但是可以看到一些細節。

1.片名原來是 七龍珠超 布羅利，而非七龍珠 超布羅利

實際上以往的舊劇場版幾乎都不算正史，所以以正史而言布羅利是第一次出現，除了傳說的
超塞亞人這點還在，其他的設定做了大幅調整。

最大的特徵，在於舊布羅利跟卡卡洛特是嬰兒鄰居兼競爭對手，所以舊版一聽到卡卡洛特就
會抓狂，新版布羅利的年紀跟貝吉達相仿，跟悟空沒有半毛錢關係，所以這次的布羅利戰鬥
只是本能，而不是惡，對於洗白非常有幫助。

而且私心覺得新布羅利比較帥。

2.不知道是不是因為中配的關係，這次的大王的翻譯字幕上太過謙卑了，大王是很有禮貌沒
錯，但是這並不代表他會對下屬說【小的】【前輩】等用語，看起來就是超級奇怪的。

3.所以布羅利兩個人在外星球待了41年......看起來像個1X歲少年，果然塞亞人都不會老。
（還是說因為重力源不同所以他們星球剛好時間過得比較慢？）（科幻梗）

4.布馬從這次劇場版改由詩寇蒂（？）配音，我的鶴女神啊.....

5.雖然片名叫七龍珠，但是本片幾乎所有人都不把這個神奇道具當作一回事。

破壞神跟維斯：那是啥玩具
悟空跟貝吉達：又沒辦法變強
大王：只是想高五公分
布馬：只是想年輕五歲
（話說您兩位的科技力都很容易辦到吧）

6.大王在降落前通知了一下，然後悟空兩人眼看大王一行人降落。

7.過了40年，大王軍隊的太空船都沒什麼改變嘛，真是落後。

8.大王一下機就隔空取珠，兩人也完全沒阻止的意思，看來你們果然覺得龍珠完全不重要。

9.大王拿到龍珠也是顧著觀戰，外全沒有許願的意思，看起來大王也覺得龍珠完全不重要。

10.連布馬看到大王也完全不害怕，不愧是宇宙最勇敢的女性（敢使喚破壞神做這個那個）
，不過她也完全沒讓維斯把龍珠奪回來，顯然對她來說龍珠也不重要（喂）

11.話說大王上次來的時候有看到破壞神在兩人身邊，不知道是腦袋不好還是記憶喪失，反
正他又忘了這回事。維斯這次剛好又躲得很遠，他很長一段時間根本沒發現維斯的存在。

12.接下來毫無疑問是七龍珠多媒體化後最精彩的四十分鐘打戲開始了。

要說這系列的缺點是沒有絕望感，畢竟就是他兩打不贏，還有兩個神在他們旁邊當靠山，但
是布羅利那種越打越強的鎖血無敵的打法，還有第一人稱的視野.......

是不是？大王您是不是想起了那一天，面對塞亞人下集戰士卡卡洛特的恐怖？

13.兩個人輪流過招，其中貝吉達這次是頭號指名，然後依照國際慣例依然是被慘虐的份。
（話說這次是他爸的錯，貝吉達真的只是背鍋）

14.然後一個40年只在小行星打野怪的野人 Vs 大風大浪的二人組，一開始貝吉達的戰力優
勢還是很明顯的，但是好像幫忙布羅利培訓一樣，不管貝吉達如何升等，布羅利被虐幾秒後
馬上適應對方的強度，這種對手最討厭了。

15.然後當悟空換手時，我以為他要出極意之類的東西了，結果還是慢慢從一般型態開始打
。

16.從這個時候開始，布羅利的易力氣項圈終於壞掉了。

17.當悟空被慘虐時，他突然收到比克在桂林（？）的呼叫，話說這心靈通話距離很遠喔...
...

18.悟空居然說出【撐不下去還可以瞬間移動逃走】，這是注重榮譽的塞亞戰士能說的話嗎
？ 悟空怎麼可能會做這種事嘛

19.然後他就真的這麼做了......臨走前把鍋甩給了大王（貝吉達：我老婆還在現場欸）

20.於是兩人開始學習合體，果不其然的兩次經典失敗型態（共花一個小時）後終於成功，
感謝黃金大王的努力！！

21.慘虐大王之後，越戰越勇的布羅利這次跟維斯打上了，雙藍加黃金大王都被虐的布羅利
居然根本摸不到維斯，其實力還是遠超眾人。

22.變身後的超藍悟吉特屌虐布羅利，並且一發龜派氣功終於要了解他，而布羅利也露出了
慣例（被了結專用表情）時，神龍終於發揮了作用把布羅利傳送走了。

23.於是世界終於恢復了和平，故事也來到了尾聲，總的來說這是一部非常好看的劇場版，

試著簡單分析一下101回的風太郎是否將做出選擇，與可能劇情發展的推演

基本上，這篇文有試著努力拔掉黨派的偏見

從劇情順暢度以及發展潛力可能性下加以推論

如果客觀上還是有對每一位姊妹產生不同偏袒的話，還請各大黨見諒

置頂防雷







(一) 風太郎沒有做出選擇: 90%


從場外現實面銷售&人氣因素，以及劇情中已經埋梗的情況下

再加上劇情上完全沒有呈現風太郎有認真進行思考的過程

因此得出風太郎因為各種狀況沒有做出單一選擇的可能

而造成這樣結果的原因，各地五學討論區整理出來的可能性主要有以下幾種:



(A) 我全不要 or 還是想全都要


這種狀況春場自己已經埋好梗就是...所以機率並不低

https://i.imgur.com/9OASWMM.jpg

你前一頁說三天就要選實在太短了，那如果又選得出來不就前後矛盾XD

(B) 意外打斷


這種情形就是畫面沒有畫出到底找了誰、要說的是什麼

直接說因為一些緊急狀況，風太郎必須突然離開

這種情形雖然機率上是存在，但個人認為不會這樣畫

因為實在有點太老梗


(C) 跑其他線


這種狀況就是風太郎找來談的人是其他人，並不是實際上是煙霧彈的五姊妹

如果是這種情況的話，我會認為最有可能是兵長，其他人機率都很低


(D) 倒敘手法


因為學園祭埋了很多暗梗，所以包刮中日在內也有許多人認為

下一回開始直接跳回學園祭第一天，用五姊妹或是其他人的視角

: 基本上，這篇文有試著努力拔掉黨派的偏見
: 從劇情順暢度以及發展潛力可能性下加以推論
: 如果客觀上還是有對每一位姊妹產生不同偏袒的話，還請各大黨見諒
: 而機率值只是個人的觀點而已，做為參考即可
: 置頂防雷
: (一) 風太郎沒有做出選擇: 90%
: 從場外現實面銷售&人氣因素，以及劇情中已經埋梗的情況下
: 再加上劇情上完全沒有呈現風太郎有認真進行思考的過程
: 因此得出風太郎因為各種狀況沒有做出單一選擇的可能
: 而造成這樣結果的原因，各地五學討論區整理出來的可能性主要有以下幾種:
: (A) 我全不要 or 還是想全都要
:     這種狀況春場自己已經埋好梗就是...所以機率並不低
:     https://i.imgur.com/9OASWMM.jpg
:     你前一頁說三天就要選實在太短了，那如果又選得出來不就前後矛盾XD
:     仔細思考做出大家都繼續朋友的宣言雖然對讀者而言很X
:     但不失為一種拖戲+維持懸念的手段
: (B) 意外打斷
:     這種情形就是畫面沒有畫出到底找了誰、要說的是什麼
:     直接說因為一些緊急狀況，風太郎必須突然離開
:     這種情形雖然機率上是存在，但個人認為不會這樣畫
:     因為實在有點太老梗
: (C) 跑其他線
:     這種狀況就是風太郎找來談的人是其他人，並不是實際上是煙霧彈的五姊妹
:     可能是中野爸...可能是竹林...可能是自己老爸...可能是武田等等
:     如果是這種情況的話，我會認為最有可能是兵長，其他人機率都很低
: (D) 倒敘手法
:     因為學園祭埋了很多暗梗，所以包刮中日在內也有許多人認為
:     下一回開始直接跳回學園祭第一天，用五姊妹或是其他人的視角
:     講解一下這三天到底暗中是發生了哪些事，幾回後才切回風太郎現在的視角
:     這個方法春場在期末考的時候也玩過，所以機率也是存在的
:     而且更可以繼續多拖個幾回XD
: (二) 風太郎選擇了單一姊妹: 10%
: ->這種狀況下又可以衍生出下面兩種分枝
: 1.被該位姊妹拒絕: 70%
:   這機率就是完全場外因素考量了，如果接受然後後面就一路到底沒有變化的情況
:   對一部戀愛漫畫來講是不差啦...多發發糖就好，但就黨爭劇情張力來看機率超低
:   你後面要給沒被選到姊妹什麼劇情啊XD
:   但是這種情況只有可能是選擇一花、四葉、五月等3位姊妹才會出現此種劇情
:   因為從讀者視角而言二乃跟三玖並沒有任何拒絕的理由吧?
: 2.被該位姊妹接受: 30%
:   多數意見都是認為這種狀況是所有可能性中最低的，但春場大家都知道的
:   誰敢說有任何劇情是不可能的呢?
:   但令人擔憂的是，為了避免其他姊妹完全被邊緣化，風太郎後面變心的機率很高QQ
:   除了四葉卡了寫真之子身分可能可以倖免於難之外，其他四位恐怕都難逃啊...
: 變心機率:
: (1) 風太郎選擇的是四葉以外的姊妹: 99%
: 這個推論的原因，主要來自於寫真之子到現在都還不開盤
: 而這種狀況直接選其他人並一路維持到最後
: 四葉與寫真之子的故事在後面將幾乎喪失所有的意義
: 而研究五學多年的大家，幾乎都對風太郎寫真相認這一段劇情與大場面
: 是五等分這部作品成敗評價的一大關鍵有所共識吧?
: 那麼在前述風太郎已經心意已決且後續也沒變心的劇情下，這一段落將變成
: "原來四葉你是當年的女孩啊，謝謝妳。但我現在愛的是其他人"
: 這種雷聲大雨點小到有點好笑的故事www
: 所以畫出這種劇情的話，後面風太郎變心的機率大概可說是極高
: 而最大受益者，就是身為寫真之子本人的四葉了...
: 再來，對兵團來說最殘酷的是，有日本人認為為了避免激怒太多的讀者
: 這種狀況拿來被甩，並做球給真新娘四葉的角色的很可能是一花...
: (2) 風太郎選擇的是四葉: 50%
: 其實這種狀況跟上面差不多，差別只在可以方便交往後可以避開其他姊妹的影響

: 個人簡單小結+推論:
: 1.風太郎下回做出選擇的機率，從場內外情形綜合觀察機率都非常低
: 2.即使風太郎做出選擇的情形下，被該位姊妹拒絕的可能性也相當高

:   因為若選了她倆其中一人，後續要翻盤的難度最高，但不翻盤又會失去劇情精采度

:   而且她也是下回風太郎選其他姊妹的情況下，後續最有可能拿來翻盤的角色
