 標題  Re: [問題] 機人番要怎樣做才能重現熱度呢?                               
 時間  Fri May 29 03:14:10 2020

※ 引述《w510048 (黨的恩情比山高)》之銘言：
: 我想請問大家，
: 機人番要怎樣改變才能再度浴火重生點燃熱潮呢?
: 要改變劇本?改變設定?
: 如果未來某個國家成功將人形兵器實用於戰場上，
: 例如使用人形兵器打敗匿蹤戰機殲-20。
: 讓世界各國看到人形兵器的價值。
: 機人番是不是界能夠重啟熱潮呢?

其實會受歡迎的東西，代表了那一代人的夢想。那些人幻想受女孩子
歡迎就興起後宮漫，那些人對於現實絕望覺得無力改變就會興起穿越
作品。

機人作品在以前之所以受歡迎，是有時代背景的，主要是五六十年代
出生的一輩人，他們面對一個這樣的時代：小時候害怕核戰和第三次
世界大戰，可是世界其實是在走向和平和進步，從百廢待舉的貧窮五
六十年代，去到看到美國人登陸月球，竟然是 1969 年的事情。你想
想，一個五十年代出生的小孩子，他面對的是戰後的飢荒與廢墟，可
是剛成年時，人類卻在登陸月球，書本變成了電視機，一切東西變成
了自動，飛機也開始負擔得起了。

對於那代人來說，這是記憶非常深刻的事件，形成了那一代人的觀點
，也就是科技與世界是可以突飛猛進的，我的中學是六十年代成立的
，校歌就寫了我們這一輩子就要進入太空時代。

我留意到自己的上一輩人，都非常堅信科學昌明，能解決一切問題，
相信自己能夠健康，長壽，能夠可以做很少工作都有好的生活；畢竟
從 1950 年走到 1970 年，世界已進步那麼多，再走 20 年，30 年
，將會是無法想像。

現在我們看 2000 年是老掉牙，但是在 1970 和 1980 年的作品是怎
看 2000 年的？超時空要塞 2010 年我們去到冥王星，六神合體在這
時候就已經有宇宙艦隊，其實那時候的人類寄望的二十一世紀初是個
太空時代，這也是 1969 年的登月慣性導致的。我小時候就活在這個
時代的末期，我完全感受到當年對於科技的樂觀氣氛，導致的科幻繁
榮時代。

但科幻本身其實是很成人的東西，因為涉及科學的東西其實就不那麼
人性，其實對於兒童來說並不那麼有趣，因為難以理解，我們要怎樣
讓兒童想要看「沙丘魔堡」？所以當打進少年兒童市場的時候，其實
還是要用回最傳統的兒童物語，就是神話，英雄故事，冒險故事。

機人就是包裝的方式，其實你細心留意機人番，所謂機人其實並不是
一個單獨的類型，機人其實是一種包裝方式。在機人作品裡其實是多
種不同的古老故事的復刻。

例如包青天式的水戶黃門（無敵 ROBO），斬妖除魔的故事（太空西
遊記），西部俠客故事（太空三劍俠），大航海時代（機甲艦隊），
三角戀愛劇（超時空要塞），去到八十年代的超獸機神是將玩樂團包
裝成機械人，你會發覺機人只是讓任何一個故事變成科幻和可以賣玩
具的方式。去到今天你也可以將 Fate 加機人，Servant 全部變成機
人又有何不可？任何故事加上了我們對二十世紀未來的幻想，以及對
科技的期許，就是機人番。

然則，機人番何時沒落的？其實就九十年代，蘇聯解體之後的世界，
其實也停止了 1969 年之後的太空時代，我們從九十年代走到二十一
世紀，漸漸醒覺到一件事：我們有生之年根本就不會上太空。沒有月
球基地，沒有登陸火星，沒有遇到外星人，人類沒有因為科技而解決
經濟問題，疾病飢餓失業還是存在，2001 年沒有太空漫遊，我們還
是過著很像二十世紀的生活。

二十一世紀並沒有想像中的那麼令人興奮，我們面對的問題，還是拿
著棍子的暴力，暴政，貧窮。我小時候比現在更窮，但我對未來是充
滿希望的，我當年學習科技就是認為科技能改變我們的未來，現在應
該沒甚麼小孩會這樣想了。

為何八十年代那麼快樂？我想了很久，就是因為我們的信仰，其實相
信未來的科技進步，經濟會更繁榮，明天會更好，是快樂的最重要理
由。科技就有如神力一樣，我小時候相信，它可以令貧窮的我變富有
，可以探索未知的星球，可以健康長壽，甚至永生不死。相信有全能
的神是幸福的，當年的科學就是這個神。

現在我完全沒有這想法，今天我們在幹嘛？我們在為了一個像中世紀
的傳染病焦頭爛額，科技當然有進步，但我不再相信它有甚麼神奇力
量，也不相信有甚麼能改善我們生命的大躍進。

沒有那麼大的希望，就沒有那麼多的快樂了，現在的人相信投胎多過
相信科技。

科學走下了神檯，社會感到解決問題的不再是科學，漸漸的，受歡迎
的變成了別的東西，比起幻想我們能穿上高科技的機械人去作戰，是
否想像受一堆女孩子包圍著更現實可行呢？其實都是社會風氣導致，
去到覺得現實已如鋼鐵一樣難以改變時，穿越系就興盛，現實廢柴穿
越了就變龍傲天。今天你做出和七十年代一樣的機人番，也很難再受
歡迎，因為社會早已沒有了那種夢和氣氛。機人不機人變成了作者個
人的愛好，你今天還可以做機人，但對於讀者已沒有加成了。因為已
跟現實完全脫離了，他不再是一種希望，只是一種設定而已。

機人不機人根本不重要，會受歡迎的東西，沒有機人也會受歡迎，有
了也不會變得更受歡迎：就作者覺得要有機人就機人，沒有也沒關係
的時代。

現在不是沒有機人作品，其實每天都有，但不會有那時候的那種狂熱
和感覺，因為機人出現了我們只會感到疏離。就像看環太平洋一樣，
我們看到巨大機械人，只會聯想到是大叔的浪漫，所以很好看，絕不
會覺得這東西會在現實出現。

要怎樣才能夠令機人番復興？我想可能下一次像二戰一樣的浩劫重生
，再來一次像登月一樣興奮的事件，可能吧。

: 現在我們看 2000 年是老掉牙，但是在 1970 和 1980 年的作品是怎
: 看 2000 年的？超時空要塞 2010 年我們去到冥王星，六神合體在這
: 時候就已經有宇宙艦隊，其實那時候的人類寄望的二十一世紀初是個
: 太空時代，這也是 1969 年的登月慣性導致的。我小時候就活在這個
: 時代的末期，我完全感受到當年對於科技的樂觀氣氛，導致的科幻繁
: 榮時代。
: 但科幻本身其實是很成人的東西，因為涉及科學的東西其實就不那麼
: 人性，其實對於兒童來說並不那麼有趣，因為難以理解，我們要怎樣
: 讓兒童想要看「沙丘魔堡」？所以當打進少年兒童市場的時候，其實
: 還是要用回最傳統的兒童物語，就是神話，英雄故事，冒險故事。
: 機人就是包裝的方式，其實你細心留意機人番，所謂機人其實並不是
: 一個單獨的類型，機人其實是一種包裝方式。在機人作品裡其實是多
: 種不同的古老故事的復刻。
: 例如包青天式的水戶黃門（無敵 ROBO），斬妖除魔的故事（太空西
: 遊記），西部俠客故事（太空三劍俠），大航海時代（機甲艦隊），
: 三角戀愛劇（超時空要塞），去到八十年代的超獸機神是將玩樂團包
: 裝成機械人，你會發覺機人只是讓任何一個故事變成科幻和可以賣玩
: 具的方式。去到今天你也可以將 Fate 加機人，Servant 全部變成機
: 人又有何不可？任何故事加上了我們對二十世紀未來的幻想，以及對
: 科技的期許，就是機人番。
: 然則，機人番何時沒落的？其實就九十年代，蘇聯解體之後的世界，
: 其實也停止了 1969 年之後的太空時代，我們從九十年代走到二十一
: 世紀，漸漸醒覺到一件事：我們有生之年根本就不會上太空。沒有月
: 球基地，沒有登陸火星，沒有遇到外星人，人類沒有因為科技而解決
: 經濟問題，疾病飢餓失業還是存在，2001 年沒有太空漫遊，我們還
: 是過著很像二十世紀的生活。
: 二十一世紀並沒有想像中的那麼令人興奮，我們面對的問題，還是拿
: 著棍子的暴力，暴政，貧窮。我小時候比現在更窮，但我對未來是充
: 滿希望的，我當年學習科技就是認為科技能改變我們的未來，現在應
: 該沒甚麼小孩會這樣想了。
: 為何八十年代那麼快樂？我想了很久，就是因為我們的信仰，其實相
: 信未來的科技進步，經濟會更繁榮，明天會更好，是快樂的最重要理
: 由。科技就有如神力一樣，我小時候相信，它可以令貧窮的我變富有
: ，可以探索未知的星球，可以健康長壽，甚至永生不死。相信有全能
: 的神是幸福的，當年的科學就是這個神。
: 現在我完全沒有這想法，今天我們在幹嘛？我們在為了一個像中世紀
: 的傳染病焦頭爛額，科技當然有進步，但我不再相信它有甚麼神奇力
: 量，也不相信有甚麼能改善我們生命的大躍進。
: 沒有那麼大的希望，就沒有那麼多的快樂了，現在的人相信投胎多過
: 相信科技。
: 科學走下了神檯，社會感到解決問題的不再是科學，漸漸的，受歡迎
: 的變成了別的東西，比起幻想我們能穿上高科技的機械人去作戰，是
: 否想像受一堆女孩子包圍著更現實可行呢？其實都是社會風氣導致，
: 去到覺得現實已如鋼鐵一樣難以改變時，穿越系就興盛，現實廢柴穿
: 越了就變龍傲天。今天你做出和七十年代一樣的機人番，也很難再受
: 歡迎，因為社會早已沒有了那種夢和氣氛。機人不機人變成了作者個
: 人的愛好，你今天還可以做機人，但對於讀者已沒有加成了。因為已
: 跟現實完全脫離了，他不再是一種希望，只是一種設定而已。
: 機人不機人根本不重要，會受歡迎的東西，沒有機人也會受歡迎，有
: 了也不會變得更受歡迎：就作者覺得要有機人就機人，沒有也沒關係
: 的時代。
: 現在不是沒有機人作品，其實每天都有，但不會有那時候的那種狂熱
: 和感覺，因為機人出現了我們只會感到疏離。就像看環太平洋一樣，
: 我們看到巨大機械人，只會聯想到是大叔的浪漫，所以很好看，絕不
: 會覺得這東西會在現實出現。
: 要怎樣才能夠令機人番復興？我想可能下一次像二戰一樣的浩劫重生
: ，再來一次像登月一樣興奮的事件，可能吧。
